# Использование фреймворков для разработки адаптивного веб-дизайна. Фреймворк Bootstrap

## Цели и задачи лекции
- Познакомить студентов с концепцией адаптивного веб-дизайна (Responsive Web Design).
- Рассмотреть особенности и преимущества использования Bootstrap.
- Научить создавать простые адаптивные страницы с помощью компонентов Bootstrap.
- Разобрать практические примеры и шаблоны кода.

## Ключевые понятия
| Термин | Определение |
|--------|-------------|
| Адаптивный дизайн | Создание веб‑страниц, корректно отображающихся на разных устройствах. |
| Фреймворк | Система библиотек, инструментов и шаблонов, упрощающая разработку. |
| Bootstrap | Популярный CSS/JS‑фреймворк, разработанный Twitter, обеспечивающий адаптивность. |
| Grid‑система | Сетка, позволяющая размещать элементы в ряды и колонки. |
| Компоненты | Предварительно стилизованные элементы UI (кнопки, панели, навигация). |
| Вьюпорт | Видимая часть страницы браузера. |
| Media Queries | CSS‑правила, применяемые при определённых ширинах экрана. |
| Flexbox | CSS‑модуль для гибкого размещения элементов. |
| CDN | Content Delivery Network – сеть серверов, ускоряющая доставку ресурсов. |
| JIT (Just‑in‑Time) | Стиль разработки, где стили генерируются только при необходимости. |

## Основное содержание

### 1. Введение в адаптивный веб-дизайн (15 мин)
- История и принципы адаптивности.
- Отличие от «декоративного» дизайна.
- Роль медиазапросов и гибких единиц измерения.

### 2. Архитектура Bootstrap (15 мин)
- Структура проекта: CSS, JS, компоненты, темы.
- Как подключить Bootstrap: CDN, npm, локальные файлы.
- Важность иерархии классов и «mobile‑first» подход.

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

### 3. Grid‑система Bootstrap (20 мин)
- Колонки, ряды, контейнеры (`container`, `container-fluid`).
- Схема 12‑колонок.
- Адаптивные префиксы: `col-sm-`, `col-md-`, `col-lg-`, `col-xl-`.
- Пример: простая сетка с разным количеством колонок на разных устройствах.

```html
<div class="container">
  <div class="row">
    <div class="col-12 col-md-6 col-lg-4">Колонка 1</div>
    <div class="col-12 col-md-6 col-lg-4">Колонка 2</div>
    <div class="col-12 col-md-12 col-lg-4">Колонка 3</div>
  </div>
</div>
```

### 4. Компоненты Bootstrap (15 мин)
- Навигационные панели (`navbar`).
- Карточки (`card`), списки (`list-group`).
- Кнопки, формы, алерты.
- Пример: адаптивная карточка с изображением.

```html
<div class="card" style="width: 18rem;">
  <img src="..." class="card-img-top" alt="...">
  <div class="card-body">
    <h5 class="card-title">Заголовок</h5>
    <p class="card-text">Краткое описание.</p>
    <a href="#" class="btn btn-primary">Перейти</a>
  </div>
</div>
```

### 5. Практика: Создание простого адаптивного сайта (20 мин)
- Структура проекта: `index.html`, `style.css`.
- Подключение Bootstrap.
- Размещение элементов: заголовок, навигация, контент, footer.
- Проверка в разных разрешениях.

```html
<!doctype html>
<html lang="ru">
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <title>Bootstrap‑пример</title>
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
</head>
<body>
  <nav class="navbar navbar-expand-lg navbar-light bg-light">
    <div class="container-fluid">
      <a class="navbar-brand" href="#">Мой сайт</a>
      <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarsExample">
        <span class="navbar-toggler-icon"></span>
      </button>
      <div class="collapse navbar-collapse" id="navbarsExample">
        <ul class="navbar-nav me-auto mb-2 mb-lg-0">
          <li class="nav-item"><a class="nav-link active" href="#">Главная</a></li>
          <li class="nav-item"><a class="nav-link" href="#">О нас</a></li>
        </ul>
      </div>
    </div>
  </nav>
  <div class="container mt-4">
    <h1 class="display-4">Добро пожаловать!</h1>
    <p class="lead">Это пример адаптивной страницы на Bootstrap.</p>
  </div>
  <footer class="bg-light text-center py-3 mt-4">
    © 2025 Компания
  </footer>
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

## Выводы по лекции
- Адаптивный веб-дизайн обеспечивает удобство использования сайтов на любых устройствах.
- Bootstrap предоставляет готовые инструменты (grid, компоненты, стили), ускоряющие разработку.
- Понимание принципов «mobile‑first» и гибких сеток критично для создания качественного UI.
- Практика с Bootstrap позволяет быстро создавать адаптивные прототипы и готовые решения.
