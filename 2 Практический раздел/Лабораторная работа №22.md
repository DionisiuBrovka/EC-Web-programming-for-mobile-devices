# Лабораторная работа № 22  
Верстка шаблона с использованием компонентов Bootstrap для работы с выпадающими списками, кнопками, миниатюрами, модальными окнами, индикатором процесса  

## Цель и задачи  
1. Научиться создавать адаптивный интерфейс с помощью Bootstrap 5.  
2. Освоить работу с компонентами: выпадающими списками, кнопками, миниатюрами (thumbnails), модальными окнами и индикатором процесса (spinner).  
3. Применить полученные навыки для разработки простого одностраничного шаблона.  
4. Оценить корректность отображения компонентов в разных браузерах и на мобильных устройствах.  

## Теоретическая часть  
| Понятие | Краткое определение |
|---------|---------------------|
| Bootstrap | CSS/JS-фреймворк для быстрой разработки адаптивных веб‑страниц. |
| Grid‑система | Сетка из 12 колонок, позволяющая управлять расположением элементов. |
| Компоненты | Предварительно стилизованные элементы UI (кнопки, формы, модалы). |
| Responsive design | Дизайн, корректно отображающийся на различных размерах экранов. |
| CDN | Content Delivery Network – сеть серверов, откуда загружаются внешние ресурсы. |
| Spinner | Анимация индикатора загрузки. |
| Modal | Всплывающее окно поверх основного контента. |
| Dropdown | Выпадающий список с пунктами выбора. |
| Thumbnail | Миниатюрное изображение с подписью. |
| JavaScript‑нигма | Библиотека, позволяющая управлять состоянием компонентов без перезагрузки страницы. |

## Ход выполнения работы  

1. **Создание проекта**  
   - Создайте папку `lab22` и в ней файл `index.html`.  
   - Подключите Bootstrap 5 через CDN:  

   ```html
   <head>
     <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
     <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
   </head>
   ```

2. **Разметка основной структуры**  
   - Добавьте контейнер `container` и сетку `row`/`col` для размещения компонентов.  

   ```html
   <body>
     <div class="container my-4">
       <div class="row">
         <div class="col-md-6">...</div>
         <div class="col-md-6">...</div>
       </div>
     </div>
   </body>
   ```

3. **Выпадающий список (Dropdown)**  
   - Вставьте компонент с заголовком и несколькими пунктами.  

   ```html
   <div class="dropdown">
     <button class="btn btn-secondary dropdown-toggle" type="button" data-bs-toggle="dropdown">
       Выберите пункт
     </button>
     <ul class="dropdown-menu">
       <li><a class="dropdown-item" href="#">Пункт 1</a></li>
       <li><a class="dropdown-item" href="#">Пункт 2</a></li>
       <li><a class="dropdown-item" href="#">Пункт 3</a></li>
     </ul>
   </div>
   ```

4. **Кнопки (Buttons)**  
   - Добавьте несколько кнопок различных стилей и размеров.  

   ```html
   <button class="btn btn-primary btn-lg">Кнопка большого размера</button>
   <button class="btn btn-outline-success">Обводка</button>
   ```

5. **Миниатюры (Thumbnails)**  
   - Используйте класс `card` для создания миниатюр с изображением и подписью.  

   ```html
   <div class="card" style="width: 18rem;">
     <img src="img/sample.jpg" class="card-img-top" alt="Изображение">
     <div class="card-body">
       <h5 class="card-title">Заголовок</h5>
       <p class="card-text">Краткое описание.</p>
     </div>
   </div>
   ```

6. **Модальное окно (Modal)**  
   - Создайте кнопку, открывающую модал, и сам модал.  

   ```html
   <button type="button" class="btn btn-primary" data-bs-toggle="modal" data-bs-target="#demoModal">
     Открыть модал
   </button>

   <div class="modal fade" id="demoModal" tabindex="-1">
     <div class="modal-dialog">
       <div class="modal-content">
         <div class="modal-header">
           <h5 class="modal-title">Заголовок модала</h5>
           <button type="button" class="btn-close" data-bs-dismiss="modal"></button>
         </div>
         <div class="modal-body">
           Текст модального окна.
         </div>
         <div class="modal-footer">
           <button type="button" class="btn btn-secondary" data-bs-dismiss="modal">Закрыть</button>
         </div>
       </div>
     </div>
   </div>
   ```

7. **Индикатор процесса (Spinner)**  
   - Добавьте спиннер в центр контейнера.  

   ```html
   <div class="d-flex justify-content-center my-5">
     <div class="spinner-border" role="status">
       <span class="visually-hidden">Загрузка…</span>
     </div>
   </div>
   ```

8. **Тестирование и отладка**  
   - Откройте `index.html` в браузере, проверьте корректность отображения всех компонентов.  
   - Переключитесь на мобильный режим (DevTools) и убедитесь в адаптивности.  
   - Запишите замечания о возможных ошибках (например, неправильный порядок подключения скриптов).
