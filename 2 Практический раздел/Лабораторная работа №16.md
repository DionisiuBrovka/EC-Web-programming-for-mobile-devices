# Лабораторная работа № 16. Использование свойств и методов иерархической структуры DOM для доступа к элементам веб‑страницы

## Цель и задачи
1. Научиться работать с объектной моделью документа (DOM) в браузере.  
2. Освоить методы и свойства, позволяющие находить, изменять и создавать элементы DOM.  
3. Выполнить практическое задание по автоматизации простого взаимодействия с веб‑страницей на основе Python‑скрипта.  

## Теоретическая часть
| Понятие | Определение | Примечание |
|---------|-------------|------------|
| DOM (Document Object Model) | Структурированное представление HTML‑/XML‑документа в виде дерева объектов. | Каждому элементу соответствует узел. |
| Узел (Node) | Базовый элемент DOM: элемент, атрибут, текст, комментарий. | Методы `nodeType`, `nodeName` и др. |
| Элемент (Element) | Узел типа 1, представляющий тег HTML. | Методы `getAttribute`, `setAttribute`. |
| Текстовый узел | Узел типа 3, содержащий текст внутри элемента. | Свойство `textContent`. |
| Методы поиска: `getElementById`, `getElementsByTagName`, `getElementsByClassName`, `querySelector`, `querySelectorAll` | Возвращают узлы по ID, тегу, классу, CSS‑селектору. | `querySelectorAll` возвращает NodeList. |
| Свойства доступа: `innerHTML`, `outerHTML`, `textContent`, `value` | Позволяют читать/изменять содержимое элемента. | `innerText` отличается от `textContent`. |
| Создание узла: `document.createElement`, `document.createTextNode` | Создают новые элементы/текст. | Добавление через `appendChild`, `insertBefore`. |
| Навигация: `parentNode`, `children`, `nextSibling`, `previousSibling` | Позволяют перемещаться по дереву. | |
| События: `addEventListener`, `removeEventListener` | Позволяют реагировать на пользовательские действия. | |
| Браузерное API: `window`, `document`, `navigator` | Глобальные объекты для взаимодействия с браузером. | |
| Selenium WebDriver | Библиотека для автоматизации браузера из Python. | Позволяет управлять DOM программно. |

## Ход выполнения работы

1. **Подготовка среды**  
   - Установить Python 3.10+ и менеджер пакетов `pip`.  
   - Установить Selenium:  
     ```bash
     pip install selenium
     ```  
   - Скачать драйвер браузера (ChromeDriver/GeckoDriver) и разместить в системном пути.  

2. **Создание проекта**  
   - Создать папку `lab16` и файл `main.py`.  
   - В `main.py` импортировать необходимые модули:  
     ```python
     from selenium import webdriver
     from selenium.webdriver.common.by import By
     from selenium.webdriver.common.keys import Keys
     ```  

3. **Открытие веб‑страницы**  
   - Инициализировать драйвер и открыть страницу:  
     ```python
     driver = webdriver.Chrome()
     driver.get('https://example.com')
     ```  

4. **Нахождение элементов**  
   - Поиск элемента по ID:  
     ```python
     header = driver.find_element(By.ID, 'header')
     ```  
   - Поиск всех ссылок:  
     ```python
     links = driver.find_elements(By.TAG_NAME, 'a')
     ```  
   - Поиск по CSS‑селектору:  
     ```python
     button = driver.find_element(By.CSS_SELECTOR, 'button.btn-primary')
     ```  

5. **Чтение и изменение содержимого**  
   - Вывести текст заголовка:  
     ```python
     print(header.text)
     ```  
   - Изменить атрибут изображения:  
     ```python
     img = driver.find_element(By.TAG_NAME, 'img')
     driver.execute_script("arguments[0].src='new_image.jpg'", img)
     ```  

6. **Создание и вставка нового элемента**  
   - Создать новый `<p>` и добавить в `body`:  
     ```python
     driver.execute_script("""
       var p = document.createElement('p');
       p.textContent = 'Новый параграф';
       document.body.appendChild(p);
     """)
     ```  

7. **Навигация по DOM**  
   - Перейти к родителю элемента:  
     ```python
     parent = header.find_element(By.XPATH, '..')
     ```  
   - Получить всех потомков:  
     ```python
     children = header.find_elements(By.XPATH, './/*')
     ```  

8. **Обработчики событий (опционально)**  
   - Добавить к кнопке событие клика, которое выводит сообщение:  
     ```python
     driver.execute_script("""
       var btn = arguments[0];
       btn.addEventListener('click', function() { alert('Клик!'); });
     """, button)
     ```  

9. **Завершение работы**  
   - Закрыть браузер:  
     ```python
     driver.quit()
     ```  

10. **Анализ результатов**  
    - Сохранить скриншот страницы после выполнения скрипта:  
      ```python
      driver.save_screenshot('result.png')
      ```
