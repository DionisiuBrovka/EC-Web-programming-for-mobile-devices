# Объекты и методы в JavaScript

## Цели и задачи лекции
- Понять роль объектов как базовой структуры данных в JavaScript.
- Освоить понятие методов как функций, принадлежащих объектам.
- Научиться создавать и модифицировать объекты, используя литералы и конструкторы.
- Уметь работать с встроенными объектами и их методами.

## Ключевые понятия
- **Объект** – коллекция свойств, где каждое свойство имеет имя и значение.  
- **Свойство** – пара «ключ‑значение» внутри объекта.  
- **Метод** – свойство‑функция, выполняющая действие над объектом.  
- **Конструктор** – функция, создающая новые объекты с заданной структурой.  
- **this** – ключевое слово, указывающее на объект, к которому относится вызываемая функция.  
- **Прототип** – объект, от которого наследуются свойства и методы.  
- **Модуль** – автономный набор кода, экспортируемый и импортируемый в другие модули.  
- **Наследование** – механизм передачи свойств и методов от одного объекта к другому.  
- **Геттер/Сеттер** – методы доступа к свойствам, позволяющие контролировать чтение и запись.  
- **Инкапсуляция** – скрытие внутреннего состояния объекта от внешнего кода.

## Основное содержание

### 1. Объекты в JavaScript (30 мин)
#### 1.1. Создание объектов
- Литерал объектов
- Конструкторы и `new`
```js
// Литерал
const person = { name: 'Alice', age: 30 };

// Конструктор
function Person(name, age) {
  this.name = name;
  this.age = age;
}
const bob = new Person('Bob', 25);
```
#### 1.2. Свойства и методы
- Доступ к свойствам через точку и скобки
- Добавление/удаление свойств
```js
person.email = 'alice@example.com';
delete person.age;
```

### 2. Методы как функции внутри объектов (20 мин)
#### 2.1. Определение методов
```js
const calculator = {
  add(a, b) { return a + b; },
  subtract(a, b) { return a - b; }
};
```
#### 2.2. Контекст `this`
```js
const obj = {
  value: 10,
  increment() { this.value++; }
};
obj.increment(); // value becomes 11
```
#### 2.3. Вызов методов из других объектов
```js
const math = { sum(a, b) { return a + b; } };
const stats = {
  total: 0,
  addValue(v) { this.total = math.sum(this.total, v); }
};
```

### 3. Прототипы и наследование (25 мин)
#### 3.1. Прототипный механизм
```js
const animal = { speak() { console.log('...'); } };
const dog = Object.create(animal);
dog.bark = function() { console.log('Woof!'); };
```
#### 3.2. Наследование через конструкторы
```js
function Animal(name) { this.name = name; }
Animal.prototype.move = function() { console.log(`${this.name} moves`); };

function Dog(name, breed) {
  Animal.call(this, name);
  this.breed = breed;
}
Dog.prototype = Object.create(Animal.prototype);
Dog.prototype.constructor = Dog;
```

### 4. Модули и экспорт/импорт (15 мин)
#### 4.1. ES6‑модули
```js
// math.js
export function add(a, b) { return a + b; }

// main.js
import { add } from './math.js';
console.log(add(2, 3));
```
#### 4.2. Работа с объектами в модулях
```js
export default {
  name: 'module',
  greet() { console.log('Hello'); }
};
```

### 5. Практическое задание (10 мин)
Создать объект `library`, содержащий массив книг, методы для добавления, удаления и поиска книг по названию.  
```js
const library = {
  books: [],
  add(book) { this.books.push(book); },
  remove(title) { this.books = this.books.filter(b => b.title !== title); },
  find(title) { return this.books.find(b => b.title === title); }
};
```

## Выводы по лекции
- Объекты в JavaScript являются фундаментальной структурой данных, объединяющей свойства и методы.  
- Методы позволяют инкапсулировать поведение объекта и управлять контекстом `this`.  
- Прототипный механизм обеспечивает наследование и повторное использование кода.  
- ES6‑модули способствуют модульности и масштабируемости проектов.
