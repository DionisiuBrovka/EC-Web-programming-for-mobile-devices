# Создание пользовательских функций в JavaScript

## Цели и задачи лекции
- Понять роль функций как фундаментального строительного блока JavaScript.
- Освоить синтаксис объявления, вызова и передачи аргументов в пользовательские функции.
- Научиться использовать функции как объекты высшего порядка и изучить лексическое окружение.
- Применить полученные знания на практике через простые и продвинутые примеры.

## Ключевые понятия
| Термин | Определение |
|--------|-------------|
| Функция | Пакет кода, выполняемый при вызове, может принимать параметры и возвращать результат. |
| Аргумент | Значение, передаваемое в функцию при вызове. |
| Параметр | Переменная, объявленная в заголовке функции, принимающая аргумент. |
| Возвратное значение | Значение, которое функция передаёт наружу с помощью `return`. |
| Замыкание | Способ функции запоминать переменные из внешнего лексического окружения. |
| Анонимная функция | Функция без имени, часто используется как аргумент или в выражениях. |
| Стрелочная функция | Сокращённая запись функций, сохраняющая контекст `this`. |
| Вызываемая функция | Функция, которая может быть вызвана как часть выражения. |
| Объект-функция | Функция, которая является свойством объекта и может использовать `this`. |
| Хвостовой вызов | Оптимизация, при которой последняя операция в функции — вызов другой функции. |

## Основное содержание

### 1. Синтаксис объявлений функций (30 мин)  
#### 1.1 Объявление через ключевое слово `function`  
```js
function greet(name) {
    console.log(`Hello, ${name}!`);
}
```
#### 1.2 Функции-выражения (функциональный литерал)  
```js
const add = function(a, b) {
    return a + b;
};
```
#### 1.3 Стрелочные функции (ES6+)  
```js
const multiply = (x, y) => x * y;
```
- Отличия: отсутствие собственного `this`, `arguments`, `super`, `new.target`.

### 2. Параметры и аргументы (20 мин)  
#### 2.1 Параметры по умолчанию  
```js
function log(msg = 'No message') {
    console.log(msg);
}
```
#### 2.2 Оператор rest (`...args`)  
```js
function sum(...nums) {
    return nums.reduce((a, b) => a + b, 0);
}
```
#### 2.3 Деструктуризация в списке параметров  
```js
function show({title, author}) {
    console.log(`${title} by ${author}`);
}
```

### 3. Возврат значений и типы возвращаемых данных (15 мин)  
#### 3.1 Явный `return`  
#### 3.2 Неявный `undefined`  
#### 3.3 Объекты, массивы и функции как возвращаемые значения  
```js
function createUser(name) {
    return {name, age: 0};
}
```

### 4. Замыкания и лексическое окружение (15 мин)  
#### 4.1 Пример замыкания  
```js
function counter() {
    let count = 0;
    return function() {
        count++;
        return count;
    };
}
const next = counter();
console.log(next()); // 1
console.log(next()); // 2
```
#### 4.2 Применение замыканий для приватных данных  
```js
const secret = (() => {
    let pwd = '1234';
    return {
        get: () => pwd,
        set: newPwd => { pwd = newPwd; }
    };
})();
```

### 5. Функции как объекты высшего порядка (20 мин)  
#### 5.1 Передача функции как аргумента  
```js
function process(arr, fn) {
    return arr.map(fn);
}
process([1,2,3], x => x * 2);
```
#### 5.2 Функции, возвращающие другие функции  
```js
function multiplier(factor) {
    return num => num * factor;
}
const double = multiplier(2);
```
#### 5.3 Методы массива: `forEach`, `map`, `filter`, `reduce`  
```js
const nums = [1,2,3,4];
const squares = nums.map(n => n * n);
```

## Примеры и иллюстрации

```js
// Пример использования аргументов по умолчанию и rest
function greetAll(greeting = 'Hi', ...names) {
    names.forEach(name => console.log(`${greeting}, ${name}!`));
}
greetAll('Welcome', 'Alice', 'Bob', 'Charlie');
```

```js
// Пример замыкания для приватного счетчика
function createCounter() {
    let count = 0;
    return {
        increment: () => ++count,
        decrement: () => --count,
        value: () => count
    };
}
const counter = createCounter();
counter.increment(); // 1
counter.increment(); // 2
counter.decrement(); // 1
```

## Выводы по лекции
- Функции являются основным строительным элементом JavaScript и обладают свойствами объектов.
- Существует несколько способов объявления функций, каждый из которых имеет свои особенности.
- Параметры, аргументы и значения возврата позволяют создавать гибкие и переиспользуемые блоки кода.
- Замыкания дают мощный инструмент для создания приватных данных и состояний.
- Функции высшего порядка и методы массивов делают работу с коллекциями более выразительной и декларативной.
