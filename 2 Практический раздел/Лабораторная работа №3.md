# Лабораторная работа № 3. Внедрение форм с различными наборами элементов на веб‑страницу

## Цель и задачи  
1. Научиться создавать и стилизовать HTML‑формы, используя различные типы полей.  
2. Освоить работу с атрибутами `<input>`, `<select>`, `<textarea>` и их валидацию.  
3. Научиться обрабатывать данные формы на стороне клиента с помощью JavaScript.  
4. Разработать простую страницу, где пользователь может отправить данные, а сервер (Python‑Flask) отобразит их.

## Теоретическая часть  
- **HTML‑форма** – элемент `<form>`, который группирует поля ввода и отправляет данные на сервер.  
- **Метод POST** – отправка данных в теле запроса; безопаснее, чем GET.  
- **Метод GET** – параметры передаются в URL; подходит для запросов без изменений данных.  
- **Атрибут `action`** – адрес, куда отправляется запрос.  
- **Атрибут `method`** – тип HTTP‑запроса (`GET` или `POST`).  
- **Атрибут `required`** – делает поле обязательным.  
- **Типы `<input>`** – `text`, `email`, `password`, `number`, `date`, `file`, `radio`, `checkbox`, `submit`, `reset`.  
- **`<select>`** – выпадающий список; элементы `<option>`.  
- **`<textarea>`** – многострочное поле ввода.  
- **JavaScript‑валидация** – проверка данных перед отправкой (например, `value.trim() !== ''`).  
- **Фреймворк Flask** – минимальный веб‑сервер для Python, позволяющий обрабатывать запросы.  
- **Jinja2** – шаблонизатор, встроенный в Flask, для вывода данных в HTML.

## Ход выполнения работы  

1. **Подготовка проекта**  
   - Создать каталог проекта `lab3` и в нём файл `app.py`.  
   ```python
   from flask import Flask, render_template, request

   app = Flask(__name__)

   @app.route('/', methods=['GET', 'POST'])
   def index():
       if request.method == 'POST':
           data = request.form.to_dict()
           return render_template('result.html', data=data)
       return render_template('form.html')

   if __name__ == '__main__':
       app.run(debug=True)
   ```

2. **Создание шаблонов**  
   - В каталоге `templates` создать `form.html` и `result.html`.  
   - `form.html` содержит форму с разными полями.  
   ```html
   <!DOCTYPE html>
   <html lang="ru">
   <head>
       <meta charset="UTF-8">
       <title>Форма</title>
   </head>
   <body>
       <h1>Регистрационная форма</h1>
       <form action="/" method="post" id="regForm">
           <label>Имя: <input type="text" name="name" required></label><br>
           <label>Эл. почта: <input type="email" name="email" required></label><br>
           <label>Пароль: <input type="password" name="password" required></label><br>
           <label>Возраст: <input type="number" name="age" min="18" max="99"></label><br>
           <label>Пол:
               <select name="gender">
                   <option value="M">Мужской</option>
                   <option value="F">Женский</option>
               </select>
           </label><br>
           <label>Увлечения:
               <input type="checkbox" name="hobby" value="sport"> Спорт
               <input type="checkbox" name="hobby" value="music"> Музыка
           </label><br>
           <label>Комментарий:<br>
               <textarea name="comment" rows="4" cols="50"></textarea>
           </label><br>
           <button type="submit">Отправить</button>
       </form>
       <script src="form.js"></script>
   </body>
   </html>
   ```

3. **JavaScript‑валидация**  
   - Создать файл `static/form.js` и подключить его в шаблоне.  
   ```javascript
   document.getElementById('regForm').addEventListener('submit', function(e) {
       const name = e.target.name.value.trim();
       if (name.length < 3) {
           alert('Имя должно содержать минимум 3 символа');
           e.preventDefault();
       }
   });
   ```

4. **Отображение результата**  
   - `result.html` выводит отправленные данные.  
   ```html
   <!DOCTYPE html>
   <html lang="ru">
   <head><meta charset="UTF-8"><title>Результат</title></head>
   <body>
       <h1>Полученные данные</h1>
       <ul>
           {% for key, value in data.items() %}
               <li><strong>{{ key }}:</strong> {{ value }}</li>
           {% endfor %}
       </ul>
       <a href="/">Вернуться</a>
   </body>
   </html>
   ```

5. **Запуск проекта**  
   - В терминале выполнить `python app.py`.  
   - Открыть браузер по адресу `http://127.0.0.1:5000/`.  
   - Проверить корректность работы формы, валидацию и отображение данных.

## Контрольные вопросы / задания для самопроверки  

1. Что делает атрибут `required` у поля `<input>`?  
2. Чем отличается метод `GET` от `POST` при отправке формы?  
3. Как в Flask получить данные из POST‑запроса?  
4. Как вывести список чекбоксов, выбранных пользователем, в шаблоне Jinja2?  
5. Что произойдет, если в JavaScript‑валидации вызвать `e.preventDefault()`?  
6. Напишите HTML‑код для радиокнопок «Да» и «Нет».  
7. Как задать ограничение на диапазон чисел в поле `<input type="number">`?  
8. Объясните роль шаблонизатора Jinja2 в Flask.  
9. Как подключить внешний CSS‑файл к шаблону?
