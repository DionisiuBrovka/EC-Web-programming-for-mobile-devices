# Объектная модель браузера. Использование объектов HTML и объектной модели документа

## Цель занятия
1. Ознакомить студентов с ключевыми концепциями объектной модели браузера (DOM) и объектами HTML.  
2. Научить применять свойства и методы DOM в практике работы с веб‑страницами.

## Задачи занятия
- Понять структуру и назначение объектов HTML.  
- Научиться навигации по дереву DOM с помощью свойств `parentNode`, `childNodes`, `nextSibling` и др.  
- Разобрать основные методы манипуляции содержимым: `getElementById`, `querySelector`, `innerHTML`, `textContent`.  
- Практически применить полученные знания в простом упражнении по изменению DOM.  
- Оценить понимание материала через контрольные вопросы и практическое задание.

## План занятия (90 минут)

- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- Изучение нового материала: объектная модель браузера — 25 мин  
- Изучение нового материала: объекты HTML и работа с DOM — 20 мин  
- Практическое закрепление — 15 мин  
- Итог занятия и планирование домашнего задания — 5 мин  

## Содержание этапов

- **Организационный момент**  
  Приветствие, проверка присутствия, краткое объявление темы и целей.  
- **Актуализация знаний**  
  Быстрый опрос: что помните о работе с веб‑страницами в браузере? Какие инструменты разработчика вы использовали?  
- **Изучение нового материала: объектная модель браузера**  
  Введение в DOM, разбор его структуры как дерева объектов, объяснение ролей `document`, `window`, `navigator`.  
- **Изучение нового материала: объекты HTML и работа с DOM**  
  Погружение в свойства и методы элементов: `id`, `className`, `style`, методы поиска и изменения элементов.  
- **Практическое закрепление**  
  Студенты в парах создают небольшую страницу и применяют методы `querySelector`, `innerHTML` для динамического изменения контента.  
- **Итог занятия**  
  Подведение итогов, ответы на вопросы, объявление домашнего задания.

## Оборудование и материалы
- ПК/ноутбук с установленным Python 3 и VS Code  
- Проектор и экран  
- Браузер (Chrome/Firefox) с включёнными DevTools  
- Доступ к репозиторию GitHub с шаблоном проекта  
- Учебник «Python и веб» (п. 5–7)  
- Копии шпаргалок по DOM

## Контроль и оценивание
- Короткие устные вопросы после каждого блока (5 мин).  
- Практическое задание: в паре реализовать скрипт, меняющий текст заголовка по клику (5 мин).  
- Форма обратной связи: опросник в Google Forms о понимании ключевых понятий (5 мин).

## Домашнее задание
1. Написать скрипт, который при загрузке страницы подсчитывает количество элементов `<p>` и выводит результат в консоль.  
2. Создать небольшую HTML‑страницу с формой, где при отправке данные выводятся в реальном времени в отдельный блок без перезагрузки страницы (использовать `addEventListener('submit', …)` и методы DOM).
