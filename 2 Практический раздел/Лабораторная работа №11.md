# Лабораторная работа № 11  
Создание объектов, назначение свойств и методов объектов. Использование пользовательских объектов в сценариях на языке JavaScript

## Цель и задачи

1. Научиться определять конструкторы и создавать объекты в JavaScript.  
2. Понять принципы назначения свойств и методов объектам.  
3. Овладеть использованием пользовательских объектов в простых сценариях.  
4. Развить навыки отладки и тестирования пользовательских объектов.

## Теоретическая часть

| Понятие | Краткое определение |
|---------|---------------------|
| **Объект** | Сочетание свойств и методов, объединенных в единую сущность. |
| **Свойство** | Член объекта, представляющий данные. |
| **Метод** | Функция, являющаяся свойством объекта. |
| **Конструктор** | Функция, создающая экземпляры объектов и инициализирующая их свойства. |
| **this** | Ключевое слово, указывающее на текущий объект. |
| **Инкапсуляция** | Сокрытие внутреннего состояния объекта от внешнего доступа. |
| **Наследование** | Способ получения свойств и методов от другого объекта (через прототипы). |
| **Метод `Object.prototype`** | Базовый объект, от которого наследуются все объекты. |
| **Метод `Object.create`** | Создает новый объект с заданным прототипом. |
| **`prototype`** | Свойство конструктора, через которое наследуются методы. |

## Ход выполнения работы

1. **Создание конструктора `Person`.**  
   Пояснение: определяем функцию, которая инициализирует свойства `name`, `age` и `gender`.  
   ```js
   function Person(name, age, gender) {
       this.name = name;
       this.age = age;
       this.gender = gender;
   }
   ```

2. **Добавление метода `greet` к прототипу `Person`.**  
   Пояснение: метод выводит приветствие, используя свойства объекта.  
   ```js
   Person.prototype.greet = function() {
       console.log(`Привет, меня зовут ${this.name}, мне ${this.age} лет.`);
   };
   ```

3. **Создание экземпляров `Person`.**  
   Пояснение: используем оператор `new`.  
   ```js
   const alice = new Person('Алиса', 30, 'женский');
   const bob   = new Person('Боб', 25, 'мужской');
   ```

4. **Вызов методов у экземпляров.**  
   ```js
   alice.greet(); // Привет, меня зовут Алиса, мне 30 лет.
   bob.greet();   // Привет, меня зовут Боб, мне 25 лет.
   ```

5. **Создание объекта через `Object.create`.**  
   Пояснение: создаём объект с прототипом `Person.prototype` и добавляем свойства вручную.  
   ```js
   const charlie = Object.create(Person.prototype);
   charlie.name = 'Чарли';
   charlie.age = 28;
   charlie.gender = 'мужской';
   charlie.greet();
   ```

6. **Проверка принадлежности метода к прототипу.**  
   Пояснение: используем `hasOwnProperty`.  
   ```js
   console.log('greet' in alice);            // true
   console.log(alice.hasOwnProperty('greet')); // false
   ```

7. **Тестирование и отладка.**  
   Пояснение: проверяем, что все свойства и методы работают корректно, выводим результаты в консоль.  

## Контрольные вопросы / задания для самопроверки

3. Что произойдет, если вызвать метод `greet` у объекта, созданного через `Object.create(Person.prototype)`?  
4. Как добавить новый метод `ageInMonths` к прототипу `Person` и использовать его?  
5. Какие преимущества имеет использование прототипов перед прямым определением методов в конструкторе?

8. Как изменить прототип уже существующего объекта?
