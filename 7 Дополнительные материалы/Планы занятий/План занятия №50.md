# Лабораторная работа № 25. Установка соединения с БД MySQL. Передача запроса к БД и обработка результата

## Цель занятия
1. Научить студентов установить и настроить соединение с базой данных MySQL из Python.  
2. Познакомить студентов с отправкой SQL‑запросов и обработкой результатов в виде таблиц и отдельных записей.

## Задачи занятия
- Освоить работу с библиотекой `mysql-connector-python`.  
- Создать и настроить соединение с удалённой/локальной базой MySQL.  
- Написать и выполнить простые SELECT, INSERT, UPDATE и DELETE запросы.  
- Обработать результат запросов в виде списка словарей и вывести его в удобочитаемом виде.  
- Проверить корректность выполнения операций с помощью тестовых данных.

## План занятия (90 минут)

- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- Изучение нового материала  
  - Установка и настройка драйвера MySQL в Python — 15 мин  
  - Создание и проверка соединения с БД — 10 мин  
  - Формирование и выполнение SQL‑запросов — 15 мин  
- Закрепление  
  - Практическое задание: работа с таблицей «students» — 20 мин  
- Итог занятия — 5 мин  

## Содержание этапов

**Организационный момент**  
Краткое введение, проверка присутствия, раздача материалов.  

**Актуализация знаний**  
Повторение основных понятий: БД, таблицы, SQL‑операторы, типы данных.  

**Установка и настройка драйвера MySQL в Python**  
Пошаговая установка `mysql-connector-python`, проверка версии, подключение к репозиторию.  

**Создание и проверка соединения с БД**  
Написание скрипта, открытие соединения, проверка статуса, закрытие.  

**Формирование и выполнение SQL‑запросов**  
Построение SELECT, INSERT, UPDATE, DELETE запросов, работа с курсорами, обработка ошибок.  

**Практическое задание: работа с таблицей «students»**  
Студенты создают таблицу, заполняют данными, выполняют CRUD‑операции и выводят результаты.  

**Итог занятия**  
Краткое подведение итогов, ответы на вопросы, разъяснение дальнейших шагов.  

## Оборудование и материалы
- Компьютер со стабильным интернет‑соединением.  
- IDE/текстовый редактор (PyCharm, VS Code, Sublime Text).  
- Доступ к MySQL‑серверу (локальный или удалённый).  
- Установленный Python 3.9+ и пакет `mysql-connector-python`.  
- Схема базы данных (DDL‑скрипты) и пример данных.  
- Проектор/экран для демонстрации.  

## Контроль и оценивание
- **Тестовые вопросы** (5 мин): 5 вопросов о принципах подключения и типах запросов.  
- **Практическое задание**: выполнение CRUD‑операций и вывод результата.  
- **Критерии оценки**: корректность подключения, правильность SQL‑запросов, корректность обработки результатов, чистота кода (комментарии, читаемость).  

## Домашнее задание
1. Написать скрипт, который извлекает из таблицы «students» все записи, где возраст > 20, и сохраняет их в CSV‑файл.  
2. Добавить в таблицу новый столбец `email` и обновить значения для всех записей.
