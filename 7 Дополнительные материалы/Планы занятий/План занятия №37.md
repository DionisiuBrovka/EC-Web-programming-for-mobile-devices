# Лабораторная работа № 20: Обработка событий с использованием jQuery

## Цель занятия  
1. Научиться обрабатывать события и создавать анимацию с помощью jQuery и jQuery UI.  
2. Освоить работу с пользовательскими формами при помощи jQuery.  

## Задачи занятия  
- Ознакомиться с синтаксисом обработки событий в jQuery.  
- Создать простую анимацию методом `.animate()`.  
- Использовать виджеты jQuery UI (datepicker, slider).  
- Реализовать валидацию и отправку пользовательской формы через jQuery.  
- Подготовить рабочий проект, демонстрирующий все изученные возможности.  

## План занятия (90 минут)  

- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- Изучение нового материала: обработка событий — 20 мин  
- Изучение нового материала: анимация и виджеты — 20 мин  
- Практическое задание: форма + анимация — 20 мин  
- Итог занятия — 5 мин  

## Содержание этапов  

- **Организационный момент**  
  Приветствие, проверка присутствия, краткое напоминание о правилах лабораторных работ.  

- **Актуализация знаний**  
  Быстрый опрос: кто уже использовал jQuery, какие события знакомы, какие виджеты использовались.  

- **Изучение нового материала: обработка событий**  
  Теоретический разбор синтаксиса `on()`, `click()`, `hover()` и их применение на примерах.  

- **Изучение нового материала: анимация и виджеты**  
  Демонстрация метода `.animate()`, настройка длительности и easing. Обзор виджетов jQuery UI: `datepicker`, `slider`, `draggable`.  

- **Практическое задание: форма + анимация**  
  Студенты создают страницу с формой, которая при отправке анимирует элементы и выводит сообщение. Используются `$.ajax()` и валидация через jQuery.  

- **Итог занятия**  
  Подведение итогов, ответы на вопросы, раздача чек-листа для проверки выполненной работы.  

## Оборудование и материалы  

- Компьютеры со стабильным интернет‑подключением.  
- IDE/текстовый редактор (VS Code, Sublime Text, Atom).  
- Браузер Chrome/Firefox.  
- Установленные библиотеки jQuery и jQuery UI (CDN или локально).  
- Проектор/экран для демонстрации.  

## Контроль и оценивание  

- **Вопросы устного опроса** (5 мин): как вызвать событие `click`, какие параметры принимает `.animate()`.  
- **Краткий тест** (5 мин): выбор правильных вариантов кода.  
- **Проверка лабораторной работы**: наличие анимации, корректная работа виджетов, валидность формы.
