## Пример решения (вариант 1, задание 2)  

```python
def prime_factors(n):
    """Возвращает список простых множителей числа n."""
    i = 2
    factors = []
    while i * i <= n:
        while n % i == 0:
            factors.append(i)
            n //= i
        i += 1
    if n > 1:
        factors.append(n)
    return factors

# Тест
print(prime_factors(60))  # [2, 2, 3, 5]
```

**Пояснение:**  
- Цикл `while i * i <= n` обеспечивает O(√n) времени.  
- Внутренний цикл `while n % i == 0` делит `n` на найденный простый делитель столько раз, сколько его входит в разложение.  
- После завершения внешнего цикла, если `n > 1`, то остаток сам по себе является простым множителем.
