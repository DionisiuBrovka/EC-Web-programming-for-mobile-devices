## Ключевые понятия  
| Термин | Определение |
|--------|-------------|
| **Cookie** | Маленькая текстовая запись, сохраняемая в браузере, передающая серверу данные о сессии. |
| **Кэширование** | Сохранение промежуточных результатов для ускорения последующих запросов. |
| **Буферизация** | Хранение данных в временной памяти до их окончательной обработки. |
| **Хэширование** | Преобразование данных фиксированной длины в уникальный «отпечаток» (hash). |
| **HTTP‑аутентификация** | Механизм проверки подлинности пользователя в рамках протокола HTTP. |
| **Сессия** | Состояние взаимодействия пользователя с приложением, обычно хранится на сервере. |
| **OAuth** | Открытый стандарт авторизации, позволяющий сторонним приложениям получить доступ к ресурсам без раскрытия пароля. |
| **Ключ шифрования** | Строка, используемая для кодирования и декодирования данных. |
| **Кэш‑контроль** | Заголовки HTTP, управляющие поведением кэша. |
| **Токен доступа** | Строка, предоставляемая OAuth, позволяющая доступ к защищённым ресурсам. |

## Основное содержание  

### 1. Cookie (15 мин)  
- Структура cookie: `Name=Value; Expires; Path; Domain; Secure; HttpOnly`.  
- Синтаксис установки и чтения в Python с использованием `http.cookies`.  
```python
from http.cookies import SimpleCookie

cookie = SimpleCookie()
cookie['session_id'] = 'abc123'
cookie['session_id']['path'] = '/'
cookie['session_id']['expires'] = 'Wed, 09 Jun 2025 10:18:14 GMT'
print(cookie.output())
```

### 2. Управление кэшированием (20 мин)  
- Заголовки `Cache-Control`, `ETag`, `If-None-Match`.  
- Пример кэширования статических файлов в Flask.  
```python
from flask import Flask, send_file, request, abort
app = Flask(__name__)

@app.route('/static/<path:filename>')
def static_file(filename):
    etag = f"\"{hash(filename)}\""
    if_none_match = request.headers.get('If-None-Match')
    if if_none_match == etag:
        return '', 304
    return send_file(f'static/{filename}', conditional=True)
```

### 4. Хэширование (10 мин)  
- Алгоритмы `MD5`, `SHA1`, `SHA256`.  
- Применение хэш-функций для проверки целостности файлов.  
```python
import hashlib

def file_hash(path, algo='sha256'):
    h = hashlib.new(algo)
    with open(path, 'rb') as f:
        while chunk := f.read(8192):
            h.update(chunk)
    return h.hexdigest()
```

### 5. HTTP‑аутентификация (15 мин)  
- Basic, Digest, Bearer.  
- Реализация Basic Auth в Flask.  
```python
from flask import request, Response

@app.route('/protected')
def protected():
    auth = request.authorization
    if not auth or auth.username != 'admin' or auth.password != 'secret':
        return Response('Login required', 401,
                        {'WWW-Authenticate': 'Basic realm="Login"'})
    return 'You are authenticated'
```

### 6. Управление сессиями (15 мин)  
- Серверные и клиентские сессии.  
- Хранение сессионных данных в Redis.  
```python
import redis
import uuid

r = redis.Redis()
def create_session(user_id):
    session_id = str(uuid.uuid4())
    r.set(session_id, user_id, ex=3600)  # 1 час
    return session_id
```

### 7. Протокол OAuth (20 мин)  
- Архитектура: Resource Owner, Client, Authorization Server, Resource Server.  
- Пошаговый поток авторизации (Authorization Code Flow).  
- Создание простого OAuth‑серверного клиента в Python.  
```python
import requests

client_id = 'your_client_id'
client_secret = 'your_client_secret'
redirect_uri = 'https://localhost/callback'

auth_url = f"https://provider.com/oauth/authorize?response_type=code&client_id={client_id}&redirect_uri={redirect_uri}"
print(f'Перейдите по ссылке: {auth_url}')

# После получения кода:
code = input('Введите код: ')
token_resp = requests.post('https://provider.com/oauth/token',
    data={'grant_type': 'authorization_code',
          'code': code,
          'redirect_uri': redirect_uri,
          'client_id': client_id,
          'client_secret': client_secret})
access_token = token_resp.json()['access_token']
```
