# Лабораторная работа № 25. Установка соединения с БД MySQL. Передача запроса к БД и обработка результата

## Цель и задачи

1. Научиться устанавливать соединение с MySQL‑сервером из Python.  
2. Изучить основные параметры соединения и методы библиотеки `mysql-connector-python`.  
3. Научиться формировать и выполнять SQL‑запросы, обрабатывать результаты.  
4. Оценить корректность выполнения запроса и понять, как работать с ошибками.

## Теоретическая часть

| Понятие | Определение |
|--------|------------|
| **База данных (БД)** | Организованная коллекция данных, доступная с помощью систем управления базами данных (СУБД). |
| **СУБД MySQL** | Открытая реляционная СУБД, реализующая SQL‑стандарт и поддерживающая множество драйверов. |
| **Клиент‑серверная архитектура** | Модель, где клиент (приложение) посылает запросы серверу, который хранит и обрабатывает данные. |
| **Драйвер** | Библиотека, реализующая протокол взаимодействия между приложением и СУБД. |
| **`mysql.connector.connect()`** | Функция, создающая объект соединения с БД. |
| **`cursor()`** | Метод соединения, создающий объект курсора для выполнения запросов. |
| **`execute()`** | Метод курсора, который исполняет один SQL‑запрос. |
| **`fetchone()` / `fetchall()`** | Методы курсора, возвращающие одну строку или все строки результата запроса. |
| **Параметрический запрос** | Запрос, в котором параметры передаются отдельно от SQL‑строки, защищая от SQL‑инъекций. |
| **Контекстный менеджер** (`with`) | Удобный способ автоматически закрывать соединения и курсоры. |

## Ход выполнения работы

1. **Подготовка окружения**  
   - Установить библиотеку `mysql-connector-python`.  
     ```bash
     pip install mysql-connector-python
     ```
   - Убедиться, что MySQL‑сервер запущен и доступен по сети.

2. **Создание подключения**  
   ```python
   import mysql.connector

   conn = mysql.connector.connect(
       host='localhost',
       user='root',
       password='your_password',
       database='testdb'
   )
   ```
   - `host`: адрес сервера.  
   - `user`, `password`: учетные данные.  
   - `database`: имя выбранной БД.

3. **Создание курсора**  
   ```python
   cursor = conn.cursor()
   ```

4. **Формирование и выполнение запроса**  
   - Пример: создать таблицу.  
     ```python
     cursor.execute("""
         CREATE TABLE IF NOT EXISTS students (
             id INT AUTO_INCREMENT PRIMARY KEY,
             name VARCHAR(50),
             age INT
         )
     """)
     conn.commit()
     ```
   - Пример: вставить данные.  
     ```python
     add_student = ("INSERT INTO students (name, age) VALUES (%s, %s)")
     data_student = ('Alice', 21)
     cursor.execute(add_student, data_student)
     conn.commit()
     ```

5. **Запрос данных**  
   ```python
   cursor.execute("SELECT * FROM students")
   rows = cursor.fetchall()
   for row in rows:
       print(row)
   ```

6. **Обработка ошибок**  
   ```python
   try:
       cursor.execute("INVALID SQL")
   except mysql.connector.Error as err:
       print(f"Error: {err}")
   ```

7. **Закрытие ресурсов**  
   ```python
   cursor.close()
   conn.close()
   ```

8. **Экспериментальная часть**  
   - Добавить несколько строк, выполнить `SELECT` с условием `WHERE age > 20`.  
   - Сравнить результаты до и после изменения данных.  
   - Сохранить вывод в файл `result.txt`.
