# Серверные и клиентские сценарии. Основы синтаксиса JavaScript. Анализ ОКР№1

## Цели и задачи лекции
1. Понять различия между серверными и клиентскими сценариями в веб-разработке.  
2. Освоить базовый синтаксис JavaScript, необходимый для работы на клиенте.  
3. Проанализировать ОКР№1 и выявить ключевые требования к реализации.  

## Ключевые понятия
- **Клиентский сценарий** – код, исполняемый в браузере пользователя.  
- **Серверный сценарий** – код, исполняемый на веб‑сервере.  
- **DOM** – Document Object Model, структура HTML‑страницы.  
- **AJAX** – асинхронный обмен данными между клиентом и сервером.  
- **Node.js** – среда выполнения JavaScript на сервере.  
- **Callback** – функция, передаваемая в другую функцию для выполнения после завершения операции.  
- **Promise** – объект, представляющий результат асинхронной операции.  
- **ES6** – шестая редакция ECMAScript, включающая стрелочные функции, шаблонные строки и др.  

## Основное содержание

### 1. Введение в клиентские и серверные сценарии (10 мин)
- Отличия по месту выполнения, доступу к ресурсам, безопасности.  
- Примеры типовых задач для каждого типа сценария.

### 2. Базовый синтаксис JavaScript (25 мин)
- Переменные и типы данных (`var`, `let`, `const`).  
- Операторы, условные конструкции, циклы.  
- Функции и области видимости.  

```js
const name = 'Alice';
let counter = 0;

function greet(user) {
  return `Hello, ${user}!`;
}

for (let i = 0; i < 5; i++) {
  console.log(greet(name));
}
```

### 3. Работа с DOM (15 мин)
- Выбор элементов (`querySelector`, `getElementById`).  
- Манипуляции с атрибутами и содержимым.  
- Обработчики событий.  

```js
document.getElementById('btn').addEventListener('click', () => {
  document.querySelector('.output').textContent = 'Clicked!';
});
```

### 4. Асинхронность и AJAX (20 мин)
- XMLHttpRequest и Fetch API.  
- Преимущества промисов и `async/await`.  

```js
async function fetchData(url) {
  const response = await fetch(url);
  const data = await response.json();
  return data;
}
```

### 5. Серверные сценарии на Node.js (20 мин)
- Установка и запуск простого сервера Express.  
- Маршрутизация и обработка запросов.  

```js
const express = require('express');
const app = express();

app.get('/api', (req, res) => {
  res.json({ message: 'Hello from server' });
});

app.listen(3000, () => console.log('Server running'));
```

### 6. Анализ ОКР№1 (10 мин)
- Техническое задание: создание клиент‑серверного приложения с CRUD‑операциями.

## Выводы по лекции
- Клиентские сценарии управляют интерфейсом и взаимодействием пользователя, серверные — логикой и хранением данных.  
- JavaScript предоставляет универсальный инструмент как для клиентской, так и для серверной разработки.  
- Понимание асинхронных паттернов и работы с DOM критично для создания динамичных веб‑приложений.
