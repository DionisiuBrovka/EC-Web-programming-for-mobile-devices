# Технология AJAX. Формат JSON и способы обработки данных в клиентских приложениях

## Цель занятия
1. Познакомить студентов с концепцией AJAX и форматом JSON как основными инструментами асинхронного взаимодействия клиент–сервер.  
2. Научить применять библиотеки Python (`aiohttp`, `requests`, `json`) для отправки запросов и обработки JSON‑ответов в клиентских приложениях.

## Задачи занятия
- Объяснить принцип работы AJAX и его роль в современных веб‑приложениях.  
- Рассмотреть структуру и синтаксис JSON, преимущества и ограничения.  
- Показать, как формировать и отправлять асинхронные HTTP‑запросы из Python.  
- Научить парсить и сериализовать данные в формате JSON.  
- Предоставить практическое задание по созданию простого клиентского приложения с использованием AJAX‑подобного взаимодействия.

## План занятия (90 минут)

- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- Введение в AJAX — 15 мин  
- Формат JSON: синтаксис и правила — 15 мин  
- Асинхронные запросы в Python (aiohttp) — 20 мин  
- Парсинг и сериализация JSON — 10 мин  
- Закрепление (мини‑проект) — 10 мин  
- Итог занятия и обратная связь — 5 мин  

## Содержание этапов

- **Организационный момент** – приветствие, проверка присутствия, краткое напоминание о предыдущем занятии.  
- **Актуализация знаний** – быстрый опрос: какие методы HTTP известны студентам, что они знают о клиент–серверном взаимодействии.  
- **Введение в AJAX** – объяснение асинхронных запросов, их преимущества, примеры из реальных приложений (Google Maps, соцсети).  
- **Формат JSON: синтаксис и правила** – основные типы данных, правила именования ключей, особенности сериализации/десериализации.  
- **Асинхронные запросы в Python (aiohttp)** – установка библиотеки, структура запроса, обработка ответа, пример кода.  
- **Парсинг и сериализация JSON** – использование модуля `json`, работа с вложенными структурами, ошибки при парсинге.  
- **Закрепление (мини‑проект)** – в группах студенты создают небольшое приложение, которое запрашивает данные из публичного API и выводит их в консоль.  
- **Итог занятия и обратная связь** – подведение итогов, ответы на вопросы, раздача чек-листа по ключевым моментам.

## Оборудование и материалы

- Вопросы на доске: 5–7 вопросов по теории AJAX и JSON.

- Оценка выполнения мини‑проекта (проверка кода, корректность запросов, правильность обработки JSON).

## Домашнее задание
1. Реализовать скрипт на Python, который получает список последних новостей из публичного API (например, NewsAPI) и сохраняет заголовки в JSON‑файл.  
2. Написать краткое отчёт (не более 200 слов) о том, какие сложности возникли при работе с асинхронными запросами и как они были решены.
