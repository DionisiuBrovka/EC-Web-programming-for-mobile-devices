# Основные методы библиотеки jQuery

## Цель и задачи занятия
- Быстро вспомнить, что такое jQuery и зачем он нужен.
- Познакомиться с четырьмя базовыми методами: `$(selector)`, `.on()`, `.css()`, `.append()`.
- Научиться применять их в простых, реальных сценариях.
- Уверенно писать маленькие скрипты без «заголовных» ошибок.

## Почему это важно?
В современных проектах часто требуется быстро менять стиль или поведение элементов на странице, не трогая основной HTML. jQuery делает это простым и понятным, а знания его методов пригодятся и в работе с Python‑фреймворками (Django, Flask) для создания интерактивных UI.

## Основные понятия и идеи
| Термин | Что это значит |
|--------|----------------|
| **DOM** | Структура документа, которую браузер отображает. |
| **Селектор** | Краткая строка, указывающая, какие элементы выбрать. |
| **Event** | Событие (клик, ввод, загрузка). |
| **Callback** | Функция, выполняемая после события. |
| **Метод** | Функция, применяемая к объекту jQuery. |
| **Chaining** | Последовательное вызов методов без промежуточных переменных. |
| **AJAX** | Асинхронный запрос к серверу без перезагрузки страницы. |

## Объяснение темы

### [10 мин] Что такое jQuery и как подключить
```html
<script src="https://code.jquery.com/jquery-3.7.0.min.js"></script>
```
- jQuery – библиотека JavaScript, упрощающая работу с DOM.
- Подключаем через CDN или скачиваем локально.

### [15 мин] Выбор элементов – `$(selector)`
```js
const btn = $('#myButton');      // id
const items = $('.list-item');   // класс
const first = $('p:first');      // первый параграф
```
**Игра**: Найдите в странице элемент с id `header` и выведите его текст в консоль.

### [15 мин] Обработка событий – `.on()`
```js
$('#myButton').on('click', function() {
  alert('Кнопка нажата!');
});
```
- `.on('event', handler)` – привязываем событие.
- Можно использовать `hover`, `keyup`, `submit` и др.

**Попробуй сам!**  
Что произойдёт, если заменить `click` на `dblclick`?  
> Подсказка: двойной клик вызывает событие `dblclick`.

### [10 мин] Изменение стилей – `.css()`
```js
$('#myButton').css({
  backgroundColor: 'green',
  color: 'white',
  padding: '10px'
});
```
- Можно передавать объект со свойствами.
- Или строку: `.css('color', 'red')`.

**Задача**: Сделай кнопку красной при наведении с помощью CSS в jQuery.

### [15 мин] Добавление контента – `.append()`
```js
$('#list').append('<li>Новый пункт</li>');
```
- Добавляет HTML внутри выбранного элемента.
- Полезно для динамических списков.

**Мини‑игра**: При нажатии кнопки добавляй новый элемент в список.  
> Подсказка: используйте `$('#add').on('click', ...)`.

## Мини-задания по ходу
- **Задание 1**: Выбери элемент с классом `.title` и поменяй его цвет на синий.
- **Задание 2**: Добавь обработчик клика к кнопке `#hideBtn`, который скрывает элемент `#content`.
- **Задание 3**: При загрузке страницы выведи в консоль количество `<li>` в списке `#menu`.

## Итог и выводы
jQuery позволяет быстро и лаконично работать с DOM: выбирать элементы, реагировать на события, менять стили и добавлять контент. Эти навыки пригодятся в любой веб‑разработке, даже если вы позже будете писать серверный код на Python. Главное – практиковаться, экспериментировать и не бояться «поиграть» с элементами страницы.
