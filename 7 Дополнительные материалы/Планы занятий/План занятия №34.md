# Лабораторная работа № 19. Обработка данных в формате JSON на стороне клиента

## Цель занятия  
1. Научить студентов загружать, парсить и манипулировать JSON‑данными в браузере с помощью JavaScript.  
2. Познакомить с основными методами работы с объектами и массивами, полученными из JSON, и с API `fetch`.

## Задачи занятия  
- Разобрать структуру JSON‑файла и его свойства.  
- Изучить способы загрузки JSON‑данных через `fetch`.  
- На практике реализовать чтение, фильтрацию и вывод данных в DOM.  
- Оценить качество кода и понимание концепций обработки данных.

## План занятия  
- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- Изучение нового материала — 25 мин  
- Закрепление — 20 мин  
- Итог занятия — 5 мин  

## Содержание этапов  

**Организационный момент**  
Приветствие, проверка наличия ноутбуков, подключение к интернету, раздача ссылок на репозиторий с шаблоном проекта.  

**Актуализация знаний**  
Краткая викторина по структуре JSON, основам JavaScript‑объектов и синтаксису `fetch`.  

**Изучение нового материала**  
Теоретическое введение в `fetch`, методы `json()`, `then()`, `catch()`. Демонстрация примера загрузки данных с публичного API и их вывода в таблицу.  

**Закрепление**  
Постановка практической задачи: студентам необходимо загрузить данные о погоде, отфильтровать записи за последние 7 дней и вывести в таблицу с подсчетом средней температуры. Время работы в паре, преподаватель консультирует.  

**Итог занятия**  
Обсуждение результатов, ответы на вопросы, раздача чек‑листов по работе с JSON.  

## Оборудование и материалы  
- ПК с доступом в интернет.  
- Браузер (Chrome/Firefox).  
- IDE/текстовый редактор (VS Code, Sublime, Atom).  
- Проектор/дисплей для демонстрации.  
- Репозиторий GitHub с шаблоном проекта (доступ по ссылке).  
- Список рекомендуемых ресурсов: MDN Web Docs, JSONPlaceholder API.  

## Контроль и оценивание  
- Короткие вопросы на каждом этапе (5 мин).  
- Оценка выполненной задачи: корректность кода, использование `fetch`, обработка ошибок, чистота оформления (20 баллов).  
- Самооценка: студент заполняет форму «Что я узнал» после занятия.  

## Домашнее задание  
1. Реализовать загрузку данных о пользователях из `https://jsonplaceholder.typicode.com/users` и вывести их в виде списка с именами и email.  
2. Добавить кнопку «Фильтр», которая при клике показывает только пользователей из города “South Christy”.
