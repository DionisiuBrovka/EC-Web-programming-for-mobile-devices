# Основные методы библиотеки jQuery

## Цели и задачи лекции
- Познакомить студентов с ключевыми методами jQuery, используемыми для манипуляции DOM.
- Показать практические примеры применения методов в реальных проектах.
- Развить навыки быстрой разработки интерактивных веб‑страниц.
- Сформировать базу знаний, необходимую для дальнейшего изучения JavaScript и front‑end разработки.

## Ключевые понятия
| Термин | Определение |
|--------|-------------|
| jQuery | JavaScript‑библиотека, упрощающая работу с DOM, событиями и AJAX. |
| Декларативный подход | Способ описания *что* нужно сделать, а не *как*. |
| Селектор | Выборка элементов DOM по CSS‑правилам. |
| Chainable | Возможность объединять несколько методов в цепочку. |
| AJAX | Асинхронный обмен данными между клиентом и сервером. |
| Event delegation | Делегирование событий через родительский элемент. |
| Deferred | Объект, представляющий отложенное выполнение кода (Promise‑подобный). |
| Animation | Анимация CSS‑свойств через методы jQuery. |
| Data API | Методы для работы с пользовательскими данными элементов. |
| Plugin | Расширение jQuery, добавляющее новые методы. |

## Основное содержание

### 1. Введение в jQuery (10 мин)
- История возникновения и преимущества.
- Установка и подключение скрипта через CDN.

```html
<script src="https://code.jquery.com/jquery-3.7.0.min.js"></script>
```

### 2. Селекторы и базовая работа с DOM (15 мин)
- Выбор элементов: `$('#id')`, `$('.class')`, `$('tag')`.
- Методы доступа к свойствам: `.text()`, `.html()`, `.val()`.
- Изменение стилей: `.css()`, `.addClass()`, `.removeClass()`.

```javascript
$('#title').text('Добро пожаловать!');
$('.btn').addClass('active').css('color', 'red');
```

### 3. Манипуляции элементами (15 мин)
- Добавление/удаление: `.append()`, `.prepend()`, `.remove()`, `.empty()`.
- Перемещение: `.insertBefore()`, `.insertAfter()`.
- Работа с атрибутами: `.attr()`, `.prop()`.

```javascript
$('#list').append('<li>Новый пункт</li>');
$('#link').attr('href', 'https://example.com');
```

### 4. Обработка событий (15 мин)
- Простейшие события: `.click()`, `.hover()`.
- Делегирование: `.on('click', 'a', handler)`.
- Сброс события: `.off()`.

```javascript
$(document).on('click', '.btn', function() {
  alert('Кнопка нажата');
});
```

### 5. Анимация и эффекты (10 мин)
- Встроенные эффекты: `.hide()`, `.show()`, `.fadeIn()`, `.slideUp()`.
- Пользовательские анимации: `.animate()`.

```javascript
$('#panel').slideToggle(400);
$('#box').animate({ left: '200px', opacity: 0.5 }, 600);
```

### 6. AJAX и асинхронность (10 мин)
- `$.ajax()`, `$.get()`, `$.post()`.
- Обработка ответа: `.done()`, `.fail()`, `.always()`.
- Пример загрузки данных без перезагрузки страницы.

```javascript
$.get('/api/data', function(data) {
  $('#output').text(JSON.stringify(data));
});
```

### 7. Создание простого плагина (10 мин)
- Структура плагина: `$.fn.myPlugin = function(options) { ... }`.
- Инициализация и опции по умолчанию.

```javascript
$.fn.highlight = function(color) {
  return this.css('backgroundColor', color || 'yellow');
};
$('p').highlight('lightblue');
```

## Примеры и иллюстрации
- **Пример 1:** Переключатель темы (светлая/темная) через `.toggleClass()`.

```javascript
$('#themeBtn').click(function() {
  $('body').toggleClass('dark-theme');
});
```

- **Пример 2:** Формирование списка из данных AJAX.

```javascript
$.get('/api/items', function(items) {
  const $ul = $('#items');
  items.forEach(item => {
    $ul.append(`<li>${item.name}</li>`);
  });
});
```

## Выводы по лекции
- jQuery предоставляет мощный, но простой набор методов для работы с DOM, событиями и AJAX.
- Понимание селекторов и chainable‑методов облегчает написание чистого, лаконичного кода.
- Практические навыки работы с анимациями и плагинами позволяют быстро расширять функциональность веб‑страниц.
- Умение использовать AJAX обеспечивает динамический контент без перезагрузки страниц.
