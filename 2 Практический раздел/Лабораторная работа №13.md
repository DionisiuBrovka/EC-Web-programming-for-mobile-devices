# Лабораторная работа № 13  
**Использование свойств и методов объекта `window` для организации взаимодействия с программой просмотра веб‑страниц, определение местоположения пользователя**

---

## Цель и задачи

1. Познакомиться с объектом `window` в веб‑приложениях и его основными свойствами и методами.  
2. Научиться использовать `window` для управления поведением браузера и взаимодействия пользователя с программой.  
3. Изучить способы определения геолокации пользователя через API `navigator.geolocation`.  
4. Практически применить полученные знания, написав скрипт, который:  
   - отображает текущую страницу в полноэкранном режиме;  
   - открывает и закрывает всплывающее окно;  
   - выводит координаты пользователя (если разрешено).

---

## Теоретическая часть

| Понятие | Определение |
|---------|-------------|
| **`window`** | Глобальный объект, представляющий окно браузера. Содержит свойства и методы, управляющие окном и его содержимым. |
| **`window.open()`** | Метод, открывающий новое окно или вкладку браузера с указанным URL. |
| **`window.close()`** | Метод, закрывающий текущее окно (если оно было открыто скриптом). |
| **`window.fullscreenElement`** | Свойство, возвращающее элемент, занимающий весь экран, либо `null`. |
| **`element.requestFullscreen()`** | Метод, переходящий элемент в полноэкранный режим. |
| **`document.exitFullscreen()`** | Метод, выходящий из полноэкранного режима. |
| **`navigator.geolocation`** | Объект, предоставляющий доступ к API геолокации. |
| **`navigator.geolocation.getCurrentPosition()`** | Метод, запрашивающий текущую позицию пользователя. |
| **`Position`** | Объект, содержащий координаты (`coords.latitude`, `coords.longitude`) и дополнительную информацию. |
| **CORS** | Политика совместного использования ресурсов, влияющая на доступ к API из сторонних доменов. |
| **HTTPS** | Протокол, необходимый для работы API геолокации в большинстве браузеров. |

## Ход выполнения работы

1. **Подготовка проекта**  
   - Создать каталог `lab13` и файл `index.html`.  
   - Добавить в `<head>` подключение CSS‑стилей (опционально).  
   ```html
   <!DOCTYPE html>
   <html lang="ru">
   <head>
       <meta charset="UTF-8">
       <title>Лабораторная работа №13</title>
   </head>
   <body>
       <h1>Лабораторная работа №13</h1>
   </body>
   </html>
   ```

2. **Создание кнопок управления окном**  
   - Добавить кнопки “Полноэкранный режим”, “Открыть окно”, “Закрыть окно”.  
   ```html
   <button id="fullscreen">Полноэкранный режим</button>
   <button id="open">Открыть окно</button>
   <button id="close">Закрыть окно</button>
   ```

3. **Реализация полноэкранного режима**  
   - В JS добавить обработчик события `click` для кнопки `fullscreen`.  
   ```javascript
   const btnFull = document.getElementById('fullscreen');
   btnFull.addEventListener('click', () => {
       if (!document.fullscreenElement) {
           document.documentElement.requestFullscreen();
       } else {
           document.exitFullscreen();
       }
   });
   ```

4. **Открытие и закрытие нового окна**  
   - При клике на `open` открывать новое окно с `about:blank`.  
   - При клике на `close` закрывать ранее открытое окно.  
   ```javascript
   let childWindow = null;
   const btnOpen = document.getElementById('open');
   const btnClose = document.getElementById('close');

   btnOpen.addEventListener('click', () => {
       if (!childWindow || childWindow.closed) {
           childWindow = window.open('about:blank', '_blank', 'width=400,height=300');
       }
   });

   btnClose.addEventListener('click', () => {
       if (childWindow && !childWindow.closed) {
           childWindow.close();
       }
   });
   ```

5. **Получение геолокации пользователя**  
   - Добавить кнопку “Получить координаты”.  
   ```html
   <button id="geo">Получить координаты</button>
   <pre id="output"></pre>
   ```
   - Реализовать запрос координат, проверку разрешения и вывод результата.  
   ```javascript
   const btnGeo = document.getElementById('geo');
   const output = document.getElementById('output');

   btnGeo.addEventListener('click', () => {
       if ('geolocation' in navigator) {
           navigator.geolocation.getCurrentPosition(
               (pos) => {
                   const { latitude, longitude } = pos.coords;
                   output.textContent = `Широта: ${latitude}\nДолгота: ${longitude}`;
               },
               (err) => {
                   output.textContent = `Ошибка: ${err.message}`;
               },
               { enableHighAccuracy: true, timeout: 5000 }
           );
       } else {
           output.textContent = 'Геолокация не поддерживается';
       }
   });
   ```

6. **Тестирование**  
   - Запустить страницу в HTTPS‑среде (например, через `http-server --ssl`).  
   - Проверить работу каждой функции: полноэкранный режим, открытие/закрытие окна, вывод координат.  
   - Зафиксировать результаты в отчёте.
