# Блочная модель документа

## Цели и задачи лекции
- Понять структуру и принципы работы блочной модели документа в современных текстовых и графических редакторах.  
- Освоить основные типы блоков и их свойства.  
- Научиться создавать и модифицировать документы, используя блочную модель, на примере Python‑скриптов.  
- Приобрести навыки применения блочной модели в автоматизации документооборота.

## Ключевые понятия
| Термин | Определение |
|--------|-------------|
| Блок | Самостоятельный элемент документа, имеющий собственные границы, свойства и содержимое. |
| Иерархия блоков | Структура вложенности блоков, создающая дерево документа. |
| Структурный блок | Блок, определяющий структуру документа (заголовок, параграф, таблица). |
| Контентный блок | Блок, содержащий пользовательский контент (текст, изображения, код). |
| Метаданные блока | Информация о блоке (тип, идентификатор, свойства). |
| Ссылка на блок | Уникальный идентификатор, позволяющий ссылаться на блок из другого места документа. |
| Префикс блока | Специальный символ или набор символов, определяющий тип блока. |
| Схема документа | Формальное описание возможных блоков и их связей. |
| Форматирование | Применение стилей к блокам (жирный, курсив, списки). |
| Инструментальная модель | Система инструментов, позволяющая создавать, редактировать и управлять блоками. |

## Основное содержание

### 1. Введение в блочную модель (15 мин)
- История развития блочных моделей в текстовых редакторах.  
- Сравнение с линейной моделью (строки, абзацы).  
- Преимущества: гибкость, масштабируемость, совместная работа.

### 2. Структура блока (20 мин)
- Компоненты блока: заголовок, тело, свойства, метаданные.  
- Форматирование заголовка (префиксы `#`, `##`, `###`).  
- Пример: Markdown‑блок заголовка первого уровня.
  ```markdown
  # Заголовок первого уровня
  ```
- Свойства блока: `id`, `class`, `style`.  
  ```markdown
  <!-- id: intro -->
  ```

### 3. Иерархия и вложенность (20 мин)
- Правила вложения блоков (параграфы внутри списков, списки внутри таблиц).  
- Пример вложенной структуры:
  ```markdown
  ## Список
  - Пункт 1
    - Подпункт 1.1
  - Пункт 2
  ```
- Алгоритм обхода дерева блоков (прямой, обратный обход).

### 4. Типы блоков в Python‑проекте (20 мин)
- Текстовые блоки: заголовки, параграфы.  
- Кодовые блоки: `python`, `shell`.  
  ```python
  ```python
  def hello():
      print("Hello, world!")
  ```
  ```
- Табличные блоки: CSV‑подобный синтаксис.  
  ```markdown
  | Имя | Возраст |
  |-----|---------|
  | Анна | 23 |
  ```

### 5. Работа с блоками через API (20 мин)
- Библиотека `python-docx` для работы с Word‑документами.  
  ```python
  from docx import Document
  doc = Document()
  doc.add_heading('Заголовок', level=1)
  doc.add_paragraph('Текст параграфа')
  doc.save('example.docx')
  ```
- Библиотека `markdown` для парсинга и генерации Markdown‑документов.  
  ```python
  import markdown
  html = markdown.markdown('# Пример')
  ```

### 6. Практическое упражнение (15 мин)
- Создание собственного документа с заголовками, списками и таблицей.  
- Экспорт в PDF при помощи `pandoc` (командная строка).

## Примеры и иллюстрации
- **Markdown‑таблица**  
  ```markdown
  | Функция | Описание |
  |---------|----------|
  | `add_heading` | Добавляет заголовок |
  | `add_paragraph` | Добавляет параграф |
  ```
- **Python‑скрипт для генерации документа**  
  ```python
  from docx import Document

  def create_doc():
      doc = Document()
      doc.add_heading('Блочная модель', level=1)
      doc.add_paragraph('Документ, созданный с помощью Python.')
      doc.add_heading('Пример таблицы', level=2)
      table = doc.add_table(rows=3, cols=2)
      cells = table.rows[0].cells
      cells[0].text = 'Заголовок 1'
      cells[1].text = 'Заголовок 2'
      doc.save('block_model.docx')

  if __name__ == "__main__":
      create_doc()
  ```
