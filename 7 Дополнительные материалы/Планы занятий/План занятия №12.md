# Лабораторная работа №6: Использование @media-запросов для создания адаптивной версии веб‑страницы в зависимости от разрешения экрана и типа устройства

## Цель занятия
1. Научить студентов применять CSS‑@media запросы для адаптивного дизайна веб‑страниц.
2. Познакомить с практическими инструментами (браузерные DevTools) для отладки адаптивного поведения.

## Задачи занятия
- Понять принципы работы @media запросов и их синтаксис.
- Научиться создавать адаптивные стили для различных разрешений экрана.
- Разобрать примеры адаптивного дизайна и отладить их в браузере.
- Реализовать собственный адаптивный проект в рамках заданного шаблона.

## План занятия (90 минут)

- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- Изучение нового материала — 25 мин  
- Практическая работа — 35 мин  
- Закрепление — 10 мин  
- Итог занятия — 5 мин  

## Содержание этапов

- **Организационный момент**  
  Прием студентов, проверка наличия необходимых средств, краткое введение в тему, объявление расписания работы.

- **Актуализация знаний**  
  Быстрый опрос о предыдущих занятиях по CSS, обсуждение, как изменялась структура веб‑страниц с появлением мобильных устройств.

- **Изучение нового материала**  
  Теоретическое изложение синтаксиса @media, типов запросов (width, device-width, orientation), примеры кода, обсуждение лучших практик.

- **Практическая работа**  
  Студенты разрабатывают адаптивную страницу, используя заданный шаблон, проверяют её в DevTools, корректируют стили под разные разрешения, фиксируют изменения в репозитории.

- **Закрепление**  
  Быстрый тест из 5 вопросов по синтаксису и применению @media, обсуждение ошибок, ответы на вопросы.

- **Итог занятия**  
  Подведение итогов, раздача чек‑листа «Ключевые моменты адаптивного дизайна», напоминание о домашнем задании.

## Оборудование и материалы
- Компьютеры со стабильным интернет‑соединением  
- Проектор/дисплей для демонстрации примеров  
- IDE (Visual Studio Code, PyCharm) с установленными расширениями для HTML/CSS  
- Браузер (Chrome, Firefox) с DevTools  
- Учебник «CSS: The Definitive Guide» (секции о @media)  
- Шаблон проекта (HTML, CSS, JavaScript) для лабораторной работы

## Контроль и оценивание
- Мини‑тест (5 вопросов) после практики  
- Оценка готового проекта по чек‑листу: корректность @media, адаптивность на 3 разрешения, чистота кода  
- Участие в обсуждении и своевременность выполнения заданий

## Домашнее задание
1. Создать адаптивную страницу «О компании», используя @media запросы для ширины 320 px, 768 px и 1024 px.  
2. Подготовить краткую документацию (до 200 слов) о выборе медиазапросов и обосновании дизайна для каждого диапазона.
