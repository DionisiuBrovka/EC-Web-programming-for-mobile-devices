# Лабораторная работа № 5  
Изменение размеров HTML‑элементов при помощи CSS‑свойств. Использование различных способов размещения HTML‑элементов на веб‑странице

## Цель и задачи

1. Освоить применение CSS‑свойств `width`, `height`, `max-width`, `max-height` для управления размером элементов.
2. Изучить принципы работы процентных и абсолютных единиц измерения в CSS.
3. Научиться использовать модели размещения: блочная модель (`block`, `inline-block`, `flex`, `grid`).
4. Провести экспериментальную проверку влияния различных подходов на адаптивность веб‑страницы.

## Теоретическая часть

| Понятие | Описание |
|---------|----------|
| **CSS‑свойство `width`** | Задает ширину элемента. Может быть задано в пикселях, процентах, em, rem и др. |
| **CSS‑свойство `height`** | Задает высоту элемента. |
| **`max-width` / `max-height`** | Ограничивают максимальный размер, при котором элемент не растягивается дальше. |
| **`min-width` / `min-height`** | Минимальный размер элемента. |
| **Блочный элемент** | Заполняет всю доступную ширину, начинается с новой строки. |
| **Inline‑блок** | Как блочный, но не переносит строку, можно задавать размеры. |
| **Flexbox** | Модель гибкого размещения, позволяет выравнивать элементы по оси X и Y. |
| **Grid** | Сеточная модель, определяет строки и колонки. |
| **Позиционирование `relative`** | Элемент смещается относительно своего нормального положения. |
| **Позиционирование `absolute`** | Элемент удаляется из потока и позиционируется относительно ближайшего предка с `position: relative`. |

## Ход выполнения работы

1. **Создание проекта**  
   - Создайте папку `lab5` и в ней файлы `index.html`, `styles.css`.  
   - Подключите CSS:  
     ```html
     <link rel="stylesheet" href="styles.css">
     ```

2. **Разметка страницы**  
   - В `index.html` создайте простую структуру:  
     ```html
     <div class="container">
       <div class="box" id="box1">Box 1</div>
       <div class="box" id="box2">Box 2</div>
       <div class="box" id="box3">Box 3</div>
     </div>
     ```

3. **Базовые стили**  
   - В `styles.css` задайте стили для контейнера и блоков:  
     ```css
     .container {
       width: 80%;
       margin: 0 auto;
       border: 1px solid #333;
       padding: 10px;
       display: flex;
       gap: 10px;
     }
     .box {
       background: #add8e6;
       padding: 20px;
       text-align: center;
     }
     ```

4. **Эксперимент с размером**  
   - **Шаг 4.1**: Установите `width: 200px` для `#box1`.  
   - **Шаг 4.2**: Установите `width: 30%` для `#box2`.  
   - **Шаг 4.3**: Установите `max-width: 150px` для `#box3`.  
   - Проверьте, как изменяется расположение при изменении ширины окна браузера.

5. **Адаптивность**  
   - Добавьте медиазапрос:  
     ```css
     @media (max-width: 600px) {
       .container {
         flex-direction: column;
       }
       .box {
         width: 100% !important;
       }
     }
     ```
   - Откройте страницу в разных разрешениях и зафиксируйте поведение.

6. **Сравнение с Grid**  
   - Замените `display: flex` на `display: grid` и настройте `grid-template-columns: repeat(3, 1fr)`.  
   - Сравните результаты с Flexbox по гибкости и простоте настройки.

7. **Отладка**  
   - Используйте инструменты разработчика (DevTools) для проверки вычисленных стилей и размеров элементов.

8. **Подготовка отчёта**  
   - Сохраните скриншоты результата для каждого шага.  
   - Запишите наблюдения о поведении элементов при изменении размеров окна.

## Контрольные вопросы / задания для самопроверки

1. Какое влияние имеет свойство `max-width` на элемент, если задана фиксированная ширина `width`?  
2. В чем разница между `display: inline-block` и `display: block` с точки зрения размеров?  
3. Какой CSS‑свойством управлять высотой элемента, если нужно, чтобы она автоматически подстраивалась под содержимое?  
4. Опишите порядок применения медиазапросов в CSS.  
5. Какое свойство задаёт расстояние между элементами в Flexbox?  
6. Какие преимущества даёт использование `grid` перед `flex` при построении сложных макетов?  
7. Что произойдёт, если задать `position: absolute` элементу без ближайшего предка `position: relative`?  
8. Как сделать, чтобы блок занимал ровно 50 % ширины родительского контейнера независимо от окна?  
9. В чем отличие единицы измерения `rem` от `em`?  
10. Какой CSS‑свойство позволяет задать минимальный размер элемента?
