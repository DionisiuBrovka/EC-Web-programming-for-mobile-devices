# Лабораторная работа № 22. Верстка шаблона с использованием компонентов Bootstrap для работы с выпадающими списками, кнопками, миниатюрами, модальными окнами, индикатором процесса

## Цель занятия  
1. Научить студентов применять основные компоненты Bootstrap для создания интерактивных пользовательских интерфейсов.  
2. Развить навыки интеграции HTML, CSS и JavaScript в рамках одного проекта.  

## Задачи занятия  
- Ознакомиться с синтаксисом и возможностями компонентов Bootstrap.  
- Создать структуру шаблона, включающую выпадающие списки, кнопки, миниатюры и модальные окна.  
- Реализовать индикатор процесса с использованием JavaScript.  
- Проверить работоспособность шаблона в браузере.  
- Оценить готовый проект по заданным критериям.

## План занятия (90 мин)  

- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- Изучение нового материала — 20 мин  
- Практическая работа — 40 мин  
- Закрепление и контроль — 10 мин  
- Итог занятия — 5 мин  

## Содержание этапов  

### Организационный момент  
Краткое введение, проверка присутствия, распределение ролей и материалов.  

### Актуализация знаний  
Обсуждение предыдущих лабораторных работ по Bootstrap, повторение основ сетки и компонентов.  

### Изучение нового материала  
Презентация примеров выпадающих списков, кнопок, миниатюр и модальных окон. Рассмотрение синтаксиса индикатора процесса и его интеграции с JavaScript.  

### Практическая работа  
Студенты самостоятельно создают шаблон, добавляя все перечисленные компоненты, и реализуют индикатор прогресса. Преподаватель наблюдает и оказывает поддержку.  

### Закрепление и контроль  
Проверка готовых проектов, ответы на контрольные вопросы, обсуждение возникающих проблем.  

### Итог занятия  
Обзор достижений, разбор ошибок, раздача домашнего задания.  

## Оборудование и материалы  
- Компьютерные станции с установкой IDE (VS Code, PyCharm) и браузером.  
- Проектор и экран для демонстрации кода.  
- Доступ к онлайн-ресурсам Bootstrap (docs.getbootstrap.com).  
- Файлы шаблона и примеры кода (предоставлены преподавателем).  

## Контроль и оценивание  
- Вопросы на устную проверку по каждому компоненту.  
- Самостоятельное выполнение проекта (оценка по чеклисту).  
- Краткий тест (5 вопросов) в конце занятия.  

## Домашнее задание  
1. Расширить шаблон, добавив выпадающее меню с подменю и дополнительный модальный диалог.  
2. Написать скрипт, который будет обновлять индикатор прогресса при загрузке изображений миниатюр.
