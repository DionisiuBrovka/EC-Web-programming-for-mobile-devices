# Лабораторная работа № 21. Подключение и настройка Bootstrap. Создание каркаса макета с использованием элементов сетки. Мобильная верстка Bootstrap. Стилистическое оформление страницы

## Цель занятия
1. Освоить подключение Bootstrap к проекту и настройку базовой структуры проекта.
2. Научиться создавать адаптивный макет с использованием сетки Bootstrap и компонентов.
3. Применить стилистическое оформление страницы с помощью CSS и готовых классов Bootstrap.

## Задачи занятия
- Понять принципы подключения Bootstrap и его интеграцию в HTML‑проекты.
- Научиться работать с сеткой Bootstrap (row/col, breakpoints).
- Создать адаптивный макет, поддерживающий мобильные устройства.
- Оформить страницу с помощью готовых компонентов Bootstrap и собственных CSS‑правил.

- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- Подключение Bootstrap и настройка проекта — 15 мин

- Мобильная верстка и адаптация макета — 15 мин  
- Стилистическое оформление страницы — 10 мин

## Содержание этапов

- **Организационный момент**  
  Приветствие, проверка наличия необходимого оборудования, раздача инструкций по работе с IDE и браузером.  
- **Актуализация знаний**  
  Быстрый опрос: что такое CSS‑framework, какие преимущества дает Bootstrap, как ранее работали с сеткой в CSS.  
- **Подключение Bootstrap и настройка проекта**  
  Демонстрация подключения через CDN и локальный файл, настройка структуры проекта (index.html, styles.css, scripts.js).  
- **Создание каркаса макета с сеткой**  
  Практика: создание header, main, footer с использованием контейнера, строк и колонок, разбивка на 12‑колоночную сетку.  
- **Мобильная верстка и адаптация макета**  
  Работа с breakpoint‑ами (xs, sm, md, lg, xl), изменение порядка колонок, скрытие/показывание элементов.  
- **Стилистическое оформление страницы**  
  Добавление готовых компонентов (navbar, cards, buttons), кастомизация с помощью CSS‑переменных и собственного стиля.  
- **Закрепление и проверка знаний**  
  Короткий тест: 5 вопросов о подключении, сетке и адаптивности, обсуждение ответов.  
- **Итог занятия**  
  Подведение итогов, ответы на вопросы, раздача ссылок на дополнительные материалы и готовый шаблон проекта.  

## Оборудование и материалы
- Персональный компьютер (с установленным IDE – VS Code, PyCharm и т.д.)
- Проектор/дисплей для демонстрации
- Интернет‑подключение
- Браузер (Chrome/Firefox/Edge)
- Файлы шаблона проекта (index.html, styles.css, scripts.js)
- Учебник/ссылки по Bootstrap (документация, онлайн‑курсы)

## Контроль и оценивание
- Формат теста: 5 вопросов с выбором одного правильного ответа (проверка в конце занятия).
- Оценка практического задания: проверка структуры макета, корректности использования сетки, адаптивности, стилистики (через чеклист).
- Мини‑отчет: студент пишет краткое резюме (2–3 предложения) о проделанной работе и выводах.

2. Подготовить короткое видео (5–7 минут), демонстрирующее адаптивность созданной страницы на разных устройствах.
