# Лабораторная работа № 2  
Внедрение списков и таблиц различных видов и модификаций, аудио‑ и видеоконтента в веб‑документ  

## Цель и задачи  
1. Научиться создавать и форматировать списки (нумерованные, маркированные, вложенные) в HTML.  
2. Освоить создание таблиц: простых, с объединёнными ячейками, стилизованными через CSS.  
3. Интегрировать аудио‑ и видеоконтент в веб‑страницу, используя теги `<audio>` и `<video>`.  
4. Сформировать базовый шаблон веб‑документа, объединяющего все вышеперечисленные элементы.  

## Теоретическая часть  
| Понятие | Краткое описание | Пример/формула |
|---------|------------------|----------------|
| **HTML‑список** | Структура `<ul>`, `<ol>` и `<li>` | `<ul><li>Элемент</li></ul>` |
| **Вложенный список** | Список внутри другого списка | `<ul><li>Первый<ul><li>Вложенный</li></ul></li></ul>` |
| **Таблица** | Элементы `<table>`, `<tr>`, `<th>`, `<td>` | `<table><tr><th>Заголовок</th></tr></table>` |
| **Объединённая ячейка** | Атрибуты `colspan`, `rowspan` | `<td colspan="2">Объединённая</td>` |
| **Стилизация таблицы** | CSS‑правила для `<table>`, `<th>`, `<td>` | `table {border: 1px solid #000;}` |
| **Аудио‑тег** | Встроенный плеер | `<audio controls src="song.mp3"></audio>` |
| **Видео‑тег** | Встроенный плеер | `<video controls src="movie.mp4"></video>` |
| **Атрибут `controls`** | Показывает элементы управления | `<audio controls>...</audio>` |
| **Атрибут `autoplay`** | Автоматический старт | `<video autoplay>...</video>` |
| **Атрибут `loop`** | Зацикливание | `<audio loop>...</audio>` |

## Ход выполнения работы  

1. **Создание базовой HTML‑структуры**  
   - Откройте текстовый редактор, создайте файл `index.html`.  
   ```html
   <!DOCTYPE html>
   <html lang="ru">
   <head>
       <meta charset="UTF-8">
       <title>Лабораторная работа 2</title>
   </head>
   <body>
   </body>
   </html>
   ```

2. **Добавление списков**  
   - Внутри `<body>` разместите два списка: маркированный и нумерованный.  
   ```html
   <h2>Маркированный список</h2>
   <ul>
       <li>Элемент 1</li>
       <li>Элемент 2
           <ul>
               <li>Вложенный 2.1</li>
               <li>Вложенный 2.2</li>
           </ul>
       </li>
       <li>Элемент 3</li>
   </ul>

   <h2>Нумерованный список</h2>
   <ol>
       <li>Первый</li>
       <li>Второй</li>
       <li>Третий</li>
   </ol>
   ```

3. **Создание таблицы**  
   - Добавьте таблицу с заголовками и данными, включите объединённые ячейки.  
   ```html
   <h2>Таблица примеров</h2>
   <table border="1">
       <tr>
           <th>№</th>
           <th>Имя</th>
           <th colspan="2">Оценка</th>
       </tr>
       <tr>
           <td>1</td>
           <td>Иванов</td>
           <td>А</td>
           <td>Отлично</td>
       </tr>
       <tr>
           <td>2</td>
           <td>Петров</td>
           <td colspan="2">B</td>
       </tr>
   </table>
   ```

4. **Стилизация таблицы**  
   - Внутри `<head>` добавьте `<style>` с простыми CSS‑правилами.  
   ```html
   <style>
       table { width: 100%; border-collapse: collapse; }
       th, td { padding: 8px; text-align: left; }
       th { background-color: #f2f2f2; }
   </style>
   ```

5. **Встраивание аудио‑ и видеоконтента**  
   - Поместите теги `<audio>` и `<video>` с локальными файлами (предположим, что файлы находятся в той же папке).  
   ```html
   <h2>Аудио</h2>
   <audio controls>
       <source src="audio.mp3" type="audio/mpeg">
       Тег audio не поддерживается вашим браузером.
   </audio>

   <h2>Видео</h2>
   <video width="640" height="360" controls>
       <source src="video.mp4" type="video/mp4">
       Тег video не поддерживается вашим браузером.
   </video>
   ```

6. **Проверка работоспособности**  
   - Откройте `index.html` в браузере, убедитесь, что списки, таблица, аудио и видео отображаются корректно.  
   - При необходимости исправьте пути к файлам.
