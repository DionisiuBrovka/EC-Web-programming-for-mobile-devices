# Лабораторная работа № 15. Использование свойств и методов объекта `document` для управления CSS‑свойствами HTML‑элементов, динамического изменения значений свойств, создания анимации средствами JavaScript

## Цель занятия
- Освоить методы и свойства объекта `document` для работы с CSS‑свойствами элементов.
- Научиться динамически менять стили элементов и реализовывать простые анимации с помощью JavaScript.

## Задачи занятия
1. Ознакомиться с API `document` и его возможностями для манипуляции стилями.
2. Практически применить методы `getElementById`, `querySelector`, `style` и `classList` для изменения внешнего вида элементов.
3. Реализовать простую анимацию, используя таймеры и CSS‑трансформации.
4. Проверить полученные знания на практике через выполнение лабораторной задачи.

## Содержание этапов

### Организационный момент — 5 мин  
Проверка присутствия, объявление темы и целей, раздача инструкций и материалов.

### Актуализация знаний — 10 мин  
Краткое обсуждение ранее изученных методов DOM‑манипуляции, проверка понимания основ CSS‑стилей в JavaScript.

### Изучение нового материала — 20 мин  
Преподаватель демонстрирует синтаксис и примеры использования `document.getElementById`, `document.querySelector`, свойства `style`, методы `classList.add/remove/toggle`, а также `setInterval`/`requestAnimationFrame` для анимаций.

### Практическая работа – лабораторная задача — 30 мин  
Студенты реализуют задачу: при клике на кнопку меняется цвет фона, появляется/исчезает класс с анимацией вращения, а также запускается анимация изменения ширины элемента.

### Закрепление результатов — 10 мин  
Обсуждение решений, ответы на вопросы, проверка правильности работы скриптов, демонстрация готовых работ.

### Итог занятия — 5 мин  
Подведение итогов, повторение ключевых пунктов, раздача критериев оценки.
