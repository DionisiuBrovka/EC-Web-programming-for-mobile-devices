# Верстка шаблонов с помощью Bootstrap

## Цели и задачи лекции
- Ознакомить студентов с основными принципами работы Bootstrap.  
- Научить использовать сеточную систему и готовые компоненты для быстрого прототипирования.  
- Показать, как подключать Bootstrap к проекту на Python (Django/Flask).  
- Разработать простую страницу‑шаблон, применяя изученные методы.  

## Ключевые понятия
- **Bootstrap** – открытый CSS/JS-фреймворк для быстрой разработки адаптивных интерфейсов.  
- **Сетка (grid)** – система из 12 колонок, позволяющая строить гибкие макеты.  
- **Компоненты** – готовые блоки (кнопки, карточки, навбар и др.) с предопределённым стилем.  
- **Медиа‑запросы** – CSS‑правила, применяемые в зависимости от ширины экрана.  
- **Стилизация** – изменение внешнего вида элементов с помощью классов и пользовательских CSS‑правил.  
- **CDN** – сеть доставки контента, позволяющая подключать Bootstrap без скачивания.  
- **Flask/Django** – веб‑фреймворки Python, которые часто используют Bootstrap в шаблонах.  
- **Jinja2** – шаблонизатор, используемый в Flask/Django для генерации HTML.  
- **Responsive** – адаптивность, способность интерфейса корректно отображаться на разных устройствах.  
- **Utility‑классы** – небольшие классы Bootstrap, меняющие отступы, цвета, видимость и т.д.  

## Основное содержание

### 1. Введение в Bootstrap (5 минут)
- История и назначение.  
- Сравнение с другими фреймворками (Tailwind, Bulma).  
- Структура пакета: CSS, JS, шрифты, иконы.  

### 2. Подключение Bootstrap к проекту (5 минут)
- CDN‑вариант:
  ```html
  <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
  ```
- Локальный вариант: скачивание и подключение файлов.  
- Интеграция в шаблон Django/Flask:
  ```html
  <!-- templates/base.html -->
  <html lang="ru">
  <head>
      <meta charset="UTF-8">
      <title>{% block title %}{% endblock %}</title>
      <link href="{{ url_for('static', filename='css/bootstrap.min.css') }}" rel="stylesheet">
  </head>
  <body>
      {% block content %}{% endblock %}
  </body>
  </html>
  ```

### 3. Сетка Bootstrap (10 минут)
- Основы: контейнеры, строки, колонки.  
- Пример простого макета:
  ```html
  <div class="container">
      <div class="row">
          <div class="col-md-6">Левая колонка</div>
          <div class="col-md-6">Правая колонка</div>
      </div>
  </div>
  ```
- Адаптивные префиксы: `col-sm-`, `col-lg-`, `col-xl-`.  
- Перемещаемые колонки с `order-`.

### 4. Компоненты Bootstrap (10 минут)
- Кнопки, формы, навбар, карточки, модальные окна.  
- Пример карточки:
  ```html
  <div class="card" style="width: 18rem;">
      <img src="..." class="card-img-top" alt="...">
      <div class="card-body">
          <h5 class="card-title">Заголовок</h5>
          <p class="card-text">Краткое описание.</p>
          <a href="#" class="btn btn-primary">Кнопка</a>
      </div>
  </div>
  ```

### 5. Utility‑классы и стилизация (5 минут)
- Отступы: `mt-3`, `mb-2`, `ps-4`.  
- Цвета: `bg-primary`, `text-white`.  
- Вспомогательные классы: `d-flex`, `align-items-center`.  
- Переопределение стилей через собственный CSS:
  ```css
  .custom-btn {
      background-color: #ff5733;
      color: #fff;
  }
  ```

### 6. Создание простого шаблона (5 минут)
- Сбор всех элементов в один файл.  
- Интеграция с Jinja2 переменными:
  ```html
  <h1>{{ page_title }}</h1>
  ```

## Примеры и иллюстрации
- **Пример навбара**:
  ```html
  <nav class="navbar navbar-expand-lg navbar-light bg-light">
      <div class="container-fluid">
          <a class="navbar-brand" href="#">Brand</a>
          <button class="navbar-toggler" type="button" data-bs-toggle="collapse"
                  data-bs-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false"
                  aria-label="Toggle navigation">
              <span class="navbar-toggler-icon"></span>
          </button>
          <div class="collapse navbar-collapse" id="navbarNav">
              <ul class="navbar-nav">
                  <li class="nav-item"><a class="nav-link active" aria-current="page" href="#">Home</a></li>
                  <li class="nav-item"><a class="nav-link" href="#">Features</a></li>
              </ul>
          </div>
      </div>
  </nav>
  ```

- **Схема сетки** (схема в виде текста):

  ```
  | 6  | 6  | 6  | 6  | 6  | 6  | 6  | 6  | 6  | 6  | 6  | 6  |
  ```

## Выводы по лекции
- Bootstrap – мощный инструмент для быстрой и адаптивной верстки.  
- Сеточная система и готовые компоненты значительно ускоряют разработку интерфейсов.  
- Веб‑фреймворки Python (Flask, Django) легко интегрируются с Bootstrap через шаблонизатор Jinja2.  
- Понимание utility‑классов позволяет гибко менять внешний вид без написания большого объёма CSS.
