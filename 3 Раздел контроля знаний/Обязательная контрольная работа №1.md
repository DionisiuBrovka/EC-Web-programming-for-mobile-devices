# Обязательная контрольная работа №1

## Цель и задачи контрольной работы
- Проверить знание основ Python и инструментального ПО.
- Оценить умение применять синтаксис и стандартную библиотеку.
- Проверить навыки анализа и проектирования простых программных решений.

## Краткое теоретическое введение
- **Python** – интерпретируемый язык высокого уровня с динамической типизацией.
- **Типы данных**: `int`, `float`, `str`, `bool`, `list`, `tuple`, `dict`, `set`.
- **Операторы**: арифметические (`+ - * / // % **`), логические (`and or not`), сравнения (`== != < > <= >=`).
- **Контрольные структуры**: `if-elif-else`, `for`, `while`.
- **Функции**: объявление (`def`), аргументы, возвращаемое значение.
- **Модули**: импорт (`import`, `from ... import ...`), поиск в `sys.path`.
- **Файловый ввод/вывод**: `open`, методы `read`, `write`, контекстный менеджер `with`.

## Пример решения (задание 2 варианта – практическое)

```python
# grades.py
grades = {
    'Alice': 78,
    'Bob': 54,
    'Charlie': 92,
    'Diana': 63
}

for name, score in grades.items():
    grades[name]['passed'] = score >= 60
```

**Пояснение**  
1. Создаём словарь `grades`.  
2. Проходим по каждому элементу через `items()`.  
3. Добавляем вложенный словарь `passed` с логическим значением.  
4. Итоговый словарь содержит исходные оценки и статус прохождения.
