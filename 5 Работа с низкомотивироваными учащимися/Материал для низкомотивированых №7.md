# Определение классов в JavaScript

## Цель и задачи занятия
- Понять, что такое классы и почему они полезны в JavaScript.
- Научиться объявлять классы и создавать их экземпляры.
- Освоить основные методы и свойства классов.
- Увидеть, как классы упрощают работу с повторяющимся кодом.

## Почему это важно?
В реальной работе программистам часто приходится создавать похожие объекты: пользователь, продукт, заказ и т.п. Классы дают удобный способ описать «шаблон» этих объектов, чтобы не писать один и тот же код снова и снова. Это экономит время и уменьшает количество ошибок.

## Основные понятия и идеи
1. **Класс** – шаблон, описывающий свойства и методы объектов.  
2. **Экземпляр (объект)** – конкретный объект, созданный по шаблону класса.  
3. **Конструктор** – специальный метод `constructor()`, который инициализирует новый объект.  
4. **Метод** – функция, принадлежащая классу, доступная всем его экземплярам.  
5. **Наследование** – механизм, позволяющий одному классу наследовать свойства другого.  
6. **`this`** – ссылка на текущий экземпляр класса.  
7. **Статический метод** – метод, который вызывается на самом классе, а не на его экземпляре.

## Объяснение темы

### [15 мин] Что такое класс и зачем он нужен
- Класс похож на чертёж: он описывает, какие свойства и функции будет иметь объект.  
- Пример:  
  ```js
  class Car {
    constructor(make, model) {
      this.make = make;
      this.model = model;
    }
    drive() {
      console.log(`${this.make} ${this.model} goes vroom!`);
    }
  }
  ```

### [10 мин] Создание экземпляра класса
- Экземпляр создаётся через оператор `new`.  
- Пример:  
  ```js
  const myCar = new Car('Toyota', 'Corolla');
  myCar.drive(); // Toyota Corolla goes vroom!
  ```

### [10 мин] Методы и свойства
- Свойства задаются в конструкторе.  
- Методы объявляются как обычные функции внутри класса.  
- Пример:  
  ```js
  class Person {
    constructor(name, age) {
      this.name = name;
      this.age = age;
    }
    greet() {
      console.log(`Hi, I’m ${this.name} and I’m ${this.age} years old.`);
    }
  }
  ```

### [10 мин] Наследование и `super`
- С помощью `extends` можно наследовать класс.  
- `super()` вызывает конструктор родительского класса.  
- Пример:  
  ```js
  class Employee extends Person {
    constructor(name, age, job) {
      super(name, age);
      this.job = job;
    }
    work() {
      console.log(`${this.name} works as a ${this.job}.`);
    }
  }
  const alice = new Employee('Alice', 30, 'designer');
  alice.greet(); // Hi, I’m Alice and I’m 30 years old.
  alice.work(); // Alice works as a designer.
  ```

### [10 мин] Статические методы
- Вызываются на самом классе, а не на экземпляре.  
- Пример:  
  ```js
  class MathUtils {
    static add(a, b) {
      return a + b;
    }
  }
  console.log(MathUtils.add(3, 5)); // 8
  ```

## Итог и выводы
Классы в JavaScript позволяют описать структуру и поведение объектов в одном месте, делая код более читаемым и повторно используемым. Вы научились объявлять классы, создавать объекты, добавлять методы, использовать наследование и статические методы. Теперь вы готовы применять эти навыки в реальных проектах.
