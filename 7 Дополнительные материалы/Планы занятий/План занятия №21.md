# Лабораторная работа № 11. Создание объектов, назначение свойств и методов объектов. Использование пользовательских объектов в сценариях на языке JavaScript

## Цель занятия
1. Научиться создавать пользовательские объекты в JavaScript, задавать им свойства и методы.  
2. Освоить применение этих объектов в скриптах, повышая читаемость и модульность кода.  

## Задачи занятия
- Определить структуру объектов и синтаксис их создания.  
- Научиться добавлять свойства и методы после создания объекта.  
- Разработать простую модель пользовательского объекта и использовать её в сценарии.  
- Понять принципы инкапсуляции и повторного использования кода через объекты.  
- Оценить работу с объектами, выявив типичные ошибки.  

## План занятия (90 минут)

- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- Изучение нового материала — 20 мин  
- Практическое задание — 30 мин  
- Закрепление — 10 мин  
- Итог занятия — 5 мин  

## Содержание этапов

- **Организационный момент**  
  Вступительное слово, проверка готовности оборудования, раздача ссылок на материалы.  

- **Актуализация знаний**  
  Быстрый опрос о ранее изученных объектах в JavaScript: литералы, конструкторы, `this`.  

- **Изучение нового материала**  
  Лекция с примерами: создание объектов через литералы, функции-конструкторы, классы ES6, добавление свойств и методов после создания.  

- **Практическое задание**  
  Студенты пишут программу, создают объект «Автомобиль» со свойствами `марка`, `модель`, `год` и методом `топливо()`, после чего используют его в небольшом сценарии.  

- **Закрепление**  
  Групповая работа: каждый участник добавляет новый метод к объекту, демонстрируя работу с `this`.  

- **Итог занятия**  
  Подведение итогов, ответы на вопросы, раздача домашнего задания.  

## Оборудование и материалы
- Компьютеры с установленным Node.js и VS Code (или аналогичная IDE).  
- Проектор и экран для демонстрации.  
- Справочный материал: ссылка на MDN Object и классы ES6.  
- Вспомогательные файлы: шаблоны кода, примеры задач.  

## Контроль и оценивание
- В ходе практики преподаватель наблюдает за выполнением задания, фиксирует ошибки.  
- В конце занятия проводится короткий тест (5 вопросов) по темам: создание объектов, добавление свойств/методов, использование `this`.  
- Оценка по выполненному заданию: корректность кода, использование методов, чистота оформления.
