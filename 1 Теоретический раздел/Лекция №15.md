# Библиотека jQuery

## Цели и задачи лекции  
1. Познакомить студентов с историей и назначением jQuery.  
2. Показать основные возможности библиотеки для работы с DOM, событиями и AJAX.  
3. Научить писать простые скрипты, используя jQuery‑синтаксис.  
4. Оценить преимущества и ограничения использования jQuery в современных проектах.

## Ключевые понятия  
- **DOM** – объектная модель документа, структура HTML‑страницы.  
- **Selector** – выражение, выбирающее элементы DOM (например, `$(".btn")`).  
- **Event** – событие, происходящее в элементе (клик, загрузка, изменение).  
- **Ajax** – асинхронный обмен данными с сервером без перезагрузки страницы.  
- **Chaining** – последовательное выполнение нескольких методов jQuery без промежуточных переменных.  
- **Plugin** – расширение jQuery, добавляющее новые методы и функциональность.  
- **Callback** – функция, передаваемая как аргумент и вызываемая после завершения операции.  
- **Deferred / Promise** – объект, представляющий отложенное действие и его результат.  
- **Data attributes** – пользовательские атрибуты `data-*` для хранения данных в элементе.  
- **NoConflict** – режим, позволяющий использовать jQuery совместно с другими библиотеками.

## Основное содержание  

### 1. Введение в jQuery (10 мин)  
- История создания (2006 г.) и причины популярности.  
- Сравнение с чистым JavaScript (Vanilla JS).  
- Как подключить библиотеку: CDN и локальная копия.  

```html
<!-- CDN -->
<script src="https://code.jquery.com/jquery-3.7.0.min.js"></script>
<!-- Локально -->
<script src="/js/jquery-3.7.0.min.js"></script>
```

### 2. Работа с DOM (20 мин)  
- Выбор элементов: `$()`, `$(selector)`.  
- Методы получения/установки значений: `.text()`, `.html()`, `.val()`.  
- Модификация атрибутов: `.attr()`, `.prop()`.  
- Добавление и удаление элементов: `.append()`, `.prepend()`, `.remove()`, `.empty()`.  
- Стилизация: `.css()`, `.addClass()`, `.removeClass()`.  

```js
// Пример: менять заголовок и добавить класс
$("#title").text("Новый заголовок").addClass("highlight");
```

### 3. События (15 мин)  
- Привязка событий: `.on()`, `.click()`, `.hover()`.  
- Делегирование событий: `$(parent).on('click', 'child', handler)`.  
- Остановка распространения и предотвращение действия: `event.stopPropagation()`, `event.preventDefault()`.  

```js
$(document).on('click', '.btn', function(e){
  e.preventDefault();
  alert('Кнопка нажата');
});
```

### 4. Ajax и асинхронность (20 мин)  
- Основы AJAX: `$.ajax()`, `$.get()`, `$.post()`.  
- Обработка JSON и XML.  
- Использование `$.when()` и `$.Deferred` для последовательных запросов.  

```js
$.get('/api/users', function(data){
  console.log('Пользователи:', data);
});
```

### 5. Plugins и расширения (10 мин)  
- Как подключать сторонние плагины (e.g., Slick, Fancybox).  
- Создание собственного плагина: шаблон и пример.  

```js
(function($){
  $.fn.greenify = function() {
    return this.css('color', 'green');
  };
})(jQuery);

// Использование
$('.text').greenify();
```

### 6. Современные альтернативы и совместимость (5 мин)  
- Сравнение с фреймворками (React, Vue, Angular).  
- Как использовать jQuery в проектах ES6+ с модулями.  

## Примеры и иллюстрации  

```html
<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <title>jQuery Demo</title>
  <script src="https://code.jquery.com/jquery-3.7.0.min.js"></script>
  <style>
    .highlight { background: yellow; }
  </style>
</head>
<body>
  <h1 id="title">Заголовок</h1>
  <button class="btn">Нажми меня</button>
  <ul id="list"></ul>

  <script>
    $(function(){
      // Изменяем заголовок
      $("#title").text("Обновлённый заголовок").addClass("highlight");
      
      // Добавляем элемент в список
      $("#list").append("<li>Элемент 1</li>");
      
      // Обработчик клика
      $(".btn").on("click", function(){
        alert("Кнопка нажата");
      });
      
      // AJAX-запрос (пример)
      $.get("/api/items", function(data){
        console.log(data);
      });
    });
  </script>
</body>
</html>
```

## Выводы по лекции  
- jQuery существенно упрощает работу с DOM, событиями и AJAX, особенно в проектах без тяжёлых фреймворков.  
- Понимание основ jQuery позволяет быстро писать интерактивные веб‑страницы и подключать сторонние плагины.  
- С учётом современных трендов, jQuery остаётся полезной, но для новых проектов стоит оценивать необходимость его использования.
