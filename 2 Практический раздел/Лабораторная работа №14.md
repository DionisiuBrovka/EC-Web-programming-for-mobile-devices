# Лабораторная работа № 14  
Использование свойств и методов объекта `document` для доступа к элементам веб‑страницы  

## Цель и задачи  
1. Познакомиться с объектом `document` в DOM‑API и его основными свойствами.  
2. Научиться получать элементы по id, имени, тегу, классу и CSS‑селектору.  
3. Изучить методы `querySelector` и `querySelectorAll` для выборки элементов.  
4. Практически применить полученные знания для динамического изменения содержимого страницы.  

## Теоретическая часть  
| Понятие | Краткое описание |
|---------|------------------|
| `document` | Глобальный объект, представляющий документ в браузере. |
| `getElementById(id)` | Возвращает элемент с заданным `id`. |
| `getElementsByName(name)` | Возвращает коллекцию элементов с заданным `name`. |
| `getElementsByTagName(tagName)` | Возвращает коллекцию элементов заданного тега. |
| `getElementsByClassName(className)` | Возвращает коллекцию элементов с заданным классом. |
| `querySelector(selector)` | Возвращает первый элемент, удовлетворяющий CSS‑селектору. |
| `querySelectorAll(selector)` | Возвращает коллекцию всех элементов, удовлетворяющих CSS‑селектору. |
| `innerText` | Текстовое содержимое элемента. |
| `innerHTML` | HTML‑содержимое элемента. |
| `style` | Объект стилей элемента, позволяющий менять CSS‑свойства. |
| `addEventListener(type, listener)` | Добавляет обработчик события к элементу. |

## Ход выполнения работы  

1. **Подготовка HTML‑файла**  
   ```html
   <!DOCTYPE html>
   <html lang="ru">
   <head>
       <meta charset="UTF-8">
       <title>Lab14</title>
   </head>
   <body>
       <h1 id="header">Заголовок</h1>
       <p class="text">Текстовый абзац.</p>
       <div name="container">Контейнер</div>
       <ul id="list">
           <li class="item">Элемент 1</li>
           <li class="item">Элемент 2</li>
       </ul>
       <script src="script.js"></script>
   </body>
   </html>
   ```
   Создайте файл `script.js` рядом с HTML.

2. **Получение элемента по `id`**  
   ```js
   const header = document.getElementById('header');
   console.log(header.innerText); // "Заголовок"
   ```
   Объяснение: `getElementById` возвращает одиночный элемент.

3. **Получение элементов по имени**  
   ```js
   const containers = document.getElementsByName('container');
   console.log(containers.length); // 1
   ```
   Объяснение: возвращает `NodeList` (похож на массив).

4. **Получение элементов по тегу**  
   ```js
   const items = document.getElementsByTagName('li');
   console.log(items[0].innerText); // "Элемент 1"
   ```

5. **Получение элементов по классу**  
   ```js
   const texts = document.getElementsByClassName('text');
   console.log(texts[0].innerText); // "Текстовый абзац."
   ```

6. **Использование `querySelector`**  
   ```js
   const firstItem = document.querySelector('.item');
   console.log(firstItem.innerText); // "Элемент 1"
   ```

7. **Использование `querySelectorAll`**  
   ```js
   const allItems = document.querySelectorAll('.item');
   allItems.forEach((el, idx) => console.log(idx + 1, el.innerText));
   ```

8. **Динамическое изменение содержимого**  
   ```js
   header.innerText = 'Новый заголовок';
   header.style.color = 'blue';
   ```

9. **Обработчик события**  
   ```js
   header.addEventListener('click', () => {
       alert('Заголовок кликнут!');
   });
   ```

10. **Тестирование**  
    - Откройте страницу в браузере.  
    - Проверьте, что заголовок изменился и реагирует на клик.  
    - Откройте консоль (F12) и убедитесь, что выводы из пунктов 2‑7 корректны.

## Контрольные вопросы / задания для самопроверки  

1. Что возвращает `document.getElementsByClassName('item')`?  
2. Как изменить цвет текста элемента с id `header` на красный при помощи JavaScript?  
3. Напишите код, который выводит в консоль количество элементов `<li>` на странице.  
4. Как получить первый элемент с классом `text` при помощи `querySelector`?  
5. Что произойдет, если в `querySelectorAll` указать селектор, который не соответствует ни одному элементу?  
6. Добавьте обработчик события `mouseover` к элементу `<p>` и измените его фон на жёлтый.  
7. Как получить коллекцию всех элементов с атрибутом `name="container"` без использования `getElementsByName`?  
8. Почему `querySelector` возвращает `null`, если элемент не найден, а `querySelectorAll` — пустой `NodeList`?  
9. Как изменить внутренний HTML элемента с id `list` на `<li>Новый элемент</li>`?  
10. Опишите разницу между `innerText` и `innerHTML`.
