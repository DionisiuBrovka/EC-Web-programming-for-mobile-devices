# Лабораторная работа № 10  
Объявление и вызов функций в JavaScript. Использование функций при процедурном подходе в программировании на языке JavaScript  

## Цель и задачи  
1. Освоить синтаксис объявления и вызова функций в JavaScript.  
2. Научиться использовать функции для структурирования процедурного кода.  
3. Разработать простые программы, где функции упрощают решение задач.  
4. Оценить влияние функций на читаемость и повторное использование кода.  

## Теоретическая часть  
- **Функция** – самостоятельный блок кода, который может быть вызван по имени.  
- **Объявление функции** – конструкция `function имя(параметры) { тело }`.  
- **Параметры** – переменные, передаваемые в функцию.  
- **Возврат значения** – оператор `return`.  
- **Анонимная функция** – функция без имени, часто передаётся как аргумент.  
- **Замыкание** – функция, запоминающая переменные внешней области видимости.  
- **Область видимости** – набор переменных, доступных в текущем контексте.  
- **Вызов функции** – запись `имя(аргументы)`.  
- **Перегрузка** – в JavaScript невозможна, но можно реализовать через проверки количества аргументов.  
- **Строгое сравнение** – оператор `===` проверяет тип и значение.  

## Ход выполнения работы  

1. **Подготовка среды**  
   - Установить Node.js (если ещё не установлен).  
   - Создать проект: `mkdir js_labs && cd js_labs && touch lab10.js`.  

2. **Объявление простой функции**  
   - Пояснение: создаём функцию `greet`, которая принимает имя и выводит приветствие.  
   ```js
   function greet(name) {
       console.log(`Привет, ${name}!`);
   }
   greet('Иван');
   ```  

3. **Функция с возвращаемым значением**  
   - Пояснение: функция `sum` возвращает сумму двух чисел.  
   ```js
   function sum(a, b) {
       return a + b;
   }
   console.log(sum(5, 7)); // 12
   ```  

4. **Использование функции в цикле**  
   - Пояснение: функция `printNumbers` выводит числа от 1 до N.  
   ```js
   function printNumbers(n) {
       for (let i = 1; i <= n; i++) {
           console.log(i);
       }
   }
   printNumbers(5);
   ```  

5. **Анонимная функция и callback**  
   - Пояснение: передаём анонимную функцию как аргумент `setTimeout`.  
   ```js
   setTimeout(function() {
       console.log('Таймер завершён');
   }, 1000);
   ```  

6. **Замыкание**  
   - Пояснение: функция `makeCounter` возвращает функцию, запоминающую счётчик.  
   ```js
   function makeCounter() {
       let count = 0;
       return function() {
           count++;
           return count;
       };
   }
   const counter = makeCounter();
   console.log(counter()); // 1
   console.log(counter()); // 2
   ```  

7. **Проверка аргументов**  
   - Пояснение: функция `multiply` принимает переменное число аргументов.  
   ```js
   function multiply(...args) {
       return args.reduce((acc, val) => acc * val, 1);
   }
   console.log(multiply(2, 3, 4)); // 24
   ```  

8. **Экспериментальная часть**  
   - Создать программу, которая генерирует случайное число, проверяет, является ли оно простым, и выводит результат.  
   - Разделить код на функции: `generateRandom`, `isPrime`, `main`.  

9. **Анализ результатов**  
   - Сравнить код с и без функций, оценить читаемость и повторное использование.
