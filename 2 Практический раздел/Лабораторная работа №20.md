# Лабораторная работа № 20  
**Обработка событий с использованием jQuery. Создание анимации: метод `animate()`. Использование стандартных виджетов библиотеки jQuery UI. Обработка данных пользовательских форм при помощи библиотеки jQuery.**

---

## Цель и задачи

1. Освоить работу с библиотекой jQuery: подключение, синтаксис, основные методы.
2. Научиться создавать анимацию с помощью `animate()` и управлять её параметрами.
3. Изучить стандартные виджеты jQuery UI (datepicker, slider, dialog) и их применение.
4. Разработать обработку пользовательских форм с валидацией и отправкой данных через AJAX.

---

## Теоретическая часть

| Понятие | Определение |
|---------|-------------|
| **jQuery** | Легковесная библиотека JavaScript, упрощающая манипуляцию DOM, события, анимацию и AJAX. |
| **DOM‑объект** | Представление HTML‑элемента в виде объекта, доступного через JavaScript. |
| **Селектор** | Выражение, определяющее набор DOM‑элементов (например, `$('#id')`, `$('.class')`). |
| **Событие** | Действие пользователя (click, hover, keyup), которое можно отследить через jQuery. |
| **Метод `animate()`** | Позволяет изменять CSS‑свойства элемента в течение заданного времени с плавной интерполяцией. |
| **jQuery UI** | Расширение jQuery, предоставляющее готовые виджеты и эффекты. |
| **AJAX** | Асинхронный запрос к серверу, позволяющий обновлять часть страницы без перезагрузки. |
| **JSON** | Формат обмена данными (JavaScript Object Notation), часто используемый в AJAX. |
| **Валидация формы** | Проверка корректности введённых пользователем данных перед отправкой. |
| **Event delegation** | Техника привязки обработчика к родительскому элементу для обработки событий дочерних элементов. |

## Ход выполнения работы

1. **Подготовка проекта**  
   *Создайте каталог `lab20` и в нём файлы `index.html`, `style.css`, `script.js`. Подключите jQuery и jQuery UI.*  

   ```html
   <!DOCTYPE html>
   <html lang="ru">
   <head>
       <meta charset="UTF-8">
       <title>Lab 20</title>
       <link rel="stylesheet" href="https://code.jquery.com/ui/1.13.2/themes/base/jquery-ui.css">
       <link rel="stylesheet" href="style.css">
   </head>
   <body>
       <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script>
       <script src="https://code.jquery.com/ui/1.13.2/jquery-ui.min.js"></script>
       <script src="script.js"></script>
   </body>
   </html>
   ```

2. **Создание элемента для анимации**  
   Добавьте в `index.html` прямоугольник, который будем анимировать.  

   ```html
   <div id="box" style="width:100px;height:100px;background:#3498db;position:absolute;"></div>
   ```

3. **Анимация методом `animate()`**  
   В `script.js` напишите функцию, которая перемещает блок по координатам x=200, y=200 за 1 с.  

   ```javascript
   $(function() {
       $('#box').on('click', function() {
           $(this).animate({
               left: '200px',
               top: '200px',
               opacity: 0.5
           }, 1000);
       });
   });
   ```

   *Пояснение*: `animate()` принимает объект свойств и время в миллисекундах.  

4. **Вставка виджета jQuery UI**  
   Добавьте поле даты и слайдер.  

   ```html
   <label>Выберите дату: <input type="text" id="datepicker"></label>
   <div id="slider" style="width:200px;margin-top:20px;"></div>
   ```

   Инициализируйте их в `script.js`.  

   ```javascript
   $('#datepicker').datepicker();
   $('#slider').slider({
       min: 0,
       max: 100,
       value: 50,
       slide: function(event, ui) {
           $('#slider-value').text(ui.value);
       }
   });
   ```

5. **Форма и её валидация**  
   Добавьте простую форму.  

   ```html
   <form id="contactForm">
       <label>Имя: <input type="text" name="name" required></label><br>
       <label>Email: <input type="email" name="email" required></label><br>
       <button type="submit">Отправить</button>
   </form>
   <div id="formResult"></div>
   ```

   В `script.js` реализуйте проверку и AJAX‑запрос (псевдосервер).  

   ```javascript
   $('#contactForm').on('submit', function(e) {
       e.preventDefault();
       if (!this.checkValidity()) {
           alert('Пожалуйста, заполните все поля корректно.');
           return;
       }
       const data = $(this).serialize();
       $.ajax({
           url: 'https://example.com/submit',
           method: 'POST',
           data: data,
           success: function() {
               $('#formResult').text('Данные успешно отправлены!');
           },
           error: function() {
               $('#formResult').text('Ошибка при отправке.');
           }
       });
   });
   ```

   *Пояснение*: `serialize()` преобразует форму в строку запроса.  

6. **Экспериментальная часть**  
   Проведите серию тестов:  
   - Клик по блоку → анимация.  
   - Выбор даты и изменение значения слайдера → вывод выбранных значений.  
   - Отправка формы с валидными и невалидными данными → проверка сообщений.  

   Запишите результаты в таблицу: «Входные данные», «Ожидаемый результат», «Фактический результат», «Вывод».
