# Адаптивная верстка

## Цель и задачи занятия
- Понять, что такое адаптивная верстка и зачем она нужна.
- Научиться использовать медиазапросы и гибкие единицы измерения.
- Практически применить принципы адаптивности в простом проекте.

## Почему это важно?
В современном мире сайты открываются на мобильных телефонах, планшетах и десктопах. Если ваш проект выглядит плохо на одном из устройств, пользователи уйдут. Адаптивная верстка позволяет создать удобный интерфейс для всех, не создавая несколько разных версий.

## Основные понятия и идеи
1. **Медиазапросы (media queries)** – правила CSS, которые применяются только при определённых условиях (размер экрана, разрешение и т.д.).  
2. **Гибкие единицы измерения** – `rem`, `em`, `%`, `vh`, `vw`. Они позволяют элементам менять размер относительно родителя или окна.  
3. **Flexbox** – модель раскладки, где элементы автоматически распределяются по доступному пространству.  
4. **Grid** – двумерная раскладка, даёт полный контроль над строками и колонками.  
5. **Mobile‑first** – подход, при котором сначала разрабатывается версия для мобильного, а затем добавляются стили для больших экранов.  
6. **Viewport** – область браузера, в которой отображается контент.  
7. **Responsive images** – изображения, которые подстраиваются под размер экрана (например, `srcset`, `sizes`).

## Объяснение темы

### [10 мин] Что такое адаптивность?
- Коротко о проблеме фиксированных ширин.
- Пример: картинка шириной 800 px выглядит слишком большой на телефоне.

### [15 мин] Медиазапросы
```css
/* Базовый стиль */
body { font-size: 16px; }

/* Для экранов шириной до 600px */
@media (max-width: 600px) {
  body { font-size: 14px; }
}
```
- Попросите студентов изменить `max-width` и увидеть эффект.

### [20 мин] Гибкие единицы и Flexbox
```html
<div class="flex">
  <div>Кнопка 1</div>
  <div>Кнопка 2</div>
  <div>Кнопка 3</div>
</div>
```
```css
.flex {
  display: flex;
  gap: 10px;
}
.flex div {
  flex: 1;
}
```
- Мини‑игра: разместить карточки так, чтобы они всегда занимали ровно треть экрана.

### [15 мин] Grid и mobile‑first
```css
.grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 20px;
}
```
- Попросите изменить `minmax` и посмотреть, как меняется макет.

### [10 мин] Responsive images
```html
<img src="small.jpg"
     srcset="small.jpg 500w, medium.jpg 1000w, large.jpg 2000w"
     sizes="(max-width: 600px) 100vw, 50vw"
     alt="Пример">
```
- Задача: подобрать правильные размеры для разных экранов.

## Мини‑задания по ходу

> Как изменится макет, если убрать `flex: 1` у элементов Flexbox?

> Попробуй добавить `grid-template-columns: 1fr 2fr` и наблюдай, как меняется ширина колонок.

## Итог и выводы
- Адаптивность делает сайт доступным для всех устройств.  
- Медиазапросы и гибкие единицы измерения – ваши главные инструменты.  
- Flexbox и Grid позволяют быстро и надёжно выстраивать макет.  
- Mobile‑first упрощает поддержку и улучшает производительность.
