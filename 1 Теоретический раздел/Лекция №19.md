# Установка и настройка веб‑сервера. Основы синтаксиса PHP. Анализ ОКР №2

## Цели и задачи лекции
- Ознакомить студентов с процессом установки и конфигурации веб‑сервера Apache и PHP.
- Познакомить с базовым синтаксисом PHP и его интеграцией в HTML‑страницы.
- Провести анализ ОКР №2 и показать, как применить полученные знания на практике.

## Ключевые понятия
| Термин | Определение |
|--------|-------------|
| Веб‑сервер | Программное обеспечение, обрабатывающее HTTP‑запросы и отдающее ответы. |
| Apache | Один из самых популярных открытых веб‑серверов, использующий конфигурационные файлы *.conf*. |
| PHP | Серверный язык программирования, выполняющийся на стороне сервера и генерирующий HTML. |
| mod_php | Модуль Apache, позволяющий обрабатывать PHP‑файлы. |
| ОКР (Объектно‑концентрированное рефакторинг) | Методика улучшения внутренней структуры кода без изменения внешнего поведения. |
| VirtualHost | Конфигурация виртуального хоста, позволяющая обслуживать несколько сайтов на одном сервере. |
| .htaccess | Файл конфигурации, применяемый к отдельному каталогу в Apache. |
| MIME‑тип | Идентификатор формата контента, передаваемого в HTTP‑ответе. |
| PHP‑интерпретатор | Программа, которая читает и исполняет PHP‑код. |
| PHP‑скрипт | Файл, содержащий PHP‑код, обычно с расширением *.php*. |

## Основное содержание

### 1. Установка и настройка Apache (20 мин)
- Выбор дистрибутива (Ubuntu, CentOS, Windows).
- Команды установки (`apt install apache2`, `yum install httpd`, `choco install apache-http-server`).
- Проверка работы сервера (`systemctl status apache2`, `http://localhost/`).
- Настройка базовых параметров (`Listen`, `DocumentRoot`, `ServerName`).

```bash
sudo apt update
sudo apt install apache2
sudo systemctl enable --now apache2
```

### 2. Установка PHP и модулей (15 мин)
- Установка через пакетный менеджер (`apt install php libapache2-mod-php`).
- Проверка версии (`php -v`).
- Добавление дополнительных расширений (`php-mysql`, `php-curl`).

```bash
sudo apt install php libapache2-mod-php php-mysql
```

### 3. Интеграция PHP с Apache (10 мин)
- Конфигурация `apache2.conf` и `php.ini` (путь к логам, директива `display_errors`).
- Перезапуск сервера (`sudo systemctl reload apache2`).
- Создание тестового PHP‑файла (`/var/www/html/info.php`).

```php
<?php
phpinfo();
?>
```

### 4. Базовый синтаксис PHP (20 мин)
- Структура скрипта: открывающий тег `<?php` и закрывающий `?>`.
- Переменные, типы данных, операторы.
- Условные конструкции `if`, `switch`.
- Циклы `for`, `foreach`.
- Функции и глобальные переменные.

```php
<?php
$greeting = "Hello, world!";
echo $greeting;
?>
```

### 5. Встраивание PHP в HTML (10 мин)
- Переменные в шаблонах.
- Файлы шаблонов, `require` и `include`.
- Пример: динамический список товаров.

```php
<ul>
<?php foreach ($products as $product): ?>
  <li><?= htmlspecialchars($product) ?></li>
<?php endforeach; ?>
</ul>
```

### 6. Анализ ОКР №2 (15 мин)
- Выбор кода из ОКР №2 для рефакторинга.
- Определение «объектно‑концентрированного» подхода в PHP.
- Применение паттернов: фабрика, синглтон, DAO.
- Практический пример: превращение процедурного кода в классы.

```php
class UserRepository {
    private $pdo;

    public function __construct(PDO $pdo) {
        $this->pdo = $pdo;
    }

    public function findById(int $id): ?User {
        $stmt = $this->pdo->prepare('SELECT * FROM users WHERE id = :id');
        $stmt->execute(['id' => $id]);
        $data = $stmt->fetch(PDO::FETCH_ASSOC);
        return $data ? new User($data) : null;
    }
}
```

## Примеры и иллюстрации
- Конфигурационный файл VirtualHost.
- Схема взаимодействия клиента, Apache и PHP‑интерпретатора.

```apache
<VirtualHost *:80>
    ServerName example.com
    DocumentRoot /var/www/example
    <Directory /var/www/example>
        AllowOverride All
        Require all granted
    </Directory>
</VirtualHost>
```
