# Лабораторная работа № 1. Создание веб‑страниц. Использование тегов для форматирования текста. Внедрение ссылок и графических объектов в веб‑документ

## Цель и задачи
1. Ознакомиться с базовой структурой HTML‑документа.
2. Научиться использовать теги для форматирования текста (заголовки, параграфы, списки, цитаты).
3. Познакомиться с внедрением гиперссылок и графических объектов (изображений, видео).
4. Разработать простую веб‑страницу, демонстрирующую навыки форматирования и встраивания мультимедиа.

## Теоретическая часть
| Понятие | Определение |
|---------|-------------|
| **HTML** | Язык гипертекстовой разметки, используемый для создания веб‑страниц. |
| **Тег** | Маркер, обрамляющий содержимое, указывающий браузеру, как его отобразить. |
| **Элемент** | Пара тегов `<tag> … </tag>` вместе с их содержимым. |
| **Атрибут** | Параметр, задающий свойства элемента, например `href` в `<a>`. |
| **Семантические теги** | Теги, описывающие смысл содержимого: `<header>`, `<nav>`, `<article>`, `<footer>`. |
| **Ссылка (хиперссылка)** | Элемент `<a>` с атрибутом `href`, указывающим адрес перехода. |
| **Изображение** | Элемент `<img>` с атрибутом `src`, указывающим путь к файлу. |
| **Список** | Нумерованный (`<ol>`) и ненумерованный (`<ul>`) списки. |
| **Код** | Элемент `<code>` для отображения программного кода. |
| **Мультимедиа** | Элементы `<video>`, `<audio>`, `<iframe>` для встраивания видео, аудио и внешних ресурсов. |

## Ход выполнения работы
1. **Создание базового шаблона**  
   Откройте текстовый редактор и создайте файл `index.html`. Введите следующий код:  
   ```html
   <!DOCTYPE html>
   <html lang="ru">
   <head>
     <meta charset="UTF-8">
     <title>Моя первая страница</title>
   </head>
   <body>
   </body>
   </html>
   ```
   *Пояснение*: `<!DOCTYPE html>` объявляет документ как HTML 5. Атрибут `lang="ru"` сообщает браузеру язык.

2. **Добавление заголовков и параграфов**  
   Внутри `<body>` добавьте заголовки `<h1>`–`<h3>` и параграфы `<p>`.  
   ```html
   <h1>Заголовок первого уровня</h1>
   <p>Это первый абзац текста, демонстрирующий базовое форматирование.</p>
   <h2>Заголовок второго уровня</h2>
   <p>Второй абзац содержит <strong>жирный</strong> и <em>курсивный</em> текст.</p>
   ```
   *Пояснение*: `<strong>` усиливает важность, `<em>` добавляет акцент.

3. **Создание списков**  
   Добавьте нумерованный и ненумерованный списки.  
   ```html
   <h3>Ненумерованный список</h3>
   <ul>
     <li>Элемент списка 1</li>
     <li>Элемент списка 2</li>
   </ul>
   <h3>Нумерованный список</h3>
   <ol>
     <li>Первый пункт</li>
     <li>Второй пункт</li>
   </ol>
   ```
   *Пояснение*: `<ul>` — неупорядоченный, `<ol>` — упорядоченный.

4. **Вставка гиперссылок**  
   Добавьте ссылку на внешний ресурс и внутреннюю ссылку.  
   ```html
   <p>Перейти на <a href="https://www.python.org" target="_blank">Python.org</a>.</p>
   <p>Вернуться к <a href="#top">верхней части</a> страницы.</p>
   ```
   *Пояснение*: `target="_blank"` открывает ссылку в новой вкладке; `href="#top"` ссылается на якорь.

5. **Вставка изображения**  
   Сохраните картинку в папку проекта, например `image.jpg`, и вставьте её:  
   ```html
   <h3>Изображение</h3>
   <img src="image.jpg" alt="Пример изображения" width="300">
   ```
   *Пояснение*: `alt` обеспечивает доступность, `width` задаёт ширину.

6. **Встраивание видео**  
   Скачайте короткое видео в формате MP4 (или используйте `<iframe>` для YouTube).  
   ```html
   <h3>Видео</h3>
   <video controls width="480">
     <source src="demo.mp4" type="video/mp4">
     Ваш браузер не поддерживает видео.
   </video>
   ```
   *Пояснение*: атрибут `controls` добавляет элементы управления.

7. **Проверка в браузере**  
   Откройте файл `index.html` в любом браузере. Убедитесь, что все элементы отображаются корректно: заголовки, списки, ссылки, изображения и видео.

## Контрольные вопросы / задания для самопроверки
1. Что означает строка `<!DOCTYPE html>` в начале документа?  
2. Какой тег используется для создания нумерованного списка?  
3. Как сделать так, чтобы ссылка открывалась в новой вкладке?  
4. Как задать альтернативный текст для изображения и зачем он нужен?  
5. В чем разница между `<a href="#top">` и `<a href="https://example.com">`?  
6. Как добавить кнопку воспроизведения видео в `<video>`?  
7. Что делает атрибут `width` у тега `<img>`?  
8. Как вставить цитату в документ?  
9. Какие семантические теги можно использовать для обозначения навигационной панели?  
10. Как проверить, что ваш HTML валиден?
