# Объектная модель браузера. Использование объектов HTML и объектной модели документа

## Цели и задачи лекции  
- Познакомить студентов с концепцией объектной модели браузера (DOM).  
- Показать, как объекты HTML отражаются в DOM и как их можно использовать в Python.  
- Рассмотреть основные методы и свойства объектов DOM, применяемые в веб‑разработке.  
- Выполнить практические упражнения по манипуляции DOM с использованием Python‑скриптов.

## Ключевые понятия  
| Термин | Определение |
|--------|-------------|
| **DOM (Document Object Model)** | Иерархическая структура объектов, представляющих элементы, атрибуты и текст веб‑страницы. |
| **Node** | Базовый элемент DOM, может быть элементом, атрибутом, текстом и т.д. |
| **Element** | Узел, представляющий тег HTML (например, `<div>`, `<p>`). |
| **Attribute** | Свойство элемента, содержащее имя и значение (например, `class="header"`). |
| **Text node** | Узел, содержащий текст внутри элемента. |
| **Browser object model (BOM)** | Набор объектов, предоставляемых браузером (window, history, location). |
| **XMLHttpRequest / Fetch API** | Методы для выполнения HTTP‑запросов из браузера. |
| **Event** | Событие, происходящее в DOM (клик, загрузка, изменение). |
| **Python‑webdriver** | Инструмент для автоматизации браузера из Python (Selenium, Playwright). |
| **XPath / CSS селекторы** | Языки для поиска узлов в DOM. |

### 2. Структура DOM‑дерева (20 мин)  
- Узлы: Document, Element, Attribute, Text, Comment.  
- Взаимосвязи: parent, child, sibling.  
- Пример построения DOM‑дерева для простого HTML‑страницы.  

```html
<!DOCTYPE html>
<html>
  <head><title>Demo</title></head>
  <body>
    <h1 id="title">Hello</h1>
    <p class="text">World</p>
  </body>
</html>
```

### 3. Работа с элементами и атрибутами в Python (25 мин)  
- Использование Selenium WebDriver.  
- Получение элементов по ID, имени тега, классу, XPath, CSS.  
- Чтение и изменение атрибутов.  
- Добавление, удаление и перемещение элементов.  

```python
from selenium import webdriver

driver = webdriver.Chrome()
driver.get("https://example.com")

title = driver.find_element(By.ID, "title")
print(title.text)          # Hello

p = driver.find_element(By.CSS_SELECTOR, "p.text")
driver.execute_script("arguments[0].innerText = 'New Text';", p)
```

### 4. Работа с событиями (15 мин)  
- Описание событий: click, submit, load, change.  
- Подписка на события через JavaScript из Python.  
- Обработка событий в тестовых сценариях.  

```python
script = """
document.getElementById('btn').addEventListener('click', function() {
    console.log('Button clicked');
});
"""
driver.execute_script(script)
```

### 5. Автоматизация браузера через Python (15 мин)  
- Selenium vs Playwright: преимущества и особенности.  
- Работа с фреймами, окнами, всплывающими окнами.  
- Сценарий: заполнить форму, отправить и проверить результат.  

```python
from playwright.sync_api import sync_playwright

with sync_playwright() as p:
    browser = p.chromium.launch(headless=True)
    page = browser.new_page()
    page.goto("https://example.com/form")
    page.fill("input[name='name']", "Alice")
    page.click("button[type='submit']")
    assert page.text_content("#result") == "Success"
    browser.close()
```

## Примеры и иллюстрации

```javascript
var div = document.createElement('div');
div.id = 'newDiv';
div.textContent = 'Созданный элемент';
document.body.appendChild(div);
```
