# Использование событий на веб-странице

## Цели и задачи лекции
- Понять роль событий в динамическом поведении веб‑страниц.
- Изучить основные типы событий и способы их обработки на стороне клиента.
- Научиться использовать события в рамках проекта на Python с помощью фреймворка Flask и шаблонов Jinja2.
- Овладеть практическими навыками создания интерактивных элементов: кнопок, форм, всплывающих окон.

## Ключевые понятия
- **Событие** – действие, происходящее в браузере (нажатие, ввод, изменение размера).  
- **Обработчик** – функция, привязанная к событию, выполняющая код.  
- **DOM (Document Object Model)** – объектная модель документа, через которую происходит взаимодействие с элементами.  
- **Браузерный API** – набор функций, предоставляемых браузером для работы с событиями.  
- **Кастомное событие** – событие, определённое разработчиком.  
- **Event delegation** – техника, при которой один обработчик назначается родительскому элементу для управления множеством дочерних событий.  
- **PreventDefault** – метод, отменяющий действие по умолчанию.  
- **Propagation (bubbling, capturing)** – механизм распространения события по дереву DOM.  
- **JavaScript‑Event Loop** – цикл событий, управляющий асинхронностью.  
- **AJAX** – технология, позволяющая обновлять часть страницы без перезагрузки.

### 2. Типы событий (15 мин)
- **Мышь**: click, dblclick, mouseover, mouseout, mousedown, mouseup.  
- **Клавиатура**: keydown, keyup, keypress.  
- **Формы**: submit, change, focus, blur.  
- **Состояние окна**: load, unload, resize, scroll.  
- **Кастомные события**: CustomEvent.

```js
// Пример простого обработчика click
document.getElementById('btn').addEventListener('click', function(e) {
  alert('Кнопка нажата');
});
```

### 3. Обработчики событий в JavaScript (20 мин)
- **addEventListener** vs **on*** атрибуты.  
- Передача аргументов и контекст `this`.  
- Удаление обработчика: `removeEventListener`.  
- Пример: динамическое добавление элементов и их обработка.

```js
// Event delegation
document.getElementById('list').addEventListener('click', function(e) {
  if (e.target && e.target.matches('li.item')) {
    console.log('Нажата:', e.target.textContent);
  }
});
```

### 4. Взаимодействие с сервером (30 мин)
- Отправка данных через AJAX (XMLHttpRequest / Fetch API).  
- Обработка ответов и обновление DOM.  
- Интеграция с Flask: маршрут, возвращающий JSON.  
- Пример: форма обратной связи без перезагрузки.

```python
# Flask маршрут
@app.route('/submit', methods=['POST'])
def submit():
    data = request.get_json()
    # обработка данных
    return jsonify({'status': 'ok', 'message': 'Спасибо!'})
```

```js
// JS отправка
fetch('/submit', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({name: nameInput.value})
}).then(res => res.json())
  .then(data => alert(data.message));
```

### 5. Кастомные события и их применение (15 мин)
- Создание и диспатчинг кастомного события.  
- Пример: обновление UI после изменения данных.

```js
const event = new CustomEvent('dataUpdated', {detail: {id: 5, value: 42}});
document.dispatchEvent(event);

document.addEventListener('dataUpdated', function(e) {
  console.log('Обновлено:', e.detail);
});
```

### 6. Лучшие практики и распространённые ошибки (10 мин)
- Использование `event.preventDefault()` и `stopPropagation()`.  
- Минимизация количества обработчиков через делегирование.  
- Управление состоянием UI: классы CSS, флаги.  
- Тестирование событий с Jest / Cypress.

## Примеры и иллюстрации
```html
<!-- Пример формы с AJAX -->
<form id="feedback">
  <input type="text" name="name" placeholder="Имя" required>
  <button type="submit">Отправить</button>
</form>

<script>
document.getElementById('feedback').addEventListener('submit', function(e) {
  e.preventDefault();
  const formData = new FormData(e.target);
  fetch('/submit', {
    method: 'POST',
    body: formData
  }).then(r => r.json())
    .then(data => alert(data.message));
});
</script>
```
