# Лабораторная работа № 14. Использование свойств и методов объекта `document` для доступа к элементам веб‑страницы  

## Цель занятия  
1. Научить студентов работать с объектом `document` в JavaScript, используя его свойства и методы для доступа к элементам DOM.  
2. Развить навыки практической реализации простых интерактивных веб‑страниц с использованием Python‑сервера (Flask) и JavaScript.  

## Задачи занятия  
- Ознакомить студентов с основными свойствами и методами `document`.  
- Показать, как получать доступ к элементам по id, имени, тегу и классу.  
- Демонстрировать изменение свойств элементов через JavaScript.  
- Провести практическое задание: создание интерактивной страницы, где пользователь вводит данные, а скрипт изменяет DOM.  

## План занятия (90 минут)  
- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- Изучение нового материала — 25 мин  
- Практическая работа — 35 мин  
- Закрепление — 5 мин  
- Итог занятия — 5 мин  

## Содержание этапов  

### Организационный момент  
Кратко приветствие, проверка готовности оборудования, объявление целей и задач занятия.  

### Актуализация знаний  
Проверка понимания предыдущего материала о DOM и работе с JavaScript, обсуждение вопросов по последней лабораторной работе.  

### Изучение нового материала  
Вводный теоретический блок: свойства `document` (`document.body`, `document.title`), методы (`getElementById`, `getElementsByClassName`, `querySelector`, `querySelectorAll`), события (`DOMContentLoaded`, `click`). Демонстрация примеров в IDE.  

### Практическая работа  
Студенты работают в парах над проектом:  
1. Создают Flask‑приложение, которое отдаёт HTML‑страницу.  
2. Добавляют JavaScript‑код, который после загрузки страницы меняет текст заголовка, меняет цвет фона и выводит введённые пользователем данные в реальном времени.  
3. Используют методы `document.querySelectorAll` для изменения стилей всех элементов с классом `.highlight`.  

### Закрепление  
Краткая викторина с 5 вопросами по материалу, ответы проверяются на доске.  

### Итог занятия  
Резюме проделанной работы, раздача чек-листа для самостоятельной проверки кода, объявление домашнего задания.  

## Оборудование и материалы  
- 2 ПК с установленным Python 3.10+, Flask, VS Code.  
- Проектор и экран.  
- Интернет‑связь.  
- Учебник: «Python и JavaScript: практическое руководство» (стр. 145‑158).  

## Контроль и оценивание  
- Вопросы на доске (5 мин).  
- Проверка кода в конце практики: наличие корректного доступа к элементам, изменение свойств и обработка событий.  
- Оценка лабораторной работы по чек‑листу: 40 % – корректность кода, 30 % – функциональность, 30 % – читаемость и комментарии.  

## Домашнее задание  
1. Создать страницу с формой, где пользователь вводит имя и возраст. При отправке данные выводятся в `<div id="output">` без перезагрузки страницы, используя `document.getElementById`.  
2. Добавить кнопку «Скрыть», которая при клике скрывает `<div id="output">` с помощью `style.display = 'none'`.
