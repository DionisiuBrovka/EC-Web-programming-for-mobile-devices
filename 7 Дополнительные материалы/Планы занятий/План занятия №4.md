# Лабораторная работа № 1. Создание веб‑страниц. Использование тегов для форматирования текста. Внедрение ссылок и графических объектов в веб‑документ

## Цель занятия
- Научить студентов создавать простые HTML‑страницы, используя основные теги форматирования текста.  
- Развить навыки вставки гиперссылок и изображений, а также базовых стилей с помощью атрибутов.

## Задачи занятия
1. Ознакомиться с синтаксисом основного набора HTML‑тегов.  
2. Практически применить теги `<p>`, `<h1>`‑`<h6>`, `<strong>`, `<em>`, `<a>`, `<img>` в одном документе.  
3. Сформировать структуру веб‑страницы, включающую заголовок, абзацы, списки и визуальные элементы.  
4. Проверить корректность кода с помощью простых средств в браузере.  
5. Подготовить базовый шаблон для дальнейших лабораторных работ.

## План занятия (90 мин)

- Организационный момент — 10 мин  
- Актуализация знаний — 15 мин  
- Изучение нового материала — 30 мин  
- Практическая работа — 25 мин  
- Итог занятия — 10 мин  

### Содержание этапов

**Организационный момент**  
Проверка присутствия, объявление темы и целей. Установка рабочего окружения: открытие IDE (VS Code, Sublime Text) и браузера.  

**Актуализация знаний**  
Краткое обсуждение ранее изученного HTML‑структурирования: `<html>`, `<head>`, `<body>`. Быстрый тест из 5 вопросов по основам разметки.  

**Изучение нового материала**  
Презентация ключевых тегов: заголовки `<h1>`‑`<h6>`, абзацы `<p>`, выделения `<strong>`, `<em>`, ссылки `<a>`, изображения `<img>`. Демонстрация атрибутов `href`, `src`, `alt`, `title`.  

**Практическая работа**  
Студенты создают собственную страницу, используя предоставленный шаблон. Требуется добавить заголовок, два абзаца с разным выделением, гиперссылку на официальный сайт Python и изображение логотипа. Проверка кода в браузере.  

**Итог занятия**  
Обсуждение выполненных работ, разбор типичных ошибок, ответы на вопросы. Составление краткого отчёта о проделанной работе.  

## Оборудование и материалы
- Персональные компьютеры с доступом к интернету.  
- IDE (VS Code, Sublime Text, Atom) или любой текстовый редактор.  
- Браузер (Chrome, Firefox, Edge).  
- Учебник по HTML (доступный в электронном виде).  
- Шаблон HTML‑файла, предоставленный преподавателем.  

## Контроль и оценивание
- Опросник из 10 вопросов по теоретическому материалу (пояснительный ответ).  
- Проверка выполненного HTML‑файла на наличие всех требуемых тегов и корректность атрибутов.  
- Оценка работы по критериям: структура, корректность разметки, визуальное оформление (до 10 баллов).  

## Домашнее задание
1. Расширить созданную страницу, добавив список (`<ul>`, `<ol>`) с 5 пунктами и стилизовать его простыми атрибутами.  
2. Сохранить страницу в формате PDF и отправить по электронной почте преподавателю.
