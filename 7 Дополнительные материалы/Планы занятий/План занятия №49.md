# Функции PHP для работы с БД MySQL

## Цель занятия  
1. Познакомить студентов с основными функциями PHP, используемыми для подключения к MySQL и выполнения CRUD‑операций.  
2. Позволить студентам самостоятельно реализовать простую веб‑приложение с базой данных.

## Задачи занятия  
- Объяснить принцип работы PDO и mysqli.  
- Научить формировать SQL‑запросы и обрабатывать результаты.  
- Разработать практическое задание с подключением к БД и выводом данных.  
- Оценить понимание студентами ключевых понятий и навыков работы с БД.

## План занятия (90 мин)  

- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- Теоретический блок: подключение к MySQL (PDO) — 15 мин  
- Практическое упражнение: CRUD‑операции — 30 мин  
- Закрепление и обсуждение результатов — 15 мин  
- Итог занятия, вопросы — 5 мин  

## Содержание этапов  

- **Организационный момент**  
  Знакомство, проверка присутствия, постановка целей и расписания.  

- **Актуализация знаний**  
  Быстрый опрос о предыдущем опыте работы с PHP и базами данных.  

- **Теоретический блок: подключение к MySQL (PDO)**  
  Рассказ о преимуществах PDO, синтаксисе подключения, настройке опций и обработке ошибок.  

- **Практическое упражнение: CRUD‑операции**  
  Студенты пишут скрипт, создающий таблицу, вставляющий, обновляющий, удаляющий и выводящий данные. Групповые обсуждения кода.  

- **Закрепление и обсуждение результатов**  
  Обсуждение типичных ошибок, обзор готовых решений, ответ на вопросы.  

- **Итог занятия, вопросы**  
  Подведение итогов, раздача краткого чек‑листа для самостоятельной работы.  

## Оборудование и материалы  
- Компьютеры с установленным PHP 8+ и MySQL 8.  
- Проектор и ноутбук преподавателя.  
- IDE (VS Code, PhpStorm) с плагином для PHP.  
- Доступ к базе данных (localhost).  
- Учебные материалы: шпаргалка по PDO, пример CRUD‑скрипта.  

## Контроль и оценивание  
- Вопросы на доске по каждому этапу (5 мин).  
- Короткий тест (4‑5 вопросов) после практики (5 мин).  
- Оценка выполненного скрипта по чек‑листу (проверка подключения, ошибок, структуры кода).  

## Домашнее задание  
1. Создать скрипт, который реализует пагинацию вывода данных из таблицы.  
2. Написать функцию, которая принимает массив данных и сохраняет их в таблицу через PDO, используя подготовленные запросы.
