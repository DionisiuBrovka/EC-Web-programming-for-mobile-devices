# Лабораторная работа № 8  
**Внедрение сценариев на языке JavaScript в веб‑документ.  
Обработка событий с использованием DHTML.  
Использование управляющих конструкций при написании сценариев на языке JavaScript.**

## Цель и задачи

1. Научиться подключать скрипты JavaScript к HTML‑документу и управлять их выполнением.  
2. Освоить работу с событиями элементов DHTML и обработку пользовательских действий.  
3. Разобрать основные управляющие конструкции (условные операторы, циклы, функции) в контексте динамических сценариев.  
4. Выполнить экспериментальное задание, демонстрирующее взаимодействие скрипта с DOM‑элементами, и проанализировать результаты.

## Теоретическая часть

| Понятие | Краткое определение |
|---------|---------------------|
| **JavaScript** | Язык сценариев, выполняемый в браузере, позволяющий изменять содержимое и стиль веб‑страницы. |
| **DOM** | Document Object Model – объектная модель документа, представляющая структуру HTML‑страницы как дерево объектов. |
| **DHTML** | Dynamic HTML – техника динамического изменения элементов страницы без перезагрузки. |
| **Событие** | Акт взаимодействия пользователя (клик, ввод, наведение) или браузера (загрузка, изменение размера окна). |
| **Обработчик события** | Функция, привязанная к событию, выполняющаяся при его наступлении. |
| **`addEventListener`** | Метод, позволяющий привязать обработчик события к элементу без перезаписи существующих обработчиков. |
| **Условный оператор `if`** | Конструкция, выполняющая блок кода, если заданное условие истинно. |
| **Цикл `for`** | Конструкция, повторяющая блок кода заданное число раз. |
| **Функция** | Набор инструкций, объединенных под именем и вызываемых при необходимости. |
| **`console.log`** | Метод, выводящий сообщения в консоль разработчика для отладки. |

## Ход выполнения работы

1. **Создание HTML‑файла**  
   - Создайте файл `index.html`.  
   - Добавьте базовую структуру и элементы, которые будут управляться скриптом (кнопки, поля ввода, `div`).  

   ```html
   <!DOCTYPE html>
   <html lang="ru">
   <head>
       <meta charset="UTF-8">
       <title>Lab 8</title>
   </head>
   <body>
       <h1 id="header">Привет, мир!</h1>
       <button id="btnChange">Изменить заголовок</button>
       <input type="text" id="inputName" placeholder="Введите имя">
       <div id="output"></div>
       <script src="script.js"></script>
   </body>
   </html>
   ```

2. **Подключение скрипта**  
   - Создайте файл `script.js` и подключите его через тег `<script>` как показано выше.  
   - Убедитесь, что скрипт загружается после DOM‑элементов (путём размещения тега внизу `<body>`).  

3. **Получение элементов DOM**  
   - В `script.js` получите ссылки на элементы по `id` с помощью `document.getElementById`.  

   ```js
   const header = document.getElementById('header');
   const btn = document.getElementById('btnChange');
   const input = document.getElementById('inputName');
   const output = document.getElementById('output');
   ```

4. **Обработчик события `click`**  
   - Привяжите к кнопке обработчик, который изменит текст заголовка.  

   ```js
   btn.addEventListener('click', function () {
       header.textContent = 'Заголовок изменён!';
   });
   ```

5. **Обработчик события `input`**  
   - При вводе в поле `inputName` выводите введённое значение в `div#output`.  

   ```js
   input.addEventListener('input', function () {
       output.textContent = 'Вы ввели: ' + input.value;
   });
   ```

6. **Условный оператор**  
   - Добавьте кнопку «Проверить имя», которая проверит, что имя не пустое, и выведет сообщение.  

   ```html
   <button id="btnCheck">Проверить имя</button>
   ```

   ```js
   const btnCheck = document.getElementById('btnCheck');
   btnCheck.addEventListener('click', function () {
       if (input.value.trim() === '') {
           output.textContent = 'Имя пустое!';
       } else {
           output.textContent = `Привет, ${input.value}!`;
       }
   });
   ```

7. **Цикл `for`**  
   - Создайте массив дней недели и выведите их в список.  

   ```js
   const days = ['Понедельник', 'Вторник', 'Среда', 'Четверг',
                 'Пятница', 'Суббота', 'Воскресенье'];
   const list = document.createElement('ul');
   for (let i = 0; i < days.length; i++) {
       const li = document.createElement('li');
       li.textContent = days[i];
       list.appendChild(li);
   }
   output.appendChild(list);
   ```

8. **Функция**  
   - Создайте функцию `greet(name)`, которая возвращает строку приветствия, и вызовите её при клике на кнопку.  

   ```js
   function greet(name) {
       return `Привет, ${name}!`;
   }
   btnCheck.addEventListener('click', function () {
       if (input.value.trim() !== '') {
           output.textContent = greet(input.value);
       }
   });
   ```

9. **Экспериментальная часть**  
   - Запустите страницу в браузере.  
   - Проверьте работу всех событий, убедитесь, что DOM меняется динамически.  
   - Откройте консоль разработчика и убедитесь, что ошибок нет.  

10. **Анализ результатов**  
    - Оцените, как изменения в скрипте влияют на пользовательский интерфейс.  
    - Приведите выводы о роли JavaScript в создании интерактивных веб‑страниц.
