# Правила создания CSS документа. Синтаксис CSS

## Цель и задачи занятия  
- Понять структуру CSS‑документа и как он взаимодействует с HTML.  
- Научиться писать простые правила стилей.  
- Узнать, как подключить CSS к странице.  
- Освоить базовые селекторы и свойства для визуальной настройки элементов.

## Почему это важно?  
CSS отвечает за «облик» веб‑страницы. Без него сайт выглядит как пустой лист бумаги. Знание CSS позволяет быстро менять цвета, шрифты, отступы – всё, что делает сайт приятным для глаз и удобным для пользователя. Это ключевой навык в любой веб‑девелопмент‑профессии.

## Основные понятия и идеи  
1. **Селектор** – «позыв» к элементу (например, `p`, `.class`, `#id`).  
2. **Свойство** – то, что меняется (например, `color`, `margin`).  
3. **Значение** – конкретный параметр (например, `red`, `10px`).  
4. **Каскад** – порядок применения стилей, когда несколько правил влияют на один элемент.  
5. **Внешний/внутренний/inline‑стиль** – способы подключения CSS к HTML.  
6. **Блок правил** – пара фигурных скобок `{}` с перечнем свойств.  
7. **Псевдоклассы** – специальные состояния элементов (`:hover`, `:active`).

## Объяснение темы

### [10 мин] Что такое CSS и как он выглядит  
```css
/* Пример простого CSS‑файла */
body {
    background-color: #fafafa;
    font-family: Arial, sans-serif;
}
```
- `body` – селектор, `background-color` и `font-family` – свойства, `#fafafa` и `Arial` – значения.  
- Файл обычно сохраняют с расширением `.css`.

### [15 мин] Как подключить CSS к HTML  
#### Вариант 1 – внешняя ссылка  
```html
<link rel="stylesheet" href="styles.css">
```
Размещаем в `<head>`.

#### Вариант 2 – внутренний стиль  
```html
<style>
h1 { color: navy; }
</style>
```
Ставим прямо в `<head>`.

#### Вариант 3 – inline‑стиль  
```html
<h1 style="color: navy;">Заголовок</h1>
```
Незачем, но полезен для быстрых экспериментов.

### [20 мин] Синтаксис правил CSS  
1. Селектор + `{`  
2. Перечень свойств: `свойство: значение;`  
3. `}` – закрываем блок.  
```css
p {
    color: #333;
    margin-top: 10px;
}
```
- Важно ставить точку с запятой после каждого свойства, но последняя может быть опущена.

### [15 мин] Практика: стилизуем страницу  
1. Создайте файл `index.html` и `styles.css`.  
2. В `styles.css` добавьте правило для `.card`.  
```css
.card {
    border: 1px solid #ddd;
    padding: 20px;
    background: #fff;
}
```
3. В HTML добавьте `<div class="card">Контент</div>` и посмотрите результат.

### [10 мин] Мини‑игра: «Найди ошибку»  
В следующем блоке CSS есть две ошибки. Найдите их и исправьте.  
```css
h2 {
    font-size 24px
    color: blue
}
```

- CSS – язык стилей, который делает страницы живыми и удобными.  
- Правильный синтаксис – селектор, фигурные скобки, свойства и значения.  
- Подключение CSS к HTML можно делать разными способами, но внешняя ссылка — самый чистый и масштабируемый.  
- Практика и эксперименты ускоряют обучение и повышают уверенность.
