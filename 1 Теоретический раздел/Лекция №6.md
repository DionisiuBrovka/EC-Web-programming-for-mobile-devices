# Адаптивная верстка

## Цели и задачи лекции
- Понять основные принципы адаптивной верстки и её отличие от «responsive design».
- Освоить ключевые техники и инструменты для создания адаптивных интерфейсов.
- Научиться применять гибкие сетки, гибкие единицы измерения и медиазапросы.
- Реализовать простое адаптивное веб‑приложение с помощью CSS и HTML.

## Ключевые понятия
- **Адаптивная верстка** – подход к созданию страниц, которые корректно отображаются на разных устройствах, используя фиксированные макеты для конкретных диапазонов размеров экрана.
- **Медиазапросы** – CSS‑правила, которые применяются при определённых условиях (ширина/высота окна, ориентация и т.д.).
- **Гибкая сетка** – система колонок, основанная на процентных ширинах или flexbox/grid, позволяющая контенту изменять размер.
- **Гибкие единицы измерения** – `em`, `rem`, `vh`, `vw`, `%` и т.п., которые масштабируются относительно родительских элементов или окна браузера.
- **Флексбокс** – модуль CSS, обеспечивающий гибкое размещение элементов в одном измерении.
- **Grid** – модуль CSS для двухмерной раскладки элементов.
- **Кроссбраузерность** – способность веб‑страницы корректно работать во всех браузерах.
- **Префикс** – часть кода, необходимая для поддержки старых браузеров (`-webkit-`, `-moz-`, `-ms-`).
- **Пиксели** – фиксированная единица измерения, не масштабируемая.
- **Мобильный первый** – стратегия разработки, при которой основное внимание уделяется мобильной версии, а затем расширяется до десктопа.

## Основное содержание

### 1. Введение в адаптивную верстку (10 мин)
- История развития веб‑дизайна.
- Понятие «адаптивность» vs. «responsive».
- Примеры сайтов с адаптивной версткой.

### 2. Гибкие единицы измерения (15 мин)
- `px` vs. `em`/`rem`/`%`/`vh`/`vw`.
- Как выбрать единицу измерения для конкретных элементов.
- Пример: изменение размера шрифта в зависимости от ширины окна.

```css
html { font-size: 16px; }
body { font-size: 1rem; }
h1 { font-size: 2.5rem; }
@media (max-width: 600px) {
  html { font-size: 14px; }
}
```

### 3. Сетки и флексбокс (20 мин)
- Основы flexbox: `display:flex`, `justify-content`, `align-items`.
- Создание гибкой колонки с помощью flexbox.
- Переход к grid: `display:grid`, `grid-template-columns`, `grid-gap`.

```html
<div class="grid">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
  <div class="item">4</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
  gap: 10px;
}
```

### 4. Медиазапросы (25 мин)
- Синтаксис `@media`.
- Типы медиазапросов: `width`, `min-width`, `max-width`, `orientation`, `resolution`.
- Практическое применение: скрытие/показ элементов, изменение макета.

```css
@media (max-width: 768px) {
  .sidebar { display: none; }
  .content { width: 100%; }
}
```

### 5. Кроссбраузерные техники (10 мин)
- Префиксы и автопрефиксер.
- Проверка совместимости в старых браузерах.
- Политика «mobile first» и её преимущества.

### 6. Практический пример: адаптивный карусель (10 мин)
- HTML‑структура.
- CSS‑стили с медиазапросами.
- Минимальный JavaScript для переключения слайдов.

```html
<div class="carousel">
  <div class="slides">
    <img src="img1.jpg" alt="">
    <img src="img2.jpg" alt="">
  </div>
  <button class="prev">←</button>
  <button class="next">→</button>
</div>
```

```css
.carousel { position: relative; overflow: hidden; }
.slides { display: flex; transition: transform 0.3s ease; }
@media (max-width: 600px) {
  .slides img { width: 100%; }
}
```

## Выводы по лекции
- Адаптивная верстка обеспечивает корректное отображение контента на различных устройствах.
- Гибкие единицы измерения и сетки (flexbox/grid) позволяют создавать масштабируемые макеты.
- Медиазапросы — ключевой инструмент для изменения стилей в зависимости от условий.
- Кроссбраузерность и «mobile first» повышают доступность и устойчивость сайта.
