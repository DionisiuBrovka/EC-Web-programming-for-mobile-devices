# Основные функции PHP

## Цели и задачи лекции
- Понять роль встроенных функций в PHP и их влияние на производительность.
- Изучить основные группы функций и их применение в типовых задачах.
- Научиться использовать функции для работы с строками, массивами, датами и файлами.
- Разработать навыки поиска и использования подходящих функций из стандартной библиотеки.

## Ключевые понятия
| Термин | Определение |
|--------|-------------|
| **Функция** | Самодостаточный блок кода, принимающий аргументы и возвращающий результат. |
| **Параметр** | Переменная, передаваемая в функцию. |
| **Возвращаемое значение** | Результат выполнения функции. |
| **Библиотека** | Набор предопределённых функций, доступных в PHP. |
| **Namespace** | Пространство имён, позволяющее избежать конфликтов имён функций. |
| **Массив** | Составной тип данных, содержащий упорядоченный набор элементов. |
| **Регулярное выражение** | Строка, описывающая шаблон для поиска и замены. |
| **Поток** | Поток ввода/вывода, например файл или сокет. |
| **Фильтр** | Функции, применяемые к строкам для преобразования или проверки. |
| **Callback** | Функция, передающаяся в качестве аргумента другой функции. |

## Основное содержание

### 1. Введение в функции (10 мин)
- Что такое функции в PHP.
- Синтаксис объявления и вызова.
- Параметры по значению и по ссылке.

```php
function greet(string $name): string {
    return "Привет, $name!";
}
echo greet('Андрей'); // Привет, Андрей!
```

### 2. Стандартная библиотека PHP (15 мин)
- Как подключать и использовать библиотеки.
- `phpinfo()` и поиск функций.
- Классы `Spl` и их применение.

### 3. Работа со строками (20 мин)
- Основные функции: `strlen`, `substr`, `str_replace`, `preg_match`.
- Регулярные выражения: синтаксис, флаги, примеры.

```php
$pattern = '/^\w+@\w+\.\w+$/';
$email = 'user@example.com';
if (preg_match($pattern, $email)) {
    echo 'Email валиден';
}
```

### 4. Массивы и массивные функции (20 мин)
- Создание, добавление, удаление элементов.
- Функции `array_push`, `array_pop`, `array_merge`, `array_filter`.
- Сортировка: `sort`, `usort`, `array_multisort`.

```php
$numbers = [3, 1, 4, 1, 5];
sort($numbers);
print_r($numbers); // [1, 1, 3, 4, 5]
```

### 5. Работа с датой и временем (10 мин)
- Класс `DateTime`, методы `format`, `modify`.
- Временные зоны и форматирование.

```php
$dt = new DateTime('now', new DateTimeZone('Europe/Moscow'));
echo $dt->format('Y-m-d H:i:s');
```

### 6. Файловые операции (10 мин)
- Открытие, чтение, запись, закрытие файлов.
- Функции `fopen`, `fwrite`, `fread`, `file_get_contents`, `file_put_contents`.

```php
file_put_contents('log.txt', "Новый лог\n", FILE_APPEND);
$log = file_get_contents('log.txt');
echo $log;
```

### 7. Коллбэки и анонимные функции (10 мин)
- Определение анонимных функций.
- Использование в `array_map`, `array_filter`, `usort`.

```php
$nums = [1, 2, 3];
$double = array_map(fn($n) => $n * 2, $nums);
print_r($double); // [2, 4, 6]
```

## Примеры и иллюстрации
- **Функция для проверки палиндрома**

```php
function isPalindrome(string $str): bool {
    $clean = preg_replace('/\W+/', '', strtolower($str));
    return $clean === strrev($clean);
}
echo isPalindrome('А роза упала на лапу Азора') ? 'Да' : 'Нет';
```
