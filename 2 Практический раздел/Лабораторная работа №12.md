# Лабораторная работа №12. Использование стандартных объектов JavaScript в сценариях

## Цель и задачи

1. Познакомиться с основными стандартными объектами JavaScript (Object, Array, String, Number, Date, Math, RegExp).  
2. Научиться создавать, модифицировать и использовать эти объекты в практических задачах.  
3. Развивать навыки отладки и анализа работы скриптов в браузере.  
4. Научиться документировать и представлять результаты лабораторной работы.

## Теоретическая часть

| Объект | Краткое описание | Ключевой метод / свойство |
|--------|------------------|---------------------------|
| `Object` | Базовый объект, от которого наследуются все остальные. | `Object.keys()`, `Object.entries()` |
| `Array` | Последовательность элементов с индексами. | `push()`, `filter()`, `reduce()` |
| `String` | Последовательность символов. | `split()`, `replace()`, `match()` |
| `Number` | Числовой тип. | `Math.floor()`, `Math.random()` |
| `Date` | Дата и время. | `getFullYear()`, `toLocaleString()` |
| `Math` | Статический объект, содержащий математические функции. | `Math.PI`, `Math.sqrt()` |
| `RegExp` | Регулярные выражения. | `test()`, `exec()` |

**Формулы / примеры**  
- Вычисление среднего значения массива: `const avg = arr.reduce((s, v) => s + v, 0) / arr.length;`  
- Проверка даты: `const isFuture = new Date() < new Date('2025-12-31');`

## Ход выполнения работы

1. **Создание объекта `Person`**  
   ```js
   const Person = {
     firstName: "Иван",
     lastName: "Иванов",
     age: 30,
     getFullName() {
       return `${this.firstName} ${this.lastName}`;
     }
   };
   console.log(Person.getFullName()); // Иван Иванов
   ```

2. **Работа с массивом сотрудников**  
   ```js
   const employees = [
     { id: 1, name: "Алексей", salary: 50000 },
     { id: 2, name: "Мария", salary: 60000 },
     { id: 3, name: "Сергей", salary: 55000 }
   ];

   const totalSalary = employees.reduce((sum, e) => sum + e.salary, 0);
   console.log(totalSalary); // 165000
   ```

3. **Использование строковых методов**  
   ```js
   const text = "Пример строки для работы с регулярными выражениями.";
   const words = text.split(' ');
   console.log(words[0]); // Пример
   ```

4. **Работа с датой и временем**  
   ```js
   const now = new Date();
   console.log(now.toLocaleString()); // Текущая дата и время
   const future = new Date('2026-01-01');
   console.log(now < future); // true
   ```

5. **Математические вычисления**  
   ```js
   const radius = 5;
   const area = Math.PI * radius ** 2;
   console.log(area.toFixed(2)); // 78.54
   ```

6. **Регулярные выражения**  
   ```js
   const email = "user@example.com";
   const emailPattern = /^[^\s@]+@[^\s@]+\.[^\s@]+$/;
   console.log(emailPattern.test(email)); // true
   ```

7. **Тестирование и отладка**  
   - Запустить скрипт в консоли браузера.  
   - Проверить корректность вывода в каждом шаге.  
   - При ошибках использовать `console.log()` для вывода промежуточных значений.
