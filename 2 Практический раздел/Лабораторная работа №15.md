# Лабораторная работа № 15  
Использование свойств и методов объекта `document` для управления CSS‑свойствами HTML‑элементов, динамического изменения значений свойств, создания анимации средствами JavaScript

## Цель и задачи

1. Научиться работать с объектом `document` и его свойствами для получения и изменения элементов DOM.  
2. Овладеть методами `style`, `classList` и `setAttribute` для управления CSS‑свойствами элементов.  
3. Создать простую анимацию с помощью JavaScript, используя методы `requestAnimationFrame` и CSS‑транзишн.  
4. Проанализировать результаты работы скрипта и оценить влияние динамических изменений на пользовательский интерфейс.

## Теоретическая часть

| Понятие | Описание |
|---------|----------|
| **DOM** | Дерево объектов, представляющее структуру документа. |
| **`document.getElementById`** | Метод получения элемента по `id`. |
| **`document.querySelector`** | Метод выбора первого элемента, удовлетворяющего CSS‑селектору. |
| **`element.style`** | Объект, содержащий инлайн‑стили элемента. |
| **`element.classList`** | Набор методов для работы с классами (`add`, `remove`, `toggle`). |
| **`element.setAttribute`** | Устанавливает значение атрибута элемента. |
| **`requestAnimationFrame`** | Метод, позволяющий вызывать функцию перед следующим перерисовкой окна. |
| **CSS‑транзишн** | Плавное изменение CSS‑свойств в течение заданного времени. |
| **`transition-property`** | Свойство, определяющее, какие CSS‑свойства переходят. |
| **`transition-duration`** | Время, за которое выполняется переход. |
| **`transition-timing-function`** | Функция ускорения/замедления перехода (ease, linear и т.д.). |

## Ход выполнения работы

1. **Создание структуры проекта**  
   - Создать папку `lab15`.  
   - В ней разместить `index.html`, `styles.css`, `script.js`.  

   ```html
   <!DOCTYPE html>
   <html lang="ru">
   <head>
       <meta charset="UTF-8">
       <title>Лабораторная работа №15</title>
       <link rel="stylesheet" href="styles.css">
   </head>
   <body>
       <div id="box" class="box">Перемещаемый блок</div>
       <button id="moveBtn">Переместить</button>
       <script src="script.js"></script>
   </body>
   </html>
   ```

2. **Определение базовых стилей**  
   - В `styles.css` задать размеры и фон для блока `#box` и базовый переход.  

   ```css
   .box {
       width: 100px;
       height: 100px;
       background: #3498db;
       position: relative;
       transition: transform 0.5s ease;
   }
   ```

3. **Получение элементов**  
   - В `script.js` получить ссылки на блок и кнопку.  

   ```js
   const box = document.getElementById('box');
   const btn = document.getElementById('moveBtn');
   ```

4. **Функция перемещения**  
   - Создать переменную `posX`.  
   - При клике на кнопку увеличивать `posX` на 50 px и применять `transform`.  

   ```js
   let posX = 0;
   btn.addEventListener('click', () => {
       posX += 50;
       box.style.transform = `translateX(${posX}px)`;
   });
   ```

5. **Анимация с `requestAnimationFrame`**  
   - Создать функцию `animate`, которая плавно увеличивает размер блока.  

   ```js
   function animate() {
       let size = 100;
       function step() {
           if (size < 200) {
               size += 1;
               box.style.width = box.style.height = `${size}px`;
               requestAnimationFrame(step);
           }
       }
       requestAnimationFrame(step);
   }
   animate();
   ```

6. **Динамическое добавление/удаление класса**  
   - При двойном клике на блок менять фон через класс.  

   ```js
   box.addEventListener('dblclick', () => {
       box.classList.toggle('active');
   });
   ```

   ```css
   .box.active {
       background: #e74c3c;
   }
   ```

7. **Установка атрибутов**  
   - При загрузке страницы установить атрибут `data-status`.  

   ```js
   box.setAttribute('data-status', 'ready');
   ```

## Контрольные вопросы / задания для самопроверки

1. Как получить элемент по его `id` в JavaScript?  
2. Что делает свойство `element.style`?  
3. Как добавить класс к элементу без изменения других классов?  
4. В чем разница между `setAttribute` и прямым присваиванием `element.className`?  
5. Как реализовать анимацию без использования CSS‑транзишн, только JavaScript?  
6. Что делает `requestAnimationFrame` и почему он предпочтительнее `setInterval` для анимаций?  
7. Как задать плавное изменение цвета фона при наведении без JavaScript?  
8. Что произойдет, если в `requestAnimationFrame` вызвать рекурсивно саму функцию без проверки условия?  
9. Как добавить атрибут `aria-label` к кнопке через скрипт?  
10. Какие преимущества имеет использование `classList.toggle` перед `classList.add`/`remove`?
