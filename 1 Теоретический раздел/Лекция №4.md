# Правила создания CSS документа. Синтаксис CSS

## Цели и задачи лекции
1. Понять структуру и правила написания CSS‑документа.  
2. Научиться корректно формулировать селекторы, свойства и значения.  
3. Освоить принципы каскадности и специфичности.  
4. Практически применить знания через написание простых стилей.

## Ключевые понятия
- **CSS** – каскадные таблицы стилей, язык описания внешнего вида HTML‑документа.  
- **Селектор** – выражение, определяющее, к каким элементам применяются стили.  
- **Свойство** – аспект внешнего вида (цвет, размер, отступы и т.п.).  
- **Значение** – конкретный параметр свойства (например, `red`, `16px`).  
- **Каскад** – правило, определяющее порядок применения стилей из разных источников.  
- **Специфичность** – система оценки «важности» селекторов, влияющая на выбор стиля.  
- **Семантические теги** – HTML‑элементы, описывающие смысл содержимого (`<header>`, `<article>` и т.д.).  
- **Классы и идентификаторы** – способы выделения элементов для стилизации (`.class`, `#id`).  
- **Комментарием** – блок `/* ... */`, используемый для пояснений в CSS.  
- **Браузерный движок** – компонент, который интерпретирует CSS и отображает страницу.

## Основное содержание

### 1. Структура CSS‑документа (15 мин)  
- Заголовок файла (`/* CSS Document */`).  
- Синтаксис: `{селектор} { свойство: значение; }`.  
- Пример:

```css
/* Основные стили страницы */
body {
    background-color: #f4f4f4;
    font-family: Arial, sans-serif;
}
```

### 2. Селекторы (20 мин)  
- Простые селекторы: теги, классы, id.  
- Комбинации: группировка, потомок, соседние.  
- Псевдоклассы и псевдоэлементы: `:hover`, `::after`.  
- Примеры:

```css
/* Группировка селекторов */
h1, h2, h3 {
    margin: 0;
    padding: 0;
}

/* Потомок */
nav ul li {
    list-style: none;
}

/* Псевдокласс */
a:hover {
    text-decoration: underline;
}
```

### 3. Свойства и значения (20 мин)  
- Семантические свойства: `color`, `background`, `font-size`.  
- Блочные и строчные свойства: `display`, `float`.  
- Размеры: `px`, `em`, `%`.  
- Краткие формы: `margin: 10px 20px;`.  
- Примеры:

```css
/* Краткая форма отступов */
.container {
    margin: 20px auto;
    padding: 10px 15px;
}
```

### 4. Каскад и специфичность (15 мин)  
- Порядок применения: пользовательские стили, браузерные, встроенные.  
- Таблица специфичности: id > класс > тег.  
- Использование `!important`.  
- Пример:

```css
/* Специфичность */
#main { color: blue; }          /* 100 */
.nav { color: green; }          /* 10  */
p { color: red; }               /* 1   */
```

### 5. Практическое задание (20 мин)  
- Создать файл `style.css`.  
- Стилизовать простую страницу: заголовок, параграф, список.  
- Применить разные селекторы и свойства.  
- Проверить работу в браузере.

## Примеры и иллюстрации

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Пример CSS</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1 class="title">Добро пожаловать</h1>
    <p id="intro">Это пример стилизации.</p>
    <ul class="menu">
        <li><a href="#">Главная</a></li>
        <li><a href="#">О нас</a></li>
    </ul>
</body>
</html>
```

```css
/* style.css */
.title {
    font-size: 2em;
    color: darkslateblue;
}
#intro {
    font-style: italic;
    margin-bottom: 1em;
}
.menu li {
    display: inline-block;
    margin-right: 10px;
}
.menu a:hover {
    color: crimson;
}
```

## Выводы по лекции  
- CSS‑документ имеет строго определённый синтаксис, который легко запомнить.  
- Селекторы, свойства и значения – основа стилизации.  
- Понимание каскадности и специфичности позволяет предсказывать, какой стиль будет применён.  
- Практика с простыми примерами укрепляет навыки написания CSS.
