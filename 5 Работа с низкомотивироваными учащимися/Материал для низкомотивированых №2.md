# Установка и настройка веб‑сервера. Основы синтаксиса PHP. Анализ ОКР №2

## Цель и задачи занятия  
- Установить и настроить локальный веб‑сервер (Apache + PHP).  
- Познакомиться с базовым синтаксисом PHP и вывести простое сообщение.  
- Проанализировать ОКР №2 в контексте веб‑разработки.  

## Почему это важно?  
Веб‑сервер – «дверь» к вашим приложениям. Без него сайт не будет виден пользователям.  
Понимание PHP позволит быстро создавать динамические страницы и работать с базами данных.  
Анализ ОКР помогает оценивать эффективность решений и улучшать процессы разработки.

## Основные понятия и идеи  
1. **Веб‑сервер** – программа, которая принимает запросы из интернета и возвращает страницы.  
2. **Apache** – один из самых популярных серверов, открытый и бесплатный.  
3. **PHP** – язык сценариев, который запускается на сервере и генерирует HTML.  
4. **Локальный сервер** – установка на вашем компьютере для тестирования.  
5. **ОКР (Objectives and Key Results)** – методика постановки целей и измерения результатов.  
6. **Модуль PHP для Apache** – компонент, который позволяет Apache исполнять PHP‑код.  
7. **Путь к документу** – расположение файлов, которые сервер отдаёт клиенту.

## Объяснение темы  

### 1. Установка Apache + PHP (30 мин)  
- Скачиваем XAMPP (пакет Apache, MySQL, PHP, phpMyAdmin).  
- Устанавливаем, запускаем панель управления.  
- Проверяем `http://localhost/` – должна появиться страница XAMPP.  

```bash
# Для Ubuntu
sudo apt update
sudo apt install apache2 php libapache2-mod-php
sudo systemctl start apache2
```

> **Mini‑задание:**  
> Запусти `sudo systemctl status apache2` и посмотри статус. Что значит «active (running)»?

### 2. Создание первой PHP‑страницы (20 мин)  
- В каталоге `htdocs` создаём файл `hello.php`.  
- Добавляем код:

```php
<?php
echo "Привет, мир!";
?>
```

- Открываем `http://localhost/hello.php` – увидим сообщение.  

> **Mini‑задание:**  
> Что произойдёт, если заменить `echo` на `print`? Попробуй.

### 3. Основы синтаксиса PHP (20 мин)  
- Переменные, типы данных, операторы.  
- Условные конструкции `if/else`.  
- Циклы `for/foreach`.  

```php
<?php
$name = "Игорь";
$age = 22;

if ($age > 18) {
    echo "$name, ты взрослый.";
} else {
    echo "$name, ты ещё ребёнок.";
}
?>
```

> **Mini‑задание:**  
> Добавь переменную `$city` и выведи сообщение: «Я живу в $city».  

### 4. Анализ ОКР №2 (20 мин)  
- Определяем цель: «Установить локальный сервер за 10 минут».  
- Ключевой результат: «Сервер отвечает на запросы без ошибок».  
- Как измерить: лог Apache, отклик браузера.  

> **Mini‑задание:**  
> Сформулируй собственный ОКР для следующего задания: «Создать простую форму обратной связи».  

## Итог и выводы  
Вы успешно установили и настроили веб‑сервер, создали первую PHP‑страницу, познакомились с базовым синтаксисом и научились анализировать задачи через ОКР. Теперь вы можете продолжать развивать свои навыки, создавая более сложные приложения.
