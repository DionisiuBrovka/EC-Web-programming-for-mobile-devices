# Лабораторная работа № 21  
Подключение и настройка Bootstrap. Создание каркаса макета с использованием элементов сетки. Мобильная верстка Bootstrap. Стилистическое оформление страницы

## Цель и задачи

1. Научиться подключать Bootstrap к проекту и настраивать его конфигурацию.
2. Освоить принцип работы сетки Bootstrap (grid) и разместить элементы страницы в соответствии с требованиями.
3. Разработать адаптивную верстку, корректно отображающуюся на мобильных устройствах.
4. Применить стилистические компоненты Bootstrap (кнопки, навигация, карточки) и внести собственные CSS-правила для оформления страницы.

## Теоретическая часть

| Понятие | Описание |
|---------|----------|
| Bootstrap | CSS/JS-фреймворк, предоставляющий готовые стили и компоненты для быстрой разработки адаптивных веб‑страниц. |
| CDN | Content Delivery Network – удалённый сервер, откуда загружаются файлы Bootstrap. |
| Grid System | Система сетки, основанная на 12 колонках, позволяющая размещать контент в ряды и колонки. |
| Flexbox | Механизм выравнивания и распределения пространства между элементами внутри контейнера. |
| Breakpoints | Точки разрыва, определяющие, при каком размере экрана меняется раскладка (xs, sm, md, lg, xl). |
| Responsive utilities | Встроенные классы, управляющие видимостью и расположением элементов в разных точках разрыва. |
| Components | Готовые элементы UI (navbar, card, alert и др.) с предопределёнными стилями и скриптами. |
| Custom CSS | Пользовательские стили, переопределяющие или дополняющие стили Bootstrap. |
| Viewport meta tag | Тег `<meta name="viewport" content="width=device-width, initial-scale=1">`, обеспечивающий правильную масштабировку на мобильных устройствах. |
| Container | Базовый блок, ограничивающий ширину контента и центрирующий его. |

## Ход выполнения работы

1. **Подготовка проекта**  
   Создайте папку `bootstrap_lab21` и внутри неё файлы `index.html`, `style.css`.  
   В `index.html` подключите Bootstrap через CDN:  
   ```html
   <!DOCTYPE html>
   <html lang="ru">
   <head>
     <meta charset="UTF-8">
     <meta name="viewport" content="width=device-width, initial-scale=1">
     <title>Bootstrap Lab 21</title>
     <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
     <link rel="stylesheet" href="style.css">
   </head>
   <body>
     <!-- Контент будет добавлен далее -->
     <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
   </body>
   </html>
   ```

2. **Создание каркаса страницы**  
   Внутри `<body>` добавьте контейнер и основные блоки: `header`, `main`, `footer`.  
   ```html
   <header class="bg-primary text-white p-3">
     <h1 class="text-center">Мой сайт на Bootstrap</h1>
   </header>
   <main class="container my-4">
     <!-- Сетка будет добавлена в следующем шаге -->
   </main>
   <footer class="bg-light text-center py-3">
     © 2025 My Site
   </footer>
   ```

3. **Разметка сетки**  
   Внутри `<main>` создайте ряд (`row`) и колонки (`col`).  
   ```html
   <div class="row">
     <div class="col-12 col-md-8">
       <h2>Основной контент</h2>
       <p>Текст...</p>
     </div>
     <div class="col-12 col-md-4">
       <h3>Боковая панель</h3>
       <p>Дополнительные сведения...</p>
     </div>
   </div>
   ```
   Пояснение: `col-12` обеспечивает полную ширину на мобильных, `col-md-8/4` — делят пространство 2:1 на средних экранах и выше.

4. **Мобильная верстка**  
   Добавьте в `style.css` пользовательские стили для мелких экранов:  
   ```css
   @media (max-width: 576px) {
     header h1 { font-size: 1.5rem; }
   }
   ```
   Проверьте отображение, изменяя размер окна браузера.

5. **Стилистическое оформление**  
   - Используйте компоненты Bootstrap: `btn`, `card`, `nav`.  
   - Добавьте собственные цвета через переменные SCSS (если используете Sass) или обычный CSS.  
   - Пример кнопки:  
     ```html
     <button class="btn btn-success mt-3">Купить</button>
     ```

6. **Проверка адаптивности**  
   Включите DevTools, переключите Device Toolbar и убедитесь, что все элементы корректно перестраиваются.

7. **Финальная проверка и сохранение**  
   Откройте `index.html` в браузере, убедитесь, что страница выглядит как задумано, и сохраните все файлы.
