# Библиотека jQuery

## Цель занятия
1. Познакомить студентов с основными возможностями библиотеки jQuery и её применением в веб-разработке.  
2. Научить применять jQuery для манипуляции DOM, обработки событий и выполнения AJAX-запросов.

## Задачи занятия
- Ознакомить с историей и концепцией jQuery.  
- Показать синтаксис выбора элементов и изменение их свойств.  
- Рассмотреть обработку событий и простые анимации.  
- Ввести AJAX‑запросы через jQuery.  
- Предоставить практические задания для закрепления материала.

## План занятия (90 мин)

- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- Выборка и манипуляции DOM — 15 мин  
- Обработка событий и анимация — 15 мин  
- AJAX‑запросы с jQuery — 15 мин  
- Закрепление материала — 10 мин  
- Итог занятия — 5 мин  

### Содержание этапов

**Организационный момент**  
Вступительное слово, проверка присутствия, краткое напоминание о предыдущих занятиях по JavaScript и Python, а также о цели сегодняшнего урока.

**Актуализация знаний**  
Краткое обсуждение вопросов: что такое DOM, зачем нужны библиотеки, и какие проблемы решает jQuery. Проводится быстрый опрос студентов о знакомстве с библиотекой.

**Выборка и манипуляции DOM**  
Объяснение синтаксиса селекторов jQuery, методы `.css()`, `.attr()`, `.html()` и `.text()`. Демонстрация примеров изменения содержимого страницы.

**Обработка событий и анимация**  
Показ работы событий `.click()`, `.hover()`, `.keyup()` и простых анимаций с помощью `.fadeIn()`, `.slideToggle()`. Практика: добавление интерактивных кнопок на страницу.

**AJAX‑запросы с jQuery**  
Введение в методы `.ajax()`, `.get()`, `.post()`. Пример загрузки данных с внешнего API и отображения их на странице без перезагрузки.

**Закрепление материала**  
Короткий тест (5 вопросов) и небольшое практическое задание: создать мини‑приложение, использующее выборку, события и AJAX.

**Итог занятия**  
Подведение итогов, ответы на вопросы, раздача материалов по теме и напоминание о домашнем задании.

## Домашнее задание
1. Создать небольшую веб‑страницу, где при клике на кнопку меняется текст и цвет фона, а также загружается список данных из публичного API (например, https://jsonplaceholder.typicode.com/posts).  
2. Написать комментарии к коду, объясняющие назначение каждой функции jQuery.
