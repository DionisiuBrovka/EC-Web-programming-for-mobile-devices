# Лабораторная работа № 9. Создание и использование массивов в сценариях на языке JavaScript. Применение методов работы с массивами

## Цель и задачи
1. Ознакомиться с синтаксисом создания и инициализации массивов в JavaScript.  
2. Научиться применять основные методы работы с массивами: `push`, `pop`, `shift`, `unshift`, `splice`, `slice`, `map`, `filter`, `reduce`.  
3. Понять принципы перебора элементов массива с помощью циклов и встроенных функций.  
4. Разработать практический скрипт, демонстрирующий обработку данных в массиве и вывести результаты на экран.

## Теоретическая часть
| Понятие | Определение | Ключевой момент |
|---------|-------------|-----------------|
| **Массив** | Коллекция упорядоченных значений, доступных по числовому индексу. | Индексы начинаются с нуля. |
| **`push(item)`** | Добавляет элемент в конец массива. | Возвращает новую длину массива. |
| **`pop()`** | Удаляет последний элемент массива. | Возвращает удалённый элемент. |
| **`shift()`** | Удаляет первый элемент массива. | Возвращает удалённый элемент. |
| **`unshift(item)`** | Добавляет элемент в начало массива. | Возвращает новую длину массива. |
| **`splice(start, deleteCount, ...items)`** | Вставляет/удаляет элементы, начиная с индекса `start`. | Модифицирует исходный массив. |
| **`slice(start, end)`** | Возвращает новый массив с элементами от `start` до `end-1`. | Не меняет исходный массив. |
| **`map(fn)`** | Создаёт новый массив, применяя функцию `fn` к каждому элементу. | Функция возвращает новый элемент. |
| **`filter(fn)`** | Создаёт новый массив, содержащий элементы, прошедшие фильтр `fn`. | Функция возвращает булево значение. |
| **`reduce(fn, init)`** | Сводит массив к одному значению, применяя аккумуляторную функцию `fn`. | `init` - начальное значение аккумулятора. |
| **`for...of`** | Перебор элементов массива без индексов. | Удобен для чтения. |

## Ход выполнения работы

1. **Создание массива**  
   Пояснение: определяем массив чисел.  
   ```javascript
   const numbers = [5, 12, 7, 3, 9];
   ```

2. **Добавление элементов**  
   Пояснение: добавляем в конец и в начало.  
   ```javascript
   numbers.push(15);          // [5,12,7,3,9,15]
   numbers.unshift(1);        // [1,5,12,7,3,9,15]
   ```

3. **Удаление элементов**  
   Пояснение: удаляем последний и первый элемент.  
   ```javascript
   const last = numbers.pop();   // 15, массив без 15
   const first = numbers.shift(); // 1, массив без 1
   ```

4. **Вставка и удаление с помощью `splice`**  
   Пояснение: удаляем 2 элемента начиная с индекса 2 и вставляем новые.  
   ```javascript
   numbers.splice(2, 2, 8, 10); // заменяем 12 и 7 на 8 и 10
   ```

5. **Получение подмассива с `slice`**  
   Пояснение: копируем часть массива.  
   ```javascript
   const sub = numbers.slice(1, 4); // элементы с 1 по 3
   ```

6. **Перебор массива с `for...of`**  
   Пояснение: выводим каждый элемент.  
   ```javascript
   for (const num of numbers) {
     console.log(num);
   }
   ```

7. **Преобразование массива с `map`**  
   Пояснение: создаём новый массив, умножая каждый элемент на 2.  
   ```javascript
   const doubled = numbers.map(n => n * 2);
   ```

8. **Фильтрация массива с `filter`**  
   Пояснение: выбираем только чётные числа.  
   ```javascript
   const evens = numbers.filter(n => n % 2 === 0);
   ```

9. **Сокращение массива с `reduce`**  
   Пояснение: считаем сумму элементов.  
   ```javascript
   const sum = numbers.reduce((acc, n) => acc + n, 0);
   ```

10. **Вывод результатов**  
    Пояснение: печатаем все полученные значения в консоль.  
    ```javascript
    console.log('Original:', numbers);
    console.log('Doubled:', doubled);
    console.log('Evens:', evens);
    console.log('Sum:', sum);
    ```
