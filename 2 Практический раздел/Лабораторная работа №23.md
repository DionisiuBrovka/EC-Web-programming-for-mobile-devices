# Лабораторная работа № 23  
Установка и настройка веб‑сервера. Внедрение PHP‑скрипта в веб‑документ. Использование функций для работы с массивами, строками, файлами и каталогами, датой и временем в PHP‑скрипте.

## Цель и задачи  
1. Освоить процедуру установки и базовой настройки веб‑сервера Apache (или Nginx) с поддержкой PHP.  
2. Научиться размещать и запускать PHP‑скрипт в рамках веб‑документа, проверяя его работу в браузере.  
3. Приобрести навыки работы с массивами, строками, файлами и каталогами, а также с датой и временем средствами PHP.  
4. Выполнить практическое задание, демонстрирующее интеграцию всех перечисленных функций в одном скрипте.

## Теоретическая часть  
1. **Веб‑сервер** – программа, обслуживающая HTTP‑запросы от клиентов (браузеров).  
2. **Apache** – один из самых популярных открытых веб‑серверов, использующий модульную архитектуру.  
3. **PHP** – язык сценариев, интерпретируемый сервером, широко применяемый для динамического контента.  
4. **mod_php** – Apache‑модуль, позволяющий обрабатывать PHP‑файлы.  
5. **`$_SERVER`** – суперглобальный массив, содержащий информацию о сервере и текущем запросе.  
6. **`array_*`** – семейство функций для работы с массивами (создание, сортировка, поиск).  
7. **`str_*`** – функции для обработки строк (поиск, замена, разбиение).  
8. **`file_*`** – набор функций для чтения/записи файлов и работы с каталогами.  
9. **`date()`** – функция форматирования даты и времени.  
10. **`strtotime()`** – преобразует строковое представление даты/времени в метку времени.

## Ход выполнения работы  

1. **Установка веб‑сервера**  
   - Откройте терминал и выполните:  
     ```bash
     sudo apt update
     sudo apt install apache2
     ```  
   - Убедитесь, что сервис запущен:  
     ```bash
     sudo systemctl status apache2
     ```  
   - Откройте браузер и перейдите по адресу `http://localhost`. Должна отобразиться стандартная страница Apache.

2. **Установка PHP и модуля для Apache**  
   - Установите PHP и модуль `libapache2-mod-php`:  
     ```bash
     sudo apt install php libapache2-mod-php
     ```  
   - Перезапустите сервер:  
     ```bash
     sudo systemctl restart apache2
     ```  
   - Проверьте работу PHP, создав файл `info.php` в `/var/www/html`:  
     ```php
     <?php
     phpinfo();
     ?>
     ```  
     Перейдите в браузере по адресу `http://localhost/info.php` и убедитесь, что отображается информация о PHP.

3. **Создание простого PHP‑скрипта**  
   - В каталоге `/var/www/html` создайте файл `lab23.php`:  
     ```php
     <?php
     echo "<h1>Добро пожаловать в лабораторную работу №23</h1>";
     ?>
     ```  
   - Откройте `http://localhost/lab23.php` в браузере; должна появиться заголовок.

4. **Работа с массивами**  
   - В `lab23.php` добавьте код:  
     ```php
     <?php
     $colors = ["red", "green", "blue"];
     echo "Первый цвет: " . $colors[0] . "<br>";
     sort($colors);
     echo "Сортированный массив: " . implode(", ", $colors) . "<br>";
     ?>
     ```  
   - Перезагрузите страницу и проверьте вывод.

5. **Обработка строк**  
   - Добавьте:  
     ```php
     <?php
     $text = "Hello, World!";
     echo "Строка в верхнем регистре: " . strtoupper($text) . "<br>";
     echo "Длина строки: " . strlen($text) . "<br>";
     ?>
     ```  

6. **Работа с файлами и каталогами**  
   - Создайте файл `data.txt` в `/var/www/html`:  
     ```txt
     Line 1
     Line 2
     Line 3
     ```  
   - В скрипте добавьте чтение файла:  
     ```php
     <?php
     $lines = file("data.txt", FILE_IGNORE_NEW_LINES);
     echo "Содержимое файла:<br>";
     foreach ($lines as $line) {
         echo htmlspecialchars($line) . "<br>";
     }
     ?>
     ```  

7. **Дата и время**  
   - Добавьте:  
     ```php
     <?php
     echo "Текущая дата и время: " . date("Y-m-d H:i:s") . "<br>";
     $future = strtotime("+1 week");
     echo "Дата через неделю: " . date("Y-m-d", $future) . "<br>";
     ?>
     ```  

8. **Финальный скрипт**  
   - Объедините все блоки кода в один файл `lab23.php`.  
   - Убедитесь, что страница выводит корректный HTML, данные из массива, строки, файл и дату.
