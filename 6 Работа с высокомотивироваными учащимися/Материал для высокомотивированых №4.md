# Основные методы библиотеки jQuery

jQuery — это одна из самых популярных JavaScript‑библиотек, упрощающая работу с DOM, AJAX‑запросами и анимацией. Несмотря на растущую популярность современных фреймворков, jQuery остаётся востребованным в реальных проектах: в стартапах, корпоративных системах и встраиваемых веб‑компонентах. Знание его ключевых методов позволяет быстро прототипировать, интегрировать сторонние сервисы и повышать качество кода, соблюдая принципы DRY и модульности.

## Ключевые понятия

| Термин | Расширенное определение | Зачем нужно |
|--------|------------------------|-------------|
| **$ (jQuery)** | Глобальный объект, служащий фабрикой для создания коллекций элементов и вызова методов. | Упрощает синтаксис, обеспечивает кросс‑браузерность. |
| **Коллекция jQuery** | Массив‑подобный объект, содержащий DOM‑элементы и методы для их обработки. | Позволяет выполнять операции над множеством элементов одновременно. |
| **Селектор CSS** | Строка, определяющая набор элементов по CSS‑правилам, используемая в `$()`. | Позволяет быстро находить нужные элементы без ручного обхода DOM. |
| **Метод .on()** | Устанавливает обработчик событий для выбранных элементов, включая делегирование. | Позволяет управлять пользовательским вводом и динамическим содержимым. |
| **Метод .ajax()** | Универсальный способ отправки HTTP‑запросов и обработки ответов. | Необходим для интеграции с API, загрузки данных без перезагрузки страницы. |
| **Метод .animate()** | Создаёт плавные анимации CSS‑свойств. | Улучшает UX, визуально обогащает интерфейс. |
| **Метод .css()** | Чтение/запись CSS‑свойств элементов. | Позволяет динамически менять стили в ответ на события. |
| **Метод .attr()** | Чтение/запись атрибутов HTML. | Управляет семантикой и поведением элементов. |
| **Метод .data()** | Хранит произвольные данные, связанные с элементом. | Позволяет сохранять состояние без изменения DOM. |
| **Метод .each()** | Итерация по коллекции с пользовательской функцией. | Удобен для обработки сложных наборов данных. |

## Примеры и иллюстрации

### 1. Динамическое меню

```html
<nav>
  <ul id="menu">
    <li><a href="#">Главная</a></li>
    <li><a href="#">О нас</a></li>
    <li><a href="#">Контакты</a></li>
  </ul>
</nav>

<script>
$(function() {
  $('#menu').on('click', 'a', function(e) {
    e.preventDefault();
    $('#menu li').removeClass('active');
    $(this).parent().addClass('active');
  });
});
</script>
```

> *Неочевидный кейс:* В старых версиях jQuery делегирование событий не работает для динамически добавленных элементов, если не использовать `$(document).on(...)`. Это поднимает вопросы совместимости.

### 2. AJAX‑загрузка контента

```js
$('#loadMore').on('click', function() {
  $.ajax({
    url: '/posts',
    method: 'GET',
    dataType: 'json',
    success: function(posts) {
      posts.forEach(function(post) {
        $('#posts').append(
          $('<div>').addClass('post').text(post.title)
        );
      });
    },
    error: function() {
      alert('Не удалось загрузить посты');
    }
  });
});
```

> *Кейс из индустрии:* Большие e‑commerce сайты используют подобные техники для бесконечной прокрутки каталога, снижая нагрузку на сервер.

### 3. Анимация с параметрами

```js
$('#box').on('click', function() {
  $(this).animate({
    width: '200px',
    opacity: 0.5
  }, 400, 'swing', function() {
    $(this).css('background-color', '#ff5722');
  });
});
```

> *Эксперимент:* Попробуйте заменить `swing` на `linear` и сравнить ощущение от анимации.
