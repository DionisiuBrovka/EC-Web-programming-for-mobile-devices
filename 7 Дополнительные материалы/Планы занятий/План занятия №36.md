# Основные методы библиотеки jQuery

## Цель занятия
1. Ознакомить студентов с ключевыми методами jQuery, применяемыми для манипуляции DOM, работы с событиями и AJAX‑запросами.  
2. Научить применять эти методы на практике через простые интерактивные примеры.

## Задачи занятия
- Понять структуру jQuery и его преимущества перед чистым JavaScript.  
- Изучить основные селекторы и методы изменения содержимого.  
- Освоить работу с событиями и обработчиками.  
- Научиться выполнять AJAX‑запросы через jQuery.  
- Разработать простую веб‑страницу, использующую изученные методы.

- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин

## Содержание этапов
- **Организационный момент**  
  Приём аудитории, проверка присутствия, краткое напоминание о предыдущем занятии и цели сегодняшнего урока.

- **Актуализация знаний**  
  Быстрый опрос: что такое DOM, какие есть методы изменения его содержимого в чистом JavaScript. Вопросы для проверки понимания.

- **Селекторы и манипуляции DOM**  
  Демонстрация основных jQuery‑селекторах (`$()`, `$('.class')`, `$('#id')`) и методов `html()`, `text()`, `append()`, `prepend()`, `remove()`. Практическое задание: изменить структуру примера страницы.

- **Работа с событиями**  
  Обсуждение событий `click`, `hover`, `keyup` и методов `on()`, `off()`. Взаимодействие с пользователем: добавление и удаление элементов по клику.

- **AJAX‑запросы**  
  Введение в `$.ajax`, `$.get`, `$.post`. Пример загрузки данных с публичного API и отображение результата без перезагрузки страницы.

- **Закрепление**  
  Короткая контрольная работа: студентам даётся задача объединить все методы в один мини‑проект (например, список задач с AJAX‑загрузкой и возможностью добавить/удалить пункты).

- **Итог занятия**  
  Подведение итогов, ответы на вопросы, объявление домашнего задания.

1. Создать страницу «Блог», где список статей загружается через AJAX из JSON‑файла.

Учащимся рекомендуется использовать только методы jQuery, продемонстрированные на занятии.
