# Верстка шаблонов с помощью Bootstrap

## Цель занятия  
1. Познакомить студентов с основными компонентами Bootstrap и их применением в создании адаптивных шаблонов.  
2. Научить использовать сеточную систему, компоненты и утилиты Bootstrap для быстрой разработки веб‑страниц.

## Задачи занятия  
- Ознакомить с историей и архитектурой Bootstrap.  
- Показать практическое применение сетки, карточек, навигации и форм.  
- Разработать простую страницу‑шаблон, используя Bootstrap‑компоненты.  
- Проверить понимание принципов адаптивного дизайна.  

## План занятия  
- Организационный момент — 5 мин  
- Актуализация знаний — 10 мин  
- Изучение нового материала — 20 мин  
- Закрепление — 15 мин  
- Итог занятия — 5 мин  

## Содержание этапов  

### Организационный момент — 5 мин  
Краткое приветствие, проверка присутствия, обзор плана занятия, напоминание о домашнем задании.

### Актуализация знаний — 10 мин  
Обсуждение ранее изученных тем: HTML‑семантика, CSS‑свойства, основы JavaScript. Вопросы‑ответы для проверки базовых знаний, необходимых для работы с Bootstrap.

### Изучение нового материала — 20 мин  
1. Введение в Bootstrap: CDN, локальная установка, структура файлов.  
2. Сеточная система: колонки, контейнеры, гибкая сетка.  
3. Компоненты: навигация, карточки, кнопки, формы.  
4. Утилиты: отступы, цвета, выравнивание, скрытие элементов.  
Показ примеров кода и живой демонстрации в IDE.

### Закрепление — 15 мин  
Практическое задание: студенты самостоятельно создают простую страницу‑шаблон с заголовком, навигацией, секцией карточек и формой обратной связи, используя Bootstrap‑компоненты. Преподаватель наблюдает, дает корректировки.

### Итог занятия — 5 мин  
Обзор ключевых моментов, ответы на вопросы, напоминание о домашнем задании и критериях оценки.

## Оборудование и материалы  
- Компьютер с доступом в интернет.  
- Проектор/экран для демонстрации.  
- IDE (VS Code, PyCharm) с установленным расширением для работы с HTML/CSS.  
- Браузер (Chrome, Firefox).  
- Учебные материалы: слайды, шпаргалка по Bootstrap, ссылка на официальную документацию.  

## Контроль и оценивание  
- Краткие вопросы в ходе лекции (5‑10 вопросов).  
- Оценка практического задания по критериям: корректность структуры, использование компонентов, адаптивность.  
- Мини‑тест (5 вопросов с выбором) после занятия для проверки усвоения материала.  

## Домашнее задание  
1. Создать страницу‑портфолио, используя Bootstrap‑карточки для проектов.  
2. Добавить адаптивную навигацию и форму контактов, стилизованную с помощью утилит Bootstrap.  
Сдать HTML, CSS и скриншот результата в течение недели.
