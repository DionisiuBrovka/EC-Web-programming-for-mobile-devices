# Физическое и логическое форматирование текста в веб‑документе. Внедрение контента на веб‑страницу

## Цели и задачи лекции
- Понять разницу между физическим и логическим форматированием в HTML.
- Научиться использовать семантические теги для структуры контента.
- Освоить принципы внедрения текста и кода в веб‑страницы с помощью Python‑скриптов.
- Разработать простые шаблоны веб‑страниц, где логика отделена от разметки.

## Ключевые понятия
| Термин | Определение |
|--------|-------------|
| **Физическое форматирование** | Визуальное представление текста (шрифты, отступы, цвета). |
| **Логическое форматирование** | Семантическая разметка, определяющая роль элемента (заголовок, параграф). |
| **Семантический тег** | HTML‑тег, указывающий смысл содержимого (например, `<header>`, `<article>`). |
| **Шаблон** | Файл, содержащий фиксированную разметку и переменные, подставляемые программой. |
| **Jinja2** | Шаблонизатор для Python, позволяющий генерировать HTML из шаблонов. |
| **Markdown** | Упрощённый язык разметки, конвертируемый в HTML. |
| **Код‑стиль** | Встроенный CSS, задающий визуальные свойства элементов. |
| **Серверный рендеринг** | Создание HTML на сервере (Python) перед отправкой клиенту. |
| **Клиентский рендеринг** | Генерация контента в браузере с помощью JavaScript. |
| **Кросс‑браузерность** | Способность документа корректно отображаться во всех браузерах. |

## Основное содержание

### 1. Введение в форматирование (10 мин)
- Определение физического и логического форматирования.
- Почему важно разделять структуру и стиль.
- Краткая история HTML: от `<b>` к `<strong>`.

### 2. Семантические теги и их роль (15 мин)
- **Заголовки**: `<h1>`–`<h6>`.
- **Абзацы**: `<p>`.
- **Списки**: `<ul>`, `<ol>`, `<li>`.
- **Секции**: `<section>`, `<article>`, `<aside>`, `<nav>`, `<footer>`, `<header>`.
- Примеры семантической разметки.

> ```html
> <article>
>   <header>
>     <h1>Заголовок статьи</h1>
>   </header>
>   <p>Краткое введение в тему.</p>
>   <footer>Автор: Иван Иванов</footer>
> </article>
> ```

### 3. Физическое форматирование через CSS (15 мин)
- Инлайн‑стили, атрибут `style`.
- Встроенный `<style>`.
- Внешние таблицы стилей.
- Стилизация элементов: `font-family`, `font-size`, `margin`, `padding`, `color`.
- Базовый пример: стилизация заголовка и абзаца.

> ```html
> <style>
>   h1 { color: #2c3e50; }
>   p  { font-family: Arial, sans-serif; line-height: 1.6; }
> </style>
> ```

### 4. Интеграция Python и HTML (20 мин)
- Установка и настройка Jinja2.
- Создание шаблона `template.html`.
- Подставляемые переменные: `{{ title }}`, `{% for item in list %}`.
- Генерация файла `output.html` из скрипта.

> ```python
> from jinja2 import Environment, FileSystemLoader
> 
> env = Environment(loader=FileSystemLoader('.'))
> template = env.get_template('template.html')
> data = {
>     'title': 'Пример страницы',
>     'items': ['Пункт 1', 'Пункт 2', 'Пункт 3']
> }
> with open('output.html', 'w', encoding='utf-8') as f:
>     f.write(template.render(data))
> ```

### 5. Markdown в веб‑документах (10 мин)
- Конвертация Markdown в HTML с помощью `markdown2`.
- Пример Markdown‑файла и его рендеринг.

>

> - Список
