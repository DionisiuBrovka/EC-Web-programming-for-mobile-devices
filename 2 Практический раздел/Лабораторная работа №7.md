# Лабораторная работа № 7  
Использование CSS‑свойств для добавления визуальных эффектов и анимации на веб‑страницу  

## Цель и задачи  
1. Научиться применять CSS‑свойства для создания визуальных эффектов (тени, градиенты, трансформации).  
2. Освоить принципы работы ключевых кадров (keyframes) и анимаций в CSS.  
3. Реализовать простую анимацию пользовательского интерфейса и проанализировать её влияние на восприятие.  
4. Подготовить отчёт, отражающий теоретические знания, практические шаги и результаты экспериментов.  

## Теоретическая часть  
| Понятие | Определение |
|---------|-------------|
| **CSS** | Стилевой язык, описывающий внешний вид HTML‑элементов. |
| **Селектор** | Условие, определяющее, к каким элементам применяется правило. |
| **Свойство** | Атрибут, задающий конкретный аспект стиля (цвет, размер, позиционирование). |
| **Трансформация** (`transform`) | Смена формы элемента (rotate, scale, skew, translate). |
| **Тень** (`box-shadow`, `text-shadow`) | Эффект, создающий визуальное разделение от фона. |
| **Градиент** (`linear-gradient`, `radial-gradient`) | Плавный переход между цветами. |
| **Ключевые кадры** (`@keyframes`) | Определяют промежуточные состояния анимации. |
| **Свойство анимации** (`animation-name`, `animation-duration`, `animation-timing-function`, `animation-iteration-count`) | Управление воспроизведением ключевых кадров. |
| **Псевдоэлементы** (`::before`, `::after`) | Создание виртуальных элементов для дополнительных эффектов. |
| **Псевдоклассы** (`:hover`, `:focus`) | Реакция на действия пользователя. |  

## Ход выполнения работы  

1. **Создание проекта**  
   - Создайте папку `lab7` и в ней файлы `index.html`, `style.css`.  
   ```html
   <!-- index.html -->
   <!DOCTYPE html>
   <html lang="ru">
   <head>
       <meta charset="UTF-8">
       <title>Лабораторная работа №7</title>
       <link rel="stylesheet" href="style.css">
   </head>
   <body>
       <div class="card">
           <h2>Пример карточки</h2>
           <p>Текст внутри карточки.</p>
       </div>
   </body>
   </html>
   ```

2. **Добавление базового оформления**  
   - В `style.css` задайте стили для `.card`: ширина, отступы, фон, тень.  
   ```css
   .card {
       width: 300px;
       margin: 50px auto;
       padding: 20px;
       background: #fff;
       border-radius: 8px;
       box-shadow: 0 4px 10px rgba(0,0,0,0.1);
   }
   ```

3. **Создание анимации**  
   - Определите ключевые кадры `@keyframes pulse`.  
   ```css
   @keyframes pulse {
       0%   { transform: scale(1);   background: #f0f8ff; }
       50%  { transform: scale(1.05); background: #e0ffff; }
       100% { transform: scale(1);   background: #f0f8ff; }
   }
   ```

4. **Применение анимации к элементу**  
   - Добавьте в `.card` свойства `animation`.  
   ```css
   .card {
       animation: pulse 2s ease-in-out infinite;
   }
   ```

5. **Псевдоэлементы и hover‑эффект**  
   - Добавьте `::after` для декоративного штриха и `:hover` для интерактивности.  
   ```css
   .card::after {
       content: '';
       display: block;
       width: 50px;
       height: 4px;
       background: #00f;
       margin: 10px auto 0;
   }
   .card:hover {
       box-shadow: 0 8px 15px rgba(0,0,0,0.2);
       transform: translateY(-5px);
   }
   ```

6. **Экспериментальная часть**  
   - Смените длительность, тип тайминга и количество итераций анимации.  
   - Сравните восприятие (быстрая vs медленная анимация).  
   - Сохраните скриншоты до и после изменений.

7. **Фиксация результатов**  
   - Запишите в отчёт наблюдаемые изменения, время выполнения и комментарии о том, как анимации влияют на пользовательский интерфейс.
